　　荧惑古星之上，随着众人看到的更多，他们也就越来越绝望了，因为大家都知道，这里不是地球，他们真是来到了别的星球，甚至连最后一丝希望，也灭绝了。
<br/>　　在短短的几个小时之内，众人的心态便从慌乱，震惊，惶恐，不安等，渐渐变为绝望，好奇，麻木。
<br/>　　人只有经历常人难见的困苦，才能显示出各自的心性之间的不同。如此刻，有的人绝望，而有的人却很冷静，心态经过初始的震惊，如今已经慢慢接受了现实，学会去适应变故。
<br/>　　既来之，则安之！
<br/>　　这就是一部分心态极好之人的内心写照，所以，他们注定不会甘于平凡。
<br/>　　随着众人向着光源的地方走去，渐渐发现了荧惑古星上的异状。
<br/>　　如同鲜血浇灌后的大地，残存的碎瓦片。
<br/>　　断壁残垣，一地的瓦砾，似乎是在诉说一段不为人知的往事。夜月下，这里显得格外幽远寂静，时而还伴随着呜呜的冷风吹过，更增添几分寒冷神秘的色彩。
<br/>　　过去此处应该是一片连绵的宏伟宫殿，而此刻却是一片凄惨的景象。
<br/>　　众人心中惊讶不已，种种疑惑缠绕心头，却无人能够为他们解惑。同时，他们也有一种不祥的预感，此地绝对不简单。
<br/>　　随着继续前进，终于，他们见到了一件古庙。
<br/>　　这间古庙，静静地坐落在那里，青灯古佛，一点灯光暗淡摇曳，好似随时可能熄灭。
<br/>　　走到此处，所有人都难掩眼中的惊异之色，后方的宏伟宫殿早已成了碎片，随着历史的风沙被掩盖，而此地的古庙却能存留下来，一切的一切，都说明此处的不平凡。
<br/>　　难道……此处真的如神殿中的神秘声音所说，是……大雷音寺？
<br/>　　“这里怎么会有一座古庙存在，难道真的是……大雷音寺？”有人惊疑道。
<br/>　　此话一出，顿时所有人不平静了，同时，他们的心头也有些火热。大雷音寺是什么地方？他们从小就耳熟能详。大雷音寺一直存在神话传说之中，是传说中佛祖为了镇压天下大魔所设置的地方。
<br/>　　“这里有一块铜匾，上面有字迹！”有人看见古庙前的一块匾额，上面刻有四个古字，如龙蛇盘绕，禅意无边，是钟鼎文。
<br/>　　“大什么寺？这是什么意思？”
<br/>　　“这四个字是‘大雷音寺’！”此时，叶梵同学将四个字读了出来。
<br/>　　“什么，竟然真的是……大雷音寺？！”在场众人莫不吃惊。
<br/>　　“这怎么可能？”
<br/>　　“如此说来，那混沌神殿也是真的了？”这一刻，没人能够平静下来。
<br/>　　仙人，那可是仙人啊，仙缘就在自己等人的面前，只要自己能够抓得住这个机会，以后自己说不定可以成为仙人了。
<br/>　　面对仙人神迹，没有人能够沉静下来。相比成仙，什么权势，什么钱财，什么文凭，通通都弱爆了。就问你，那什么权势能够无敌吗，那什么钱财能够长生吗？
<br/>　　若是要在两者之间做出选择，几乎所有人都会选择仙缘。
<br/>　　面对可能出现的仙缘，众人再次想起了混沌主神的任务，“任务探索大雷音寺古迹，要求是活着回到神殿”。
<br/>　　这个任务最重要的便是活着。这说明，这里很有可能有什么他们不知道的危险，会死人。
<br/>　　想到这里，众人心中一紧，顿时有些戒备的看着周围，想要发现什么。可是，四周除了破败的大地以及一间古庙，还有那呜呜抽噎着的寒风之外，便什么都没有了。
<br/>　　夜邪忽然向前走去，叶梵与庞博相互看了一眼，也向前走去。于此同时，其他人见此，好似想到了什么，脸色顿时一变，眼神闪烁，也连忙跟了进去。
<br/>　　夜邪一进古庙，便将石佛前的一根金刚杵抄起，而叶梵也眼疾手快，连忙将那唯一的青灯拿起。同一时间，其他人也进入古庙，开始默默搜寻。
<br/>　　这里乃是传说中佛陀待的地方，很有可能存在神邸留下的神器，若是他们能够得到一二，那岂不要逆天了吗。
<br/>　　于是，有人从灰烬中寻得一个陈旧的蒲团，岁月未能将其磨灭。
<br/>　　有一女同学得到一串紫色念珠，仅有六颗，如水晶般剔透，每颗都有龙眼般大小，闪烁着淡淡的光泽。
<br/>　　有李小曼在石佛脚下得到半截玉如意，擦去灰尘，依旧可见淡淡的光晕。
<br/>　　又有其他各自寻得残破的铜钟、香炉、戒尺、铜铃、鱼鼓、木鱼等佛门器物。
<br/>　　其中庞博运气最衰，一无所获，什么都没有遇到，直到回头后，看见门前的铜匾，顿时眼前一亮，只觉得能够让佛陀当做门匾的，定然十分不简单。他跑上去将之抄了起来。
<br/>　　其他人见此，纷纷后悔莫及，只叹怎么自己没有想到。
<br/>　　忽然，“轰隆”一声声响，古庙开始剧烈摇晃，其中一尊石佛突然龟裂，发出咔擦咔擦的声响。
<br/>　　而后，有佛家六字真言响起：“嗡、嘛、呢、叭、咪、吽……”
<br/>　　宏大的佛音响彻天宇，震动苍穹，天地八方具皆颤抖！
<br/>　　慈悲、庄严、玄妙的禅音无比浩大，洗净凡尘，褪去污垢。一时之间，古庙周围都沐浴在一片神圣的光芒之下。
<br/>　　同一时间，众人在古庙中寻得的所有器物，无论是完好的，还是破碎了，此刻全部散发出柔和的光芒，光芒照耀，释放出一种古意，让每一个人都心惊不已。
<br/>　　最红，轰的一声巨响，古庙内的所有石佛连着古庙，一同化为飞灰，在一场阴风之中，化为粉碎。
<br/>　　接下来，众人手中的所有器物，全都暗淡下来，再次归于平凡。
<br/>　　众人惊疑不定，不知道发生了什么，是因为他们取走佛门器物，破坏了古庙内的一切，还是因为别的原因，导致古庙毁灭？
<br/>　　这时，夜邪走到叶梵与庞博身边，慎重地说道：“接下来一定要抓紧手中的佛门器物，无论遇到什么危险，都不能丢弃。”
<br/>　　“为什么？夜邪，你是不是发现了什么？”叶梵有些疑惑道。
<br/>　　“你听我的就行了，不要问为什么，接下来会有生命危险。”夜邪淡淡道，这声音，只有他们三人能够听到。
<br/>　　“嗯。”两人听后，尽管心中仍然疑惑不解，但还是选择了听从夜邪的意见，因为他们相信夜邪是不会骗他们的。
<br/>　　“那……那是……”有人惊呼道，语无伦次，显得极为吃惊。
<br/>　　“沙暴……火星上的超级沙尘暴……！”有人惊声叫道。
<br/>　　仅仅只是一瞬间，漫天的的星月便全部消失，无穷的红褐色沙尘彻底覆盖苍穹，一场席卷整片火星的大风暴开启了。
<br/>　　众人惊恐不已，但很快他们就发现了，以五色祭坛与大雷音寺为轴线，方圆千里之内有圆罩笼罩，任由远处风暴如何，但近前却风平浪静。这让他们对神迹更加崇拜了。
<br/>　　然而，很快他们便发现了，天穹上那层光罩正在逐渐消融，很快便会暗淡下来。
<br/>　　“快跑，赶紧回到混沌神殿！”有人惊呼，想到了那唯一的出路。这一刻，他们总算有点明白了混沌神殿发布的任务了，在这样的风暴之中，他们可能全部葬身。
<br/>　　可惜，他们并不知道，真正的危机，才刚刚降临！
<br/>　　……
