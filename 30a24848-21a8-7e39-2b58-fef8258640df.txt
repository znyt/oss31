<br/>    陈明的身旁，十二柄国器飞舞，此刻的他，可是能够完全激发国器的威力的，而刚刚刺穿阴月天狼的那一柄，便是破空。
<br/>    
<br/>        十二柄国器朝着周围飞去，一阵阵金铁交鸣之声响彻在陈明的周围，那是那些妖王们正在抵抗，不过他们的抵抗并没有什么用处，因为天空之中，又是数百柄飞剑飞来。
<br/>    
<br/>        阴月天狼还没有靠近陈明，身体之中，已经是被十多柄飞剑穿过。
<br/>    
<br/>        浑身鲜血的阴月天狼从狼背上落下，摔落在地面上，他的眼神逐渐变得灰暗，他用尽最后的力气挣扎着看了他住了一辈子的妖城，“原来.........人间.......也.......可以......是炼狱.......”
<br/>    
<br/>        妖族大君看向头颅无力垂下的阴月天狼，“二弟！”
<br/>    
<br/>        妖族大君的眼中只剩下无尽的怒火，他看向让他生出无尽怒火的陈明，“陈明！给我死来！”
<br/>    
<br/>        就在这时，紫霞仙子直接一个闪身飞到妖族大君的身前，“不要忘了，你的对手，是我！”
<br/>    
<br/>        妖族大君狂暴地挥舞着手中的双刀朝着紫霞仙子劈去，“刀裂苍穹！给我滚开！”
<br/>    
<br/>        两道刀光蔓延百丈，如同可以将天地劈开来，紫霞仙子弯弓搭箭，三只长箭上弦，顿时就是三箭齐发！
<br/>    
<br/>        紫色的霞光与刀光撞击在一处，直接轰出了一个数百丈的大坑来，妖族大君看得目次欲裂，这紫霞仙子，竟然也是一位大君！
<br/>    
<br/>        在百里之外，苏青阳和雪自悟看向妖城的方向，“嗯，妖族大君的兵力怎么还没到啊？”
<br/>    
<br/>        雪自悟忽然指了指妖城，“那里出了大事了！”
<br/>    
<br/>        两人看向百里之外的妖城，即便是相隔百里，都能看到妖城之中的乱象，“不是吧，之前我们就见识过宗师的群战能力，那时，宗师还没到大君，现在，宗师成了大君，这群战能力，更恐怖了！”
<br/>    
<br/>        “妖城之中，现在恐怕是人间炼狱！”
<br/>    
<br/>        “看来宗门之中对于燕山的实力要重新划分了，宗师的实力，虽然只是新晋大君的，但是群战能力，堪比数位大君，恐怕已经比得上超脱境第二个境界的大君了。”
<br/>    
<br/>        “你的意思是，宗师现在，已经能够算君主级了吗？整个大陆之上，所有的君主可都是一方霸主！”
<br/>    
<br/>        “群战方面，应该算得上是君主级了。”
<br/>    
<br/>        苏青阳忽然道：“那既然宗师在群战方面算得上是君主级，我们还在这儿愣着干嘛，上去找对方的人打一架啊，不然就被宗师杀光了啊！”
<br/>    
<br/>        雪自悟也是道：“等于说，宗师在这三天里布置了那么久，他是在和空气斗智斗勇，恐怕他自己都没有意识到他在群战方面已经算得上是君主级了吧？”
<br/>    
<br/>        苏青阳挥手道：“你们山主正在和妖城之中三十万妖军一战，我等也不能坐视旁观，听我命令，进攻！”
<br/>    
<br/>        陈明这边，陈明将一众敢向他进攻的妖王已经屠杀殆尽，剩下的，已经是在想办法逃离这座妖城了，陈明也没有继续下去，这个法力消耗太快了。
<br/>    
<br/>        看似猛地不行，实际上确实猛地不行，就是法力消耗是个问题。
<br/>    
<br/>        无数柄灵剑再次回到陈明的万道天竹之中，陈明站在城头之上，睁开双眼，眼中所见，皆是一片狼藉，而在他的身体周围，尸体已经累积成了一个圆圈，只有他身体之外方圆十丈没有尸体，这一次，他是被自己给震撼到了，“原来，这就是大君吗？”
<br/>    
<br/>        “大君，无愧灾难的评级！”
<br/>    
<br/>        这破坏力，绝对是天灾级别的，无法对抗的，才能被称作灾难。
<br/>    
<br/>        不过就是不能撑太久，还是要想办法从这些古仙门里多骗，呸，那个，多讨要几本功法，多修些道宫出来。
<br/>    
<br/>        陈明脚踏剑龙飞上天空，看向剩下的那些妖军，他们现在还是在双眼愣神的状态，还沉浸在死亡和自己近在咫尺的恐惧之中。
<br/>    
<br/>        “这人间炼狱，结束了吗？”
<br/>    
<br/>        “那些飞剑不见了！”
<br/>    
<br/>        “我们，到底在哦对抗一个什么样的存在啊！”
<br/>    
<br/>        “若是有来生，不敢与君为敌。”
<br/>    
<br/>        陈明站在剑龙之上，青衫不染半点鲜血，“苍天有好生之德，我也不是一个喜欢杀戮的人，不想死的，都给我老实在原地待着！”
<br/>    
<br/>        陈明可不想将这些家伙杀完，杀完了自己的弟子去哪儿找兵源去，就算一个弟子召集一千个血衣剑卫，那也需要三百万，自己刚刚一时之间杀了不知道多少，估摸着应该也有几万，还真是可惜啊，这些大妖，将来可都是自己弟子的人。
<br/>    
<br/>        人啊，就是不见棺材不掉泪，不知敬畏。
<br/>    
<br/>        陈明道完，斜腿坐在破碎的城楼上，看向远方已经打到了数十里之外的紫霞仙子和妖族大君，拿出一壶酒，仰头饮下一口，而后回头看了一眼，这些大妖还愣在原地呢，过了一会儿，终于是有大妖醒转过来，不过他们看了城头上坐着的陈明一眼，顿时浑身无力瘫坐在了地面上。
<br/>    
<br/>        燕山山主，陈明，在他们的眼中根本就是无法战胜的魔鬼，至于原因，很简单啊，你连他人都没看到，自己就没了.......
<br/>    
<br/>        在这一天，他们终于是回想起了曾经听说过的那句话，大阵之下，众生平等。
<br/>    
<br/>        这句话，原来从来不是虚言。
<br/>    
<br/>        等到苏青阳和雪自悟赶到现场的时候，陈明已经坐在城头上开始饮酒了，雪自悟抬头道：“宗师，面对数十万大军还可以如此镇定的饮酒，当真是有仙人风范！”
<br/>    
<br/>        苏青阳看了城中那些面如死灰，已经吓得浑身发凉的大妖们，也是道：“弹指之间千军辟易，唯有宗师做的到！”
<br/>    
<br/>        就在这个时候，两人忽然听到噗的一声，抬头一看，陈明将口中的酒水直接喷了出去，然后他们就听到了陈明的骂骂咧咧，“到底是谁想出来酒这么个玩意儿的，辣的要死！”
<br/>    
<br/>        然后陈明摸出了一瓶陈灵鱼制作的果汁，饮了一口，“嗯，还是果汁好喝！”
<br/>    
<br/>        苏青阳和雪自悟对视了一眼，不敢冒犯陈明，最后也只能无奈道：“宗师还真是洒脱啊，放荡不羁！”
