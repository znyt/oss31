　　第448章
<br/>　　“皇侄莫慌，吾来助你！”
<br/>　　李乐天与林少羽正杀得难解难分，忽听得一声大叫，李星斜刺里冲杀过来。
<br/>　　林少羽自家人知道自家情况，无论是他，还是起义军其实都以撑不下去。抱着必死的决心，就是要杀人。
<br/>　　见李星冲过来，他大吼一声，以一杆枪硬扛两个人。
<br/>　　李星算是皇族的分支，驻守扬州，本来他是没资格觊觎皇位的。但明帝的死，给了他机会。无论是李乐天还是李行哉，空顶着一个皇子的名头，但他们的软肋一模一样，都是手里无兵。
<br/>　　可李星手中却有精锐的扬州兵，而且李星这个人，名声一直都不错，在帝国也有人心。所以他觊觎这王位，并非是痴心妄想，还是有很大机会的。
<br/>　　恰好李星和李乐天一样，用的都是盘龙枪。
<br/>　　现在是两杆枪挑战，围着林少羽厮杀。以一敌二，林少羽竟然丝毫不落下风，一杆铁枪或刺或挑，竟然逼得李星和李乐天只有防守之力，而无还手之功。
<br/>　　林少羽的实力本就大不如从前，但对战二人却依旧是行云流水，随意刺出一枪，就逼的二人狼狈不堪。
<br/>　　现在还如此，若他全盛之时，怕早就一枪一个，将李乐天与李星挑落。
<br/>　　帝国将军尉迟离看得仔细，知道如此下去，李乐天二人迟早落败，且有性命之忧。假若李星被杀死，尉迟离半点悲伤的情绪也不会有，怕还是要敲锣打鼓庆祝。但如果李乐天被杀死，他可就失去了手中重要的筹码。
<br/>　　正此时，又有一匹战马从帝国军中冲了出去。他身穿九龙缠纹甲，手持亮银枪，座下一匹白龙马，在初升的阳光下，白盔白甲白马，模样当真俊到极致。
<br/>　　李行哉终于出战。
<br/>　　他一杆枪递出，拦住了林少羽刺向李乐天的一枪，李乐天勒马止步，向他看了一眼，互相点点头，却是没有言语。
<br/>　　李乐天和李行哉的关系一直不怎么对付，这也正常，一个矜矜业业，做事一板一眼，一个花天酒地，放浪形骸，关系能好才怪呢。
<br/>　　李行哉用的也是八打盘龙枪，他的天资不如何，也不肯下功夫，所以八打盘龙枪只会一招。所以，帝国上层将李行哉看做废物，也并不是没有道理。
<br/>　　李乐天是能文能武，文武全才，但李行哉是学问不行，学武也不行。不过也不能说他不聪明，至少喝花酒唱小曲儿他内行的很。
<br/>　　而李行哉唯一会的一招枪法是：化龙点睛。
<br/>　　本来林少羽出招行云流水，不管是李星还是李乐天，都无法对他产生威胁，只不过二人守得紧，他一时半会还找不到杀人的机会。
<br/>　　但李行哉的出现，让林少羽的情况顿时紧张起来。原因不在李行哉，而在于盘龙枪的画龙点睛。
<br/>　　这招『画龙点睛』委实太变态些。
<br/>　　盘龙枪一共八打：青龙甩尾，白龙入云，黑龙砸水，离龙吐火……但画龙点睛却是盘龙枪的要诀。不会这招，盘龙八打不过就是一条墙上的死龙，但画龙点睛之后，龙却可以吞云吐雾，变化无穷。
<br/>　　当然，画龙点睛是其中最难学的一招。
<br/>　　盘龙枪是李氏皇族的家传枪法，所有皇族子弟都可以学。学的人有很多，肯吃苦的也有，但在这一代子弟中，不太肯吃苦的李行哉却是唯一一个掌握的。
<br/>　　他往日没机会出手，只会一招画龙点睛，其实也没多大用。但今日在李星与李乐天的配合下，天时地利与人和，他却能将画龙点睛的威力发挥到极致。
<br/>　　即便林少羽知道这一招的破法，也无可奈何，因为他要攻击李行哉时，李星与李乐天的两杆枪绝饶不了他。可是当他想要攻击李星或李乐天时，李行哉的枪却又如跗骨之蛆，死缠着不放。
<br/>　　李星、李乐天、李行哉三人一脉相承，体内流淌着同样的血。今天是第一次联手对敌，却是越打越默契，越打杀意越盛。
<br/>　　三人分属不同阵营，各自有各自的麻烦，对于这个帝国，各自有各自的想法。但如今三杆枪缠斗林少羽，三人似乎把这一切都抛在脑后，心中只想着一件事：
<br/>　　李家的仇当然要李家人来报！
<br/>　　林少羽出阵时何其狂妄，一口气枪挑三将，但如今却被限制得死死的。
<br/>　　不管是相府，还是将军府……在场的各路诸侯，都发出一阵阵欢呼。有壮汉捶鼓，鼓声喧天而起，士兵的士气高涨。
<br/>　　尉迟离扶着胡须，长出一口气，吐出胸中的阴霾。
<br/>　　“几多年了，没想到有生之年，还能在沙场见到盘龙八打。”
<br/>　　“将军，怕是忘了一件事。”尉迟离身边一军师模样的人道：“当年武帝打天下，用一杆枪横扫天下高手，打下一百零八座城池都姓李。但今天，李家三人联手，才能将盘龙八打学个皮毛。”
<br/>　　尉迟离微微皱眉，半晌轻轻点了点头。真龙已死，徒留些龙骨龙鳞让后人炫耀。后人想效仿祖先屠龙的事，也只是学个模样而已。
<br/>　　如果今天的李家有人学到当初李蛮子一半，林少羽早被一枪杀了，何至于三人联手，三十几招都分不出胜负。
<br/>　　尉迟离收拾心情，轻轻的抬起了手掌，只要情形有半点不对，他就立刻下令出兵，抢回李乐天。能不能杀林少羽他不着急，但李乐天决不能有事。
<br/>　　其他相府，扬州兵也是同样打算，都盯着场上的混战目不转睛。
<br/>　　之所以如此，是因为李乐天三人在场上，已经没有任何优势。
<br/>　　刚开始时，林少羽的确被限制住了。但他毕竟是绝世武将，所谓绝世，就是全天下人都算上，伸出一支手数，一支手都用不完的那种程度。
<br/>　　他可以从没有路的时候，走出路来。
<br/>　　他卖了自己一条胳膊，左肩被李乐天刺穿，右手却一枪将李星挑落马下，三人配合的局面，就此被打破。
<br/>　　只要再给林少羽一个机会，不管是马上的李行哉，李乐天，还是坠马的李星，最后都要死。
<br/>　　“杀呀！”
<br/>　　帝国军开始冲锋。
<br/>(本章完)
