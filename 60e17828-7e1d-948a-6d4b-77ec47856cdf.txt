　　第619章
<br/>　　莫将难想着聚义起事，拉拢道上的人物。这件事已经闹出不小的动静，各方面都在注意这里发生的事情。
<br/>　　程大雷虽然不在蛤蟆城，但蛤蟆城还是照常运转。阿喜负责情报工作，在得到消息后，便派白元飞过来打探情报，看莫将难究竟想搞什么。
<br/>　　白元飞却是没想到，会在这里和程大雷不期而遇。他看到程大雷落在莫将难手中，当场就要营救程大雷，也是程大雷眼神示意他稍安勿躁，他才一直等到现在。
<br/>　　“大当家，我先救你出来。”白元飞道。
<br/>　　“先帮我将绳子割断。其他事我们再等等看。”
<br/>　　白元飞顺着窗户递过来一柄短刀，将程大雷身上的牛筋绳割开一道。
<br/>　　程大雷这才算是摆脱束缚，现在他看上去像是还被严严实实捆着，但只要轻轻一挣就可以挣脱绳索。
<br/>　　“这次山寨来了多少人？”程大雷问道。
<br/>　　“只有我一个，也就是过来看看，只是没想到在这里碰到大当家，话说，大当家怎么落到这个地步？”
<br/>　　“那个……说来话长，以后有机会慢慢聊。”
<br/>　　“喔，我想定是大当家为了打探情报，故意被他们擒住，不然的话，凭他们的本事，怎么捉的住大当家？”
<br/>　　“哈，哈……还真被你猜对了。”
<br/>　　程大雷干笑两声，这时候有人往庙里来，程大雷立刻道：“你先躲一躲，等下见机行事。”
<br/>　　白元飞身影一闪，已经从窗外消失。这时候，外面的战斗差不多已经结束，柳追风一行人在多数人的围攻下，可以说伤亡惨重。他带来的同伙死了两个，其他人身上都挂了彩，尤其是柳追风更是伤得严重。
<br/>　　在稳住形势后，莫将难又令人将程大雷从庙里押了出来。
<br/>　　他握着刀，阴森森冷笑。既然见了人命，这件事就不可能善罢甘休，拉拢柳追风的事再也不用想，现在除了杀了他，再没有别的办法。
<br/>　　“柳追风，死到临头，你还有什么想说的？”
<br/>　　柳追风身上伤痕累累，双手握着剑如一头困兽。他满脸血污，口中吼道：“姓莫的，你拿着刀想要吓唬谁。大丈夫要杀就杀，我皱一皱眉头是你养的。”
<br/>　　“柳兄是个英雄人物，我就给你一个痛快。”莫将难冷哼一声，目光放在周桃身上：“小桃，你现在过来，我还可以原谅你？”
<br/>　　刚才周桃选择柳追风，未必没有与莫将难赌气的意思，她对柳追风还是利用的成分居多。但却是没想到，柳追风竟能为她做到这一步。
<br/>　　人非草木，孰能无情，周桃毕竟也不是铁石心肠。
<br/>　　她忽然笑了一声，往前跨出一步，道：“莫郎，我又怎肯负你。”
<br/>　　莫将难伸出手：“来。”
<br/>　　周桃走向莫将难，也没人阻拦，莫将难脸上露出得意的笑容，示威般看了柳追风一眼。
<br/>　　这一刻，柳追风心碎如死，但转念一想：若她能活下来，也未必是一件坏事。
<br/>　　周桃款步走向莫将难，所有人的注意力都落在她身上。看着她的倩影，不知有多少人心中一荡，想的是：不怪柳追风为她一往情深，这个女人的确是个尤物。
<br/>　　在要靠近莫将难的同时，忽然寒光一闪，从周桃袖口中划出一柄匕首。莫将难没有防备这一招，慌张用手去挡，胳膊被划出一道血淋淋的伤口。
<br/>　　莫将难怒急，他没想到周桃会这么做。就像自己养的小猫，突然抓伤自己。怒不可遏的他一掌拍过去，周桃踉跄后退，只退到三步外。
<br/>　　“尔敢！”恼羞成怒的莫将难喝道。
<br/>　　周桃冷笑一声，忽然握紧匕首，向自己胸口扎来。
<br/>　　柳追风没想到事情会演变到这个地步，他想扑过去相救周桃，可他所在的距离已经来不及。只能眼睁睁看着匕首扎向周桃胸口。
<br/>　　啪。
<br/>　　一团绳索打在周桃脸上，将周桃的匕首砸在地上，所有人都是一怔，尤其是周桃抬起头，诧异的看着绳索丢来的方向。
<br/>　　程大雷晃了晃肩膀，活动着麻木的胳膊。
<br/>　　“自己的性命，姑娘不该这么不珍惜。”
<br/>　　与程大雷离得近的几个，立刻与他拉开距离，以程大雷为中心，周围散开一个圈。
<br/>　　“你……你怎么做到的？”莫将难难以置信道，他想不通程大雷如何能挣脱绳索。
<br/>　　“哈哈哈。”程大雷仰天大笑：“本当家是何等样人，凭你们用一根绳子也能捆住。本当家只是想陪你们耍耍，现在戏看够了，一切也都该结束了吧。”
<br/>　　如猛虎出闸相仿，诸人都心惊胆战，这魔头已经摆脱束缚，谁还奈何得了他。
<br/>　　有些胆子小的，已经悄悄往外撤，最好程大雷没有看到大家的样子，不然以他的小肚鸡肠，估计不会放过这件事。
<br/>　　而莫将难腿肚子却是一阵阵抽筋，对于程大雷实在该早早杀掉的，留到现在果然就成了噩梦。
<br/>　　“程当家，这是一场误会，误会呐。”
<br/>　　“误会么？”程大雷冷冷道：“我若是摘掉你的脑袋，用不用对你的尸体说声误会。”
<br/>　　明知今天伸脖子一刀缩脖子也是一刀，莫将难突然发了狠，大吼道：“弟兄们与我上，他也不是三头六臂，大家一拥而上杀掉他。”
<br/>　　程大雷冷哼一声，手伸到一旁：“剑来。”
<br/>　　柳追风将自己佩剑抛给程大雷，程大雷单手接过剑，在身前一横，口中道：“那个来送死？”
<br/>　　大家都知道手中有剑的程大雷和手中无剑的程大雷实力差别很大。看着他横剑身前的模样，竟无一人敢上前。
<br/>　　最后还是貌不惊人的韩醒上前一步，口中道一声『领教了』，便握着刀向程大雷攻来。
<br/>　　程大雷无可奈何的叹口气，星芒一闪，剑从空中滑过，摇摇落下一滴血来。
<br/>　　韩醒的身子忽然顿住，脖子上出现了一点血红，随之血便窜了出来，他的身体重重倒地。
<br/>　　柳追风倒抽一口凉气，他自觉在剑道上也有些造诣，可与韩醒交手数个回合，也占不到半点上风。却是没想到，韩醒竟然连程大雷一剑都没有挡住。
<br/>　　盛名之下无虚士呐。
<br/>(本章完)
