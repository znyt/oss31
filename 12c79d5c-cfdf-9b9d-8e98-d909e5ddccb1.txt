　　大当家的一见到人形煞，立刻就跟被勾了魂似的，各种喜欢，那人形煞也很聪明，哄得大当家的非常开心，大当家立刻就决定要去娶她做压寨夫人。
<br/>　　本来按照土匪的规矩，没有发丧一说，因为经常死人，不管死了谁都是直接埋了，但是大当家的念着三当家拼死送回来一位貌美如花的压寨夫人，以为忠心耿耿，这才决定发丧。但因为他已经迫不及待地想要洞房，干脆就直接发丧，打发了三当家的尸体。
<br/>　　山上每次过节都要吹唢呐庆祝，因此直接找人扮成道士，吹吹打打，为三当家发了丧，而眼前这两个家伙，就是打发下山去买一些红烛红纸红盖头之类的婚礼用品回来，虽然土匪没什么将就，但大当家的喜欢那人形煞，想要做做样子，同时也算是借机会让兄弟们在一起乐呵乐呵。
<br/>　　得知这些情况之后，叶少阳又打听了山上的情况，得知这二龙山上一共有大小土匪两三百个，一共有三个当家的，老三被自己打死了，老大外号叫鬼见愁，自称是少林寺下山的，身上有些务武功，七八年前来到这二龙山的，当时聚集了一些地痞无赖，在这里开路收钱，后来吸收的流民越来越多，给江西境内最大的土匪头子刘司令送了一份大礼，于是就开了堂口，在这二龙山占山为王，打家劫舍。
<br/>　　江西境内一直都不太平，官方根本就没工夫打击这些土匪，于是二龙山入伙的土匪越来越多，势力也渐渐大了起来，两三百人的队伍，在江西省境内算是近乎中等的实力了。
<br/>　　二当家的，是个前清的落地书生，本来是在附近一个县城里的当铺当账房先生，因为暗中扣了当铺的东西，被老板羞辱了一顿，一怒之下上二龙山当了土匪，本来他一个文弱书生，当土匪是不够料的，不过这个先生号称铁算盘，不光算盘打的好，处理任何事情都井井有条，十分稳重，这正是这些大老粗们所缺乏的。
<br/>　　铁算盘上山之后，给鬼见愁出了不少好主意，每次绑票买卖都出谋划策，十分得力，因此被鬼见愁破格提升为二当家的。
<br/>　　叶少阳听到这里，心中思索起来，在他们谈话之中，他已经听出，这个什么鬼见愁，就是个大老粗，至于他会的那点工夫，自己根本就不放在眼里。
<br/>　　他真正有些在意的是那个铁算盘，无数次经验告诉他，真正难对付的反而是这种人。于是跟眼前这两个土匪打听起铁算盘的为人。
<br/>　　这两个土匪现在小命捏在他手里，也没有什么道义不道义的了，你争我抢地告诉叶少阳相关的情况，说了很多，叶少阳最印象深刻的是一件事：
<br/>　　这个铁算盘一直对之前羞辱他的那位东家怀恨在心，当他在二龙山上站稳脚跟之后，立刻就撺掇鬼见愁，策划了一次绑票行动，绑了之前那个老板的儿子，要挟老板给钱，在得到钱之后，还是撕票了……
<br/>　　叶少阳听到这里，拳头握的咯吱响，要说盗亦有道的土匪，他相信是有的，但是这伙二龙山的土匪不是，绝对不是！
<br/>　　缓了半天，叶少阳才冷静下来，又打听了一些山寨里的情况，然后开始琢磨怎么样混进山寨，仔细想了想，还是得晚上上山，这也不是一百年后，山寨里没摄像头，也没有探照灯之类的，而且那时候山上大摆宴席，防御肯定会松懈，自己就等他们喝醉酒的时候摸进去，多半能行。
<br/>　　而且，那个人形煞也要在晚上才会化形嗜血，白天一般不会怎么样。
<br/>　　在彻底问清楚山寨里的情况之后，叶少阳开始琢磨这两个土匪怎么办，不杀吧，这两个土匪不是什么好东西，手上都沾了不少血，叶少阳很想直接杀了，但自己只是个法师，没资格行使人间法律，二来这俩家伙也回答了自己不少问题，就这么杀了，总有点下不去手。
<br/>　　想了半天，叶少阳有了主意，说道：“这样吧，我不杀你们，但你们要配合我，你们看怎么样？”
<br/>　　在手枪的威胁下，能活命都是最大的愿望，两个人哪敢说不干，当下用力点头，表示一定听话。
<br/>　　让两个土匪都把上衣脱下来，从后面各自绑住了他们的双手，为了捆结实一点，又从附近树上扯下了一些藤条，给他们捆了个结结实实。
<br/>　　“大爷，您您您这是干啥？”两个土匪不敢乱动，但还是忍不住问道。
<br/>　　“我是不想杀你们，这样吧，我把你们交给警察，就当你们去自首了，你们看怎么样？”叶少阳玩着手里的枪说道。
<br/>　　两人傻傻地看着他。
<br/>　　叶少阳道：“怎么，这已经算是优待处理了，不乐意啊？”
<br/>　　“那个，您说……什么茶，什么手？”
<br/>　　“呃，我的错。”叶少阳用力拍了拍脑袋，“我忘了你们这个时代没有警察这个词，好像也没有自首……我的意思，是把你们去送官。”
<br/>　　“啊，大爷不要啊！”两人一听说要把他们送官，立刻就瘫了，跪在地上一个劲地磕头，恳求叶少阳不要这么做。
<br/>　　“放心啊，你们是自首……我就说你们是悔过，不想再当土匪为虎作伥，到时候你们把二龙山所有的事情交代清楚，也算是戴罪立功，肯定不会杀头的，再说你们也只是喽啰，很多罪行你们并不是主犯，你们不答应也行，我直接一枪崩了你们，省的麻烦，你们觉得呢？”
<br/>　　两个土匪当场就傻眼了，对了一下眼色，没奈何，能保住一条命的话，总比死了的强，只好点头答应。
<br/>　　这个问题是解决了，怎么下山又成了麻烦事：两个土匪都被绑起来了，都没办法骑马。叶少阳问他们这里离最近的县城有多远，回答是二十多里地。
<br/>　　叶少阳一想，反正自己要天黑才能上山，干脆就步行下山，然后再骑马上来，二十里下山路，绝对赶的回来。
<br/>　　于是把一匹马解了马鞍和笼头，在马屁股上拍了一巴掌，说道：“跑吧，自己在山野间过活，看你造化了！”
