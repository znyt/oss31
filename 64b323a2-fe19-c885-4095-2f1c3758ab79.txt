　　鞠文泰这货说起来也是个奇葩，历史上这货坚信西突厥能够战胜大唐，所以经常在丝绸之路上抢劫大唐商旅，甚至放言，大唐根本无力将军队派到七千里之外的瀚海之中。
<br/>　　结果，他的行为终于把李二给惹毛了，放出候君集这头恶犬。
<br/>　　然后最戏剧性的一幕就出现了，当老候带着军队出现在高昌城外时，西突厥竟然夹着尾巴远遁三千里，把这货彻底的丢给了老候，
<br/>　　而鞠文泰则因为忧惧，在老候叫骂的时候，就生生的把自己给吓死在了王座之上。
<br/>　　不过，现在这一切还没有发生，鞠文泰还是在当着他的太平国王，只是那种目中无唐的架式已经开始显露。
<br/>　　“念念，让孤听听，那些贱民都说些什么。”将舞姬们打发走，鞠文泰大咧咧的说道。
<br/>　　“大王，那些马贼想要些粮食。”王宫护卫犹豫了一下，最终没有将信的内容念出来，只是简要的说了一下信上的要求。
<br/>　　“孤让你念就念，孤很想听听！”其实只要是个人都知道，相打无好手，相骂无好口，马贼叫城哪里会有什么好话，可偏偏鞠文泰不知脑子里哪根筋抽了，非要听听信的内容。
<br/>　　“是！”护卫没有办法，想了想，最后把信递给了丞相：“丞相，小的不识字，还是麻烦您吧！”
<br/>　　该死的！丞相在心里狠狠骂了护卫一句，这个该死的家伙，必然是因为信中内容极不恭敬，怕念了之后被鞠文泰责怪，所以才装成不识字，把信交给自己的，否则如果真不识字，怎么可能知道信上的要求是什么。
<br/>　　但是没办法啊，护卫可以用不识字当借口，他却不能，所以只能把信展开，犹豫了半天念道：
<br/>　　字喻高昌鞠文泰小儿：
<br/>　　速速献上粮食万石，以充爷爷们的军粮，否则爷爷剿灭西突厥顽寇，破尔城池，灭尔满门！
<br/>　　大漠马贼·高展留字。
<br/>　　“放肆！”“哐……”
<br/>　　果然，信刚刚念完，找骂的鞠文泰就被气的火冒三丈，手中的玉石酒樽狠狠砸向丞相，但被时时注意的丞相躲了过去。
<br/>　　那东西纵然漂亮可依旧改变不了石头的本质，而且还有棱有角，如果砸到头上那可是非死即伤，只有傻子才不会躲。
<br/>　　“出兵，给孤出兵剿灭他们！”鞠文泰狠狠的拍着身边的矮几，愤怒的咆哮着。
<br/>　　“大王，不行啊，不能开城，外面的马贼人数下万人，如果开城，被他们冲进来，后果不堪设想！”丞相连声劝阻，试图让鞠文泰收回成命。
<br/>　　鞠文泰在城里不知道，可是丞相刚刚却上过城头，知道外面那伙马贼数量的可怕。
<br/>　　而且那可是马贼，不是什么讲道理的人物，真要是冲进城，烧杀一阵，就算最后能被剿灭，估计也会对高昌造成不可挽回的损失。
<br/>　　鞠文泰从丞相的眼中看到了恐惧，也有了一丝犹豫，但还是咬牙问道：“难道你让孤就这么忍了？”
<br/>　　“大王，外面这伙儿马贼没有打出旗号，但如果臣没有猜错，他们应该就是西突厥想要剿灭的那一股，所以，臣窃以为不如等西突厥的大军到了之后再作决断。”丞相小心试探着，生怕鞠文泰再丢什么东西过来。
<br/>　　“你是说那股最大的马贼？他们不是很少骚扰城邦么？”鞠文泰面色一变，隐隐心中有一丝担忧。
<br/>　　“大王，马贼大部分都乱的很，不打出旗号，很难确定他们的身份。”丞相为自己的猜测辩解了一下，如果将来确定这伙马贼不是最大的那一股，也好减轻一点自己的责任。
<br/>　　“西突厥的军队到什么地方了？”丞相的劝解之下，鞠文泰打消了出兵的念头。
<br/>　　而实际上他也不想出兵，刚刚的架式也不过就是作个样子而以。
<br/>　　现在丞相给他一个台阶，自然他就顺着台阶下来了，否则坚持下去真要出兵，他也有些舍不得。
<br/>　　必竟西域地广人稀，征兵不易，死的多了也是一件让人十分心疼的事情。
<br/>　　“按计划，大概还有四、五天的路程就会到我们这里了。”丞相见把鞠文泰劝住了，大大的松了一口气。
<br/>　　“那就紧闭城门吧，一切交给西突厥去解决。”鞠文泰眯着眼睛想了一下：“时刻注意马贼的动向，如果他们不攻城，就由他们去吧。”
<br/>　　“是，臣告退！”丞相领命之后转身离开，这破地方，他一分钟都不想多待。
<br/>　　而此时在城外，薛仁贵等人也正在观察着高昌城的反应情况。
<br/>　　待看到城头上密布的士兵之后，薛仁贵将手中的望远镜递给白文墨：“先生看看？”
<br/>　　白文墨也不客气，接过望远镜向对面城头看去，片刻之后说道：“将军之策成功了，想那鞠文泰应该在近期应该不会派兵出城了。”
<br/>　　“必须确保万无一失才好，否则我们与西突厥交战之时被他们从后成捅上刀，那才是大大的失策。”薛仁贵面色有些凝重。
<br/>　　“怕什么，我带人过去，把火药全给它埋到城门口，让他们出来多少死多少。”高展抽着鼻子说道。
<br/>　　这家伙前两天吃坏了肚子，一晚出恭十多次，第二天肚子好了，人却被夜上的寒冷冻感冒了。
<br/>　　“不要大意，用好防护！”白文墨这几天算是见到了高展的不靠谱，闻言忍不住叮嘱道。
<br/>　　“放心吧，能伤到咱们‘獠牙’的箭现在还没造出来呢。”高展大咧咧一笑，拨马便走，去招乎他的手下执行任务。
<br/>　　“先生，我们也回去吧，看看投石机造的怎么样了。”薛仁贵再次向城头看了眼，转对白文墨说道。
<br/>　　白文墨点点头：“也好，让那些学生负责这个事情，老夫也不甚放心。”
<br/>　　“嘿嘿，先生不放心那些学生，是怕他们被欺负了吧？”董建森嘿嘿怪笑着打趣白文墨。
<br/>　　“有尉迟将军在哪里，老夫道是不怕他们被欺。”老白笑着摇摇头，接着说道：“老夫是怕那些学生异想天开的瞎折腾，到时候上了战场贻误战机那就不美了！”
<br/>(本章完)
