　　拿了猎妖令，萧逸全速赶路，回了萧家一趟，将子令交于大长老。
<br/>　　子令会自动记录大长老的气息，而后彼此气息相连。
<br/>　　一旦大长老遭受攻击或者受伤，子令都会自动起反应，显露出来警示敌人，同时通知附近猎妖殿的执法队。
<br/>　　和大长老的谈话中得知，如今萧家暂时安好。
<br/>　　萧逸并未透漏自己就是易霄，也并未提及这几个月发生的事。一来是赶时间离开，二来是不想大长老他们担心。
<br/>　　事实上，萧逸真正担心的是北山慕容家，而非暗影楼。
<br/>　　毕竟，自己和钟无忧一战，暗影楼就算要报复的，也是易霄。
<br/>　　反倒是北山慕容家这种庞然大物，虽说他们重名声，但凡事都有意外。
<br/>　　萧逸不敢赌，在北山慕容家面前，萧家太脆弱了，挥挥手就会被灭掉。
<br/>　　如今，有了猎妖令的存在，自己倒是能暂时安心了。起码可以让萧家得到一年的安全时间。
<br/>　　嘱咐好大长老随身佩戴子令后，大致聊了几句，萧逸便再次离开了紫云城。
<br/>　　.....
<br/>　　几乎是在萧逸刚离开紫云城不久，破玄城猎妖殿的工作人员便有了感应。
<br/>　　紫云城，正是在破玄城十数城之内。破玄城猎妖殿，便是距离紫云城最近的猎妖殿。
<br/>　　“总执事。”
<br/>　　猎妖殿内部，总执事的房内，一个执法队武者神色着急地闯了进来。
<br/>　　“何事如此匆忙？”说话的是一个中年男子，稳如泰山般坐着，批阅着文件。见手下匆忙闯入，眉宇间显然露出了一丝不悦。
<br/>　　他便是破玄城猎妖殿的总执事，一位破玄六重的武者。
<br/>　　“总执事，附近出现了殿主级别猎妖令的气息，而且子令已经认主。”执法队武者立刻报告道。
<br/>　　“殿主级别的猎妖令？”总执事放下手头的文件，眉头皱了起来，“整个北山郡，有本事请动殿主级别猎妖令的，可没有几个。”
<br/>　　“受庇护的是家族还是个人？”总执事沉声问道。
<br/>　　“暂时不知，可需属下去查探清楚？”执法队武者恭敬地问道。
<br/>　　“不必。”总执事摇摇头，说道，“这个级别的猎妖令，只有分殿主能细查个中情况，我们没有权限。”
<br/>　　“派两支十二人的执法队，长期驻扎在子令气息百里范围之内。”
<br/>　　“不可打扰受庇护的家族或者个人。”
<br/>　　“若子令稍微有异动，立刻前往支援；若子令气息剧烈，立即发信号回猎妖殿，我会亲自带人过去。”
<br/>　　执法队武者闻言，顿时一惊，“总执事，两支十二人的执法队？还是长期驻扎？”
<br/>　　执法队也分级别。
<br/>　　六人队伍，队员半步洞玄修为，队长洞玄九重修为。
<br/>　　十二人队伍，队员清一色洞玄五重以上，队长半步破玄。
<br/>　　而且，执法队作为猎妖殿的直属力量，可并没有明显上那么闲，每一个都身肩任务，忙得很。
<br/>　　“照我吩咐即可。”总执事呵斥一声。
<br/>　　“是。”执法队武者应答一声，而后下去安排。
<br/>　　......
<br/>　　紫云城，距离百武城差不多四十余城，路途极远，逾万里之遥。
<br/>　　若是洞玄境武者，御空飞行，日行数千里轻而易举，数日就能轻松赶万里路了。
<br/>　　可萧逸只有先天三重。
<br/>　　幸好，凭着媲美先天九重妖兽的肉体，配合真气和‘豹形’。
<br/>　　加上灵器提供的源源不断真气，让他赶路的速度仅比洞玄境御空飞行慢数分而已。
<br/>　　恰好用了十二天，跑了一个来回。
<br/>　　也亏得他的肉体力量强悍，这十二天几乎不眠不休地全速赶路，竟没把他累倒。
<br/>　　当然，当他回到百武城时，也是累得够呛，满脸憔悴。
<br/>　　“呼。”萧逸深呼吸一口，自语道，“还有三天，现在赶去裂天剑派刚刚好。”
<br/>　　这时，忽然有数个猎妖师在他身边经过，兴致勃勃地谈论着什么。
<br/>　　“喂，听说了吗？白家最近在到处寻找炼药师呢。”
<br/>　　“当然知道，想要炼制雪魄丹嘛。不过这个级别的丹药，四品炼药师都不一定能炼制出来。”
<br/>　　“要不然白家怎么会到猎妖殿发布如此高额赏金的任务，让猎妖师帮忙找炼药师呢。”
<br/>　　“走，去四季城凑凑热闹也好。”
<br/>　　“....”
<br/>　　这数个猎妖师离去，萧逸却若有所思地站在原地。
<br/>　　“四季城白家要找炼药师？”萧逸喃喃自语。
<br/>　　裂天剑派，乃是北山郡武道圣地。
<br/>　　在其门派方圆三百里内，禁止一切势力的存在。
<br/>　　包括北山慕容家的盛宝商行，包括猎妖殿的分殿，一概不准出现。
<br/>　　而在三百里之外，便是四季城。
<br/>　　可以说，四季城乃是最接近裂天剑派的一座大城。
<br/>　　城内终年聚集着大量武者，一来是仰慕裂天剑派，二来就算无缘进入裂天剑派修炼，但若是有幸遇到裂天剑派的长老或者天才弟子，讨教几分，增长武道知识，也是极好的。
<br/>　　数量众多的武者，也让四季城应运而生的出现了许多势力和商行。
<br/>　　无论春夏秋冬，一年四季，城内总是武者熙攘，繁华至极，故称为四季城。
<br/>　　在这众多势力中，又以白家最为厉害。
<br/>　　同时，白家家主便是四季城城主。
<br/>　　“反正去裂天剑派是要经过四季城的，顺便把任务接下来也好。”萧逸暗暗想罢，去了一趟猎妖殿。
<br/>　　然而，当他真正看到任务时，却被吓了一跳。
<br/>　　任务的大致描述是，寻找四品以上炼药师，炼制雪魄丹。
<br/>　　任务的赏金高达五十万两，列为五级任务，任务点100。
<br/>　　再看了眼任务期限，三天。
<br/>　　“三天？白家想在这么短的时间内找到能炼制出雪魄丹的炼药师，看来很急阿，难怪要出高价在猎妖殿发任务。”
<br/>　　萧逸暗暗地想着。
<br/>　　猎妖殿内的悬赏任务，有的是只能一个人接，接了之后，任务栏上就会消失掉这个任务。
<br/>　　防止多个猎妖师做这个任务，出现纠纷。
<br/>　　就像清剿山贼。
<br/>　　有的是谁都能接，最后看谁先完成任务，任务点和赏金便归谁。
<br/>　　就像白家的这个任务。
<br/>　　而最让人狂热的，并非是这五十万两。
<br/>　　而是，白家承诺，谁能炼制出雪魄丹，除了赏金外，还能许一个要求。
<br/>　　只要是白家能力范围内的，任何要求都可以。
<br/>　　作为四季城的霸主势力，其势力之大，在整个北山郡都是排得上号的。
<br/>　　白家承诺的一个要求，可比任何东西都要值钱。
<br/>　　“任何要求。”萧逸眼前一亮。
<br/>　　离开猎妖殿，萧逸继续赶路，往四季城而去。
<br/>　　以他的速度，配合豹形，千米距离，转瞬便到；百里距离，也不过是半个时辰左右。
<br/>　　到达四季城，恰好用了半天。
<br/>　　“呼呼...咳咳。”萧逸来到四季城之下，喘起了粗气。
<br/>　　之前连续十二天的赶路，中途也会休息一下，大概每跑数天便休息一两个时辰。
<br/>　　但即使如此，还是对他的肉体造成了一定的负担。
<br/>　　要是别的武者，根本做不到这样赶路，顶多几天就累趴下了。
<br/>　　也就是他修炼了修罗战体第一层圆满，肉体太过强悍，才能承受得住。
<br/>　　“呼...”数十秒后，萧逸停下了喘气，愤愤地说道，“待得这些事了结，非睡他个天昏地暗不可。”
