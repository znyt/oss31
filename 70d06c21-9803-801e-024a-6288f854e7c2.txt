　　　　“蓬！”
<br/>　　在青云门弟子目瞪口呆的模样中，齐昊身躯倒飞出擂台，凌空几秒后，狠狠的砸在玉石铺就的地面上。
<br/>　　“铿锵！”
<br/>　　寒冰剑掉落在地面，发出清脆的声响，剑身上浮现一条细小的裂痕。
<br/>　　齐昊宛如死狗一样趴在地上，口中不断溢出鲜血，连一根手指都不得动弹，视线也逐渐变得模糊起来，望着擂台上的身影，眼中满是怨毒，气血攻心下，脑袋一歪，直接晕死过去。
<br/>　　回过神来，擂台上的苍松道人骤然色变，瞬间来到擂台下，握着齐昊的手腕查看起来。
<br/>　　五脏六腑移位，治疗修养几个月就能恢复，重要的是地上的寒冰剑，上面一丝裂痕清晰可见，若是再次使用，断裂的可能性很高。
<br/>　　“......你三番五次废我门下弟子，当真好狠的心，如此歹毒，迟早要坠入魔道，今日留不得你。”
<br/>　　说罢，苍松道人凝出一道剑气向慕清霄射去。
<br/>　　青云门周围弟子纷纷惊骇欲绝，如潮水般向擂台四周退去。
<br/>　　“轰！”
<br/>　　霎时间，擂台上响起一声震耳欲聋的轰鸣，只见一道身影握着一柄红色仙剑，挡在慕清霄面前，化去剑气。
<br/>　　场面一触即发，田不易剑指苍松道人，怒喝道：“苍松，你什么意思！”
<br/>　　“什么意思，你们都没看见吗，他对同门师兄都能下此毒手，往后若是修道有成，说不定会做出危害青云之事......”
<br/>　　“苍松！”
<br/>　　苍松道人话语还未落下，便被田不易厉声打断，道：“到底谁心肠歹毒你心里不清楚吗，想要动我的弟子，得先问过我手中的‘赤灵’。”
<br/>　　苍松道人眼色变幻不定，他自然知道齐昊在暗中做了手脚对付慕清霄，只是没有点破而已。
<br/>　　霎时间，整个云海广场的气氛都变得压抑起来。
<br/>　　陆雪琪紧紧握着手中的‘天琊’，面无血色，刚刚苍松道人出手时，真的将她给吓坏了。
<br/>　　水月瞥了眼陆雪琪，叹息一声，道：“雪琪，慕师侄不会有事的，你可以放心。”
<br/>　　“行了，在众弟子面前，你们两个成何体统，苍松师弟，比试出现危险在所难免，弟子间的争斗，让他们自己去解决，我们各脉首座参与其中像什么样？”
<br/>　　道玄真人的话语义正严辞，可任谁都能听出，语气明显偏向于慕清霄。
<br/>　　齐昊现在败了，慕清霄充分的证明了自己的恐怖天赋，还有这一届弟子中，压倒性的力量。
<br/>　　道玄真人不蠢，自然清楚偏向哪边，但齐昊被废也让他极为头疼，不禁开口道：“慕师侄，你屡次触犯门规，但念你初犯，罚你去祖师祠堂面壁半月。”
<br/>　　闻言，慕清霄心中一喜，祖师祠堂不就是第五卷天书古剑‘诛仙’所在么？
<br/>　　原本，他还想着七脉会武过后，冒着风险去一趟祖师祠堂，却没想到瞌睡就有人送枕头。
<br/>　　道玄让他去祖师祠堂面壁思过，他自然没有拒绝的理由，心中暗自欣喜，表面上却平淡无奇，微微行礼道：“谢掌门真人。”
<br/>　　“恩，七脉会武比试过后，便让人领你过去吧。”
<br/>　　道玄真人赞赏的点了点头，刚刚这孩子面对苍松的一击，竟能做到处变不惊，这份心性万万不是齐昊能够比拟的。
<br/>　　无论是心性，又或者是资质都是人中龙凤，未来当真不可限量，旋即目光落在裁判长老身上。
<br/>　　“此战，大竹峰慕清霄胜！”
<br/>　　伴随着裁判长老的高呼声，擂台周围的弟子回过神来，顿时响起一阵惊呼声。
<br/>　　“哎呦，真特么的疼，我居然不是做梦，齐昊师兄居然败了。”
<br/>　　“真是逆天啊，三场比试废了龙首锋三名弟子，居然只需要面壁半个月，我怎么就没有这种待遇。”
<br/>　　“你就别做梦了，慕师兄的天赋也是你能比的？”
<br/>　　“我刚刚看到，齐昊师兄的寒冰剑好像出现了裂痕......”
<br/>　　龙首锋弟子则变得默不作声，看向慕清霄的情绪极为复杂。
<br/>　　“哼！”
<br/>　　苍松道人目光冰冷的瞥了眼台上的慕清霄，拎起齐昊转身就走，大局已定，他现在说什么也没用，本次七脉会武，龙首锋损失惨重而告终。
<br/>　　看了眼手中，死狗一样的齐昊，苍松道人目光眯了眯，看来完重新审视这位首徒了。
<br/>　　待苍松道人离去后，道玄真人，水月大师也相继离去。
<br/>　　待苍松离开，田不易收回仙剑，脸上浮现一丝温和的笑容，拍了拍他的肩膀道：“你小子，连师傅都瞒着，玉清境九重巅峰，真给大竹峰长脸。”
<br/>　　“谢师傅。”
<br/>　　“我们师徒间说什么谢不谢的，但有一点你要注意，你修为虽然精深，在青云门我也能护你周全，但七脉会武过后，你们外出历练，必须要万般小心。”
<br/>　　“恩...”
<br/>　　慕清霄微笑着点了点头，虽然田不易修为没他高，但也传授了他很多的经验，从他刚刚护犊子的表现来看，拜他为师并没有选错。
<br/>　　半日的时间飞速而过，夕阳西下，染红青云之巅，云海广场西侧，大竹峰一众人等汇聚于此。
<br/>　　今日，慕清霄的比试最受瞩目，也是结束最快得一轮，比试结束后，早早的就在西侧等待大竹峰一脉弟子的消息。
<br/>　　宋大仁几人的比试也相继结束，只是身上都伤痕累累，眼神也有些黯然，田不易与苏茹看他们的伤势，皱了皱眉，并没有开口问比试结果，因为不问，结果也一目了然。
<br/>　　最后，只剩下田灵儿与张小凡的比试没有结束。
<br/>　　田不易与苏茹亲情连心，显得有些紧张，但看到远处擂台上，田灵儿道法灵动，丝毫不落下风后，心下也放宽了些。
<br/>　　田不易看了眼自己的妻子，见她神情紧张，拍了拍她的肩膀道：“放轻松些，灵儿没事的。”
<br/>　　苏茹看了自己丈夫一眼，微微一笑，转头又向台上看去。
<br/>　　田不易微微摇头，上午慕清霄的比试可比这凶险的多，自家女儿与之比起来，差距也太大了些。
