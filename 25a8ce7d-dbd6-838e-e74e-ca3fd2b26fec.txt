　　第605章
<br/>　　红马，北风，荒草。
<br/>　　他留给自己脆弱的时间并不太多。消沉过后，自己还将以野原火的身份活着，去做那些早已决定好的事情。
<br/>　　从草坡上长身而起，此刻夜已经黑透。今天出来得足够久，自己也该回去了，再不回去，手下人怕是会找过来。当然，草原这么大，想找到自己也并不容易。
<br/>　　“什么人？”
<br/>　　他忽然一惊，握住了刀柄。风吹草动，隐隐听到黑暗中响起脚步声。
<br/>　　在声音落地的那一刹那，从黑暗中窜出几个人。待他们走近了，野原火才看清楚他们的模样。
<br/>　　每个人都狼狈不堪，头上挂着野草，衣服扯得十分破烂。如果草原上遇到打猎的戎族，野原火也不会觉得奇怪，令他吃惊的是，对方身上是帝国服装。
<br/>　　今天有很多事情没算计到，一个人贸然走出营地，的确有些冒险。现在成了北蛮王，手下人多了，想杀自己的人也是多的。而对方又是帝国人……戎族与帝国之间的仇恨，却也不是一句话两句话扯得清的。
<br/>　　这五人正是李舜时一伙，他们失散之后，就在草原上流浪。莫说找程大雷复仇，便是自己怎么回家都是个问题。于是，就成了草原上的孤魂野鬼，也是因缘凑巧，他们竟然在这里碰到野原火。
<br/>　　当然，李舜时不知道对方就是新晋的北蛮王，野原火同样不知道对方身份。
<br/>　　“师兄，是个戎族，怎么办？”
<br/>　　“还能怎么办，杀了了事，戎族杀我们多少子民。”
<br/>　　李舜时握紧了剑，五个人将野原火团团围住。
<br/>　　野原火已经握住了刀柄，目光低沉，冷冷扫过包围自己的五人。他内心绝不像外表这样平静，他是战场上的猛将，可江湖功夫其实并不如何。而眼前这五个帝国人，明显身手都不简单。
<br/>　　“诸位什么来历？或许我们有互相认识的人，聊聊？”
<br/>　　“与你这蛮子有什么聊的，兄弟们，动手。”李舜时冷喝一声，拔剑向野原火刺来。
<br/>　　出手就能看出对方本事不简单，未必在自己之下。更何况，对方一共五个同样实力的人。野原火不得不小心防备，用力挡了一剑，又连环进刀，去斩杀李舜时的脑袋。
<br/>　　他倒是懂得擒贼先擒王的道理，不过，想要在五人合围下斩杀敌人首领，却又谈何容易。
<br/>　　一味强攻，不过逼的对方后退而已。野原火忽然撤身，翻身跃上枣红马，狠狠一拍，向远处逃去。
<br/>　　他倒也没自满自大到觉得天下无敌的地步，天下无敌的样子是给别人看的，自己几斤几两还是要清楚的。假若现在不逃，可就逃不走了，也就是仗着枣红马的脚力，方能搏杀一线生机。
<br/>　　李舜时没想到，在眼皮子底下还能让这戎族逃了。李舜时顿时恼羞成怒，连日来的憋屈终于爆发出来，要将愤怒发泄在这戎族蛮子身上。
<br/>　　五人都有不错的轻身功夫，在短时间内，脚力能赶得上马匹。五人同时使出全力，在后面紧咬着不放。
<br/>　　野原火回头看上一眼，心中暗暗心焦。他却也想不明白，自己和这几个帝国人没仇没恨，就算偶然碰见了要杀掉，但也不至于没命追杀自己。
<br/>　　如此情况，想要脱身倒也不容易。莫非霸业不成，先要在此地埋骨。
<br/>　　“贼子，哪里走。”
<br/>　　李舜时突然加快脚步，眼看便要赶上野原火。这份脚力放在江湖上，当也算是屈指可数的存在了。
<br/>　　正在此时，一个黑袍老者突然出现在面前。他看上去年迈苍苍，随时都可能一命呜呼，可就那样杵在李舜时的必经之路上，只是用手中的木杖一甩，便将李舜时逼退。
<br/>　　五人先后围过来，老人小孩女子出家人，敢在江湖上露面，人人都有一手压箱底的绝艺。所以李舜时五人并没有小看这老者。
<br/>　　老者立在五人当中，神色轻描淡写，但和刚才野原火强装镇定不同，他是真镇定。
<br/>　　“书院的功夫，你们是夫子教出来的，是他的徒弟还是他的徒孙？”老者道。
<br/>　　“你是什么人？”李舜时皱起眉头。
<br/>　　“孩儿们，我劝你们回去吧。念你们学艺不易，今天饶你们一条性命。”
<br/>　　“师兄，别与他废话，一块儿杀了了事。”已经有人按耐不住。
<br/>　　“唉。”老者叹了口气：“苦口婆心，可你们仍是迷途不肯知返。罢罢罢，夫子徒弟应该很多的，杀你们几个，当也不损误了老友情意。”
<br/>　　“老贼，你忒狂妄了。”李舜时听到这话，顿时怒不可遏，一剑向老者刺来。
<br/>　　这来势汹汹的一剑，却被老者轻描淡写接住。他手中木杖一挥，便打中李舜时的胸口，李舜时立刻像断线的风筝一般，跌倒在地上。
<br/>　　“师兄。”
<br/>　　其他几人大惊失色，李舜时的功夫在书院本就不弱，哪怕放在江湖上不算绝顶好手，断然也没有被敌人一招击杀的道理。如此也就说明，这老者的功夫当真到一个可怕的境地。
<br/>　　剩下几人为了给李舜时报仇，同时向老者围杀而来。老者手中木杖轻轻挥洒，招招打的都是对方要害。
<br/>　　不过片刻，地上便倒下了五具尸体。老者的动作不像杀人，却像是一场点到即止的武道。功夫到这种地步，已经是出神入化的程度。当然，他的动作足够飘逸，可杀起人来却是绝不含糊。一招杀敌，绝不出第二招，当然，也不需要出第二招。
<br/>　　野原火早已拨转马头回来，冷冷的看着这一幕。
<br/>　　老者转过头，微微躬身，口中道：“白无双参见北蛮王。”
<br/>　　野原火仍旧骑在马上，保持一个自己随时可以撤退的距离。他狐疑的看着对方，上上下下打量了好几遍。野原火既然想在草原上打天下，自然不会一点情报调查也不做，他明白白无双是谁。
<br/>　　不过，令他困惑不解的是，这白无双好好去做呼延力的大长老，大半夜为何要冒出来，还肯出手救自己的性命。
<br/>　　他深吸一口气，声音冷冷道：“见过了。”
<br/>(本章完)
