<br/>

<br/>
<br/>        “子羡，我跟陆寒霆之间还存在很多问题，我们能不能走下去还是一个问题，只是，除了他，我真的没有办法再爱上别人了，我现在担着你未婚妻的名号却跟陆寒霆这样纠缠不清，我觉得很愧疚。”
<br/>
<br/>        “子羡，我努力过了，可是我还是将你当朋友，当知己，这样的我根本就不能给你带去幸福，我配不上你，子羡，忘了我吧，以后你会遇到一个你爱她，她也爱你的女孩，相爱是这个世上最妙不可言的事情，我不希望你错过。”

<br/>
<br/>        陆子羡舍不得逼夏夕绾，从前没舍得，现在也舍不得。
<br/>
<br/>        “好，绾绾，我尊重你的决定，我会找个合适的时间宣布我们退婚的消息，以后我们仍然是朋友。”陆子羡笑道。
<br/>
<br/>        夏夕绾心里很感动，她用力的点头，“恩，子羡，我们永远是朋友。”
