　　“少主。”
<br/>　　楚风眠刚刚降伏这战龙宝车，宫殿外一道血色身影，便是赶了过来，正是那鲜血魔尊。
<br/>　　“那寒姑娘的消息，我已经打探到了。”
<br/>　　“说。”
<br/>　　楚风眠目光一冷。
<br/>　　要是寒月漓出了什么意外，这天羽神域中的所有人，都要为她陪葬。
<br/>　　“寒姑娘，如今逃入到了冰裂谷之中，她早些时间，跟天剑宗的人有过一次交手，天剑宗的人似乎是发现了她的踪迹，不过那一次剑祖荒并不在，所以被寒姑娘逃出了。”
<br/>　　鲜血魔尊详细的禀告到。
<br/>　　“不过眼下，剑祖荒已经是亲自带人，前去了冰裂谷，只怕寒姑娘会有极大的危险。”
<br/>　　天剑宗，七大宗门的人，想要抓捕寒月漓，其中的目的，显而易见的，便是为的就是寒月漓的先天神体。
<br/>　　本来楚风眠还想这，以寒月漓的实力，在归墟天中，真正有实力对付她的武者，应该不多，所以才比较放心。
<br/>　　没想到这天羽神域中，居然还隐藏这如此之多的强者，这一下子将寒月漓置于了危险之中。
<br/>　　幸好楚风眠如今也来到了天羽神域之中，目前还为时不晚。
<br/>　　“冰裂谷，果然是这里。”
<br/>　　听到冰裂谷这个名字，楚风眠也猜测了一下，寒月漓所在的地方，十之八九，就在这冰裂谷中。
<br/>　　冰裂谷，是天羽神域之中的一处绝地，其中常年处于极寒之中，在其中只盛产一些冰系圣药。
<br/>　　一般武者，都不会贸然进入其中，只有一些需要冰系圣药的武者，才会进入其中。
<br/>　　寒月漓被七大宗门弟子围攻，肯定受了伤，她拥有玄寒之体，一切冰系圣药对她来说就是最好的疗伤药。
<br/>　　所以寒月漓会选择如今躲在冰裂谷中，楚风眠倒是猜测了一下，如今这消息，正是证实了楚风眠的想法。
<br/>　　“剑祖荒，已经带人去了，看来没有时间拖延了，我们走。”
<br/>　　楚风眠沉声说道，他直接让鲜血魔尊上宝车，化为一道光芒，直接向着冰裂谷的方向赶去。
<br/>　　有了这战龙宝车，赶路的速度，也是要比起他动用身法要快得多，只要是有着源源不断的灵力，战龙宝车的速度比起一般武者的遁光，都要不慢。
<br/>　　冰裂谷本来是在天羽神域的边缘，一般武者想要前去，都至少两三日时间。
<br/>　　但是在这战龙宝车的速度之下，仅仅一个时辰，楚风眠便是来到了冰裂谷。
<br/>　　靠近冰裂谷，所感觉到的，便是一种寒冷，这寒冷刺人骨髓，就算是那武者都很难抗住。
<br/>　　一般就算是生死境武者，贸然进入其中，如果十天半个月内不出来，都会冻死在里面。
<br/>　　所以很少会有武者，没事进入其中。
<br/>　　但是这一次，随着寒月漓的消息传来，冰裂谷也逐渐变得热闹了许多。
<br/>　　先天神体，这样的宝藏，人人都想要得到，尤其是剑祖荒。
<br/>　　他如今只差一步，便是足以超越生死，踏入碎命之境，缺少的正是一个机遇。
<br/>　　这先天神体，正是对他最好的机遇，是他突破最好的契机。
<br/>　　正因如此，现在整个天剑宗的弟子，几乎已经是全部出动，令这寂静的冰裂谷，如今热闹了许多。
<br/>　　除了天剑宗弟子，七大宗门弟子外，其他一些势力弟子，也有不少人是来到了这冰裂谷中，都想要碰一碰运气。
<br/>　　他们许多人的资质一般，若是谁可以得到先天神体的力量，自己便是可以直接拥有先天神体的资质，倒时候一飞冲天。
<br/>　　像是这样碰热闹的武者，也绝对不少。
<br/>　　楚风眠一路上，已经是看到了不少这样人，都想要来碰一碰运气，想要夺取寒月漓的先天本源。
<br/>　　“这些人，都该死！”
<br/>　　楚风眠目光之中，尽是杀意，他现在恨不得将这冰裂谷中的人，全部都屠杀一边，一个都不留。
<br/>　　但是现在寒月漓还没有找到，还是寒月漓的安微最为重要。
<br/>　　他便是没有理会这些武者，驾驶这宝车，向着冰裂谷内部前去。
<br/>　　越是达到冰裂谷内部，便是越来越多的暴风雪袭击，迫使楚风眠也只能是在其中，缓慢前行。
<br/>　　“剑师兄，难道都无法奈何的了那女子？还需要我们赶去。”
<br/>　　就在楚风眠的前方，突然出现了一群武者，大约七八人的样子，每个人都是一袭白衣，手持灵剑，一身凌冽的剑意。
<br/>　　天剑宗弟子。
<br/>　　楚风眠看到这群武者的出现，都是悄然之间的隐藏了身上的灵力，跟了上去。
<br/>　　“不是无法对付那女子，只是剑师兄想要得到那女子的先天本源，一剑杀了她轻松，但是想要夺取先天本源，必须要活捉才行。”
<br/>　　又一名天剑宗弟子脸上露出几分阴冷笑容道。
<br/>　　“据说那女子，长的肤如凝脂，月貌花容，也许剑师兄想要擒下她，将她收为奴仆也有可能。”
<br/>　　“那倒是，得了宝藏，又得美人的确是最好的。”
<br/>　　又一名天剑宗弟子，大笑起来道。
<br/>　　“先别说这种话了，我们尽快赶过去，免得一会剑师兄看我们来得迟，怪罪我们。”
<br/>　　“好。”
<br/>　　另外几名天剑宗弟子，都是点了点头，直接向着冰裂谷中心，催动身法过去。
<br/>　　“跟着他们。”
<br/>　　楚风眠在后面，隐藏气息，紧紧的跟随这些天剑宗弟子。
<br/>　　他的眼神中，已经是杀机涌现，这剑祖荒，还是有着天剑宗弟子，今日都要死。
<br/>　　他楚风眠的女人，岂能让他人窥探。
<br/>　　今日便是他楚风眠，大开杀戒的时候。
<br/>　　楚风眠跟随这群天剑宗弟子，一直是过了半个时间之中，一路来到了冰裂谷的深处。
<br/>　　在靠近一处冰川之前，一名蓝衣女子，正站在这冰川之上。
<br/>　　在这蓝衣女子身边，足足是有着二三十名天剑宗弟子，其中的白衣男子，剑祖荒，赫然在列。
<br/>　　楚风眠一眼望去，便是一眼认出，这女子，正是寒月漓。
<br/>　　“寒月漓，你就不用挣扎了，乖乖臣服吧，现在你乖乖臣服与我，我可以只要了你的先天本源，留你一条性命，让你留下来服侍我如何。”
<br/>　　剑祖荒看着寒月漓，冷声喝道。
