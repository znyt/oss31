        童筱颖微微摇头，但非常认真地回答道：“老婆倒是没有，只不过有个未婚妻，而且还同居同房了。大哥对大嫂真的非常好，大嫂要什么，大哥就给大嫂买什么。大嫂还经常跟我炫耀说，她是这世上最幸福的女人！”
<br/>
<br/>        “是这样啊！”齐灵云的眼神闪烁了一下，看起来有些闷闷不乐地抿着唇，嘴角微微下沉。
<br/>
<br/>        童筱颖正襟危坐，不再跟齐灵云闲聊了，而是拿起手稿，认真地修改起来，待会她还得把手稿录入电脑里。
<br/>
<br/>        齐灵云像似非常扫兴地回到了自己办公桌前，心不在焉地敲击着电脑键盘。
<br/>
<br/>        比起那些无聊透顶的八卦，童筱颖更喜欢把时间用在自己的设计上。
<br/>
<br/>        而傅夜沉的名誉，也因为童筱颖那些话而被掰正。
<br/>
<br/>        童筱颖知道，齐灵云一定会把她所说的话传出去。果不其然，公司里的那些爱好八卦绯闻的女员工们，也不再在傅夜沉的背后议论他是“断背山”。
<br/>
<br/>        而她童筱颖也没有再受到公司其他女同事们的拜托，再去给大哥傅夜沉送表白礼之类的事情。
<br/>
<br/>        “爱而不得，才心生恨意”是什么滋味，童筱颖并不知道，所以也无法体会公司里那些表白傅夜沉被拒的女同事们的心理。
<br/>
<br/>        不过，她觉得，比起“爱而不得”，更可悲的是“爱情的背叛”吧！
<br/>
<br/>        在爱情上被背叛这件事情，她已经经历过一次了，她不希望自己有第二次。
<br/>
<br/>        昼景不会背叛她，她坚信！
<br/>
<br/>        中午和同事们一起在员工餐厅里吃午饭的时候，童筱颖就会向已经结婚了的女同事“取经”，询问一些夫妻之间的相处之道。
<br/>
<br/>        齐灵云见童筱颖如此在意婚姻生活上的事情，便多嘴说了一句：“婚姻亦如爱情那样，有个时候，越是抓得太紧，越是适得其反。”
<br/>
<br/>        “如果不抓，总感觉他会离我越来越远……”童筱颖不禁感慨道。
<br/>
<br/>        “你男朋友是出差了吗？”齐灵云又问道。
<br/>
<br/>        童筱颖讷讷地点了点头。
<br/>
<br/>        齐灵云不禁咧嘴一笑：“如果你非常想见你男朋友的话，那就请假去找他吧！”
<br/>
<br/>        “去找他？！”童筱颖身心一怔。
<br/>
<br/>        齐灵云微笑着接着说：“对你来说，爱情应该比工作更重要吧！既然，你那么思念你的男朋友，就买张车票直奔你男朋友身边好了。说不定，这小别胜新婚。你们两，还可以把出差变成双人旅游。”
<br/>
<br/>        “对啊！我怎么就没想到呢？”童筱颖顿时喜上眉梢地拍手赞同。
<br/>
<br/>        齐灵云看着童筱颖那一脸无邪的样子，嘴角露出了内心里最真实的笑容。
<br/>
<br/>        在职场的这几年，她很难再看到对待爱情如此单纯的女人了。
<br/>
<br/>        其实，在爱情里，越是单纯的女人，就越是最傻的女人。
<br/>
<br/>        她突然很想嘲讽一下童筱颖的爱情，却发觉自己压根就没资格嘲笑别人。
<br/>
<br/>        童筱颖说做就做，吃过午饭后，她便开始认真地赶设计图。
<br/>
<br/>        只有把手头的工作都交差了，她才能安安心心地去找昼景。
<br/>
<br/>        下午五点，下班时间，大家都收工离开了格子间。
<br/>
<br/>        齐灵云走的时候，办公室里就只剩下童筱颖了，走之前，她还跟童筱颖打了声招呼，劝她不要工作到太晚。
<br/>
<br/>        童筱颖微笑着点点头，然后继续埋头赶自己的设计稿。
<br/>
<br/>        虽然一心想把设计稿赶出来，但是童筱颖仍旧不忘自己的设计初衷，该替业主着想的地方，她都会很细心地设计进去。
<br/>
<br/>        从五点等到六点，傅夜沉见大门口仍旧没见童筱颖的影子，便从豪车里出来，返回了大厦里。
<br/>
<br/>        这个点见到傅大Boss又折回公司，两个保安惊诧地颔首问候，还误以为是傅大Boss对他们的临时抽查，于是一丝都不敢怠慢。
<br/>
<br/>        傅夜沉直接去了设计部，走到设计部集中办公室的大门口，透过玻璃隔断，他看到了正在格子间里坐在电脑桌前忙碌的童筱颖。
<br/>
<br/>        大家都下班了，而她却还在忙？
<br/>
<br/>        傅夜沉顿时不满地皱起了剑眉，掏出手机给设计部的部长直接打了通电话。
<br/>
<br/>        “为什么只有童筱颖一个人在加班？”电话刚一接通，傅夜沉便直接开门见山地兴师问罪，语气冷淡且毫不客气。
<br/>
<br/>        “什么？怎么会！我没有给大小姐安排重活啊！大小姐现在设计的那套设计图，业主一点都不急，时间一点都不赶，完全没必要加班的！”设计部部长急得声音都有些暗哑了，生怕被傅大Boss误会自己虐待了那位童筱颖大小姐。
<br/>
<br/>        “……”没有任何下文，傅夜沉二话不说，直接把电话给挂断了。
<br/>
<br/>        童筱颖并不知道办公室外的情况，随手开了办公桌上的台灯后，继续专心致志地赶自己的设计图。
<br/>
<br/>        不知不觉中，墙壁上壁钟的时针从六点指向了九点。
<br/>
<br/>        童筱颖完成手稿录入后，高兴地松了口气，然后坐在椅子上撑了个懒腰，接着保存了文件，关了电脑，开始收拾包包准备下班。
<br/>
<br/>        肚子饿得“咕咕”直叫，特别是在这幽静的环境里，除了自己的脚步声和呼吸声，声音恍若被扩大，显得格外诡异。
<br/>
<br/>        童筱颖加快了自己的步伐，走到电梯门口，急匆匆地按着电梯。
<br/>
<br/>        白天这栋大厦，热闹得就像商场，而这一到晚上就……
<br/>
<br/>        童筱颖下意识地咽了咽口水，见电梯迟迟不动，又用力地继续按着，却仍旧没反应。
<br/>
<br/>        忽然，耳畔传来了沉重的脚步声……
<br/>
<br/>        “啪嗒、啪嗒”地慢慢地向她靠近。
<br/>
<br/>        这一层，除了她，没有别人，而她现在已经驻足在等电梯，那么身后的脚步声是……
<br/>
<br/>        童筱颖抓紧手提包，心脏“突突”地快速跳动着，全身的血液都在加速循环，但她的额头却渗出了冷汗。
<br/>
<br/>        脚步声越来越近……
<br/>
<br/>        一道白色的光线突然从童筱颖身后直接打过来，照得她的影子赫然在墙壁上放大。
<br/>
<br/>        童筱颖吓了一大跳，抱紧包包撒腿就跑。
<br/>
<br/>        来不及看清楚身后的人是谁，她越跑，身后传来的脚步声也跟着跑了起来。
<br/>
<br/>        紧接着，童筱颖只觉耳畔又多了一个脚步声。
<br/>
<br/>        慌乱之中，她撞上了一堵肉墙。
<br/>
<br/>        一只温热的大手，紧紧地抓住她的手臂，拉了她一把。
<br/>
<br/>        走道上的灯，忽然间全亮了。
<br/>
<br/>        童筱颖惊慌失措地抬起头来，撞入眼帘的是傅夜沉那张巧夺天工般的俊脸。
<br/>
<br/>        “傅总，晚上好！”身后忽然有人问候。
<br/>
<br/>        童筱颖又下意识地回过头去，只见一个手握手电筒，穿着保安制服的男人，毕恭毕敬地向傅夜沉颔首。
<br/>
<br/>        傅夜沉本能地拥住躲入自己怀中的童筱颖，看了一眼男保安后，命令道：“你去别处巡逻。”
<br/>
<br/>        “是！”男保安欠了欠身，转身离去。
<br/>
<br/>        整个过程，童筱颖一脸懵然，她紧紧地搂着傅夜沉的腰肢，思绪仍旧有些惊魂未定。
<br/>
<br/>        待那男保安离开后，傅夜沉淡淡地说了一声：“刚刚停电了。”
<br/>
<br/>        童筱颖回过神来，意识到自己的失态，连忙松手放开了傅夜沉。
<br/>
<br/>        “大哥，你怎么也没回家？”
<br/>
<br/>        为了缓解刚刚的尴尬，童筱颖随便找了个话题。
<br/>
<br/>        傅夜沉抬起手来看了看手表上的时间，云淡风轻地回答道：“加班去了。”
<br/>
<br/>        其实，他并未加班，而是一直留在这里等她罢了。
<br/>
<br/>        此时，童筱颖的肚子又“咕咕”地叫了起来，再次把气氛弄得尴尬不已。
<br/>
<br/>        傅夜沉面无表情地走过去按电梯，童筱颖见状连忙跟了过去。
<br/>
<br/>        两人进了电梯，童筱颖总是下意识地远离傅夜沉。
<br/>
<br/>        傅夜沉按的电梯按钮，童筱颖也未注意到，这电梯不是下行而是往上行。
