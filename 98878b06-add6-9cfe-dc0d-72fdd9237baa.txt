        阶梯教室里坐着上百个学生，有三个班合上这一堂课，能让教授单独点名，那他要么非常优秀，要么非常垃圾。
<br/>
<br/>        明司寒收起手机，从座位上起身后走向了讲台，拿起粉笔快速解答完后，随手将粉笔扔进粉笔盒中，看向教授，淡定道：“老师，我肚子不舒服，要去一趟洗手间。”
<br/>
<br/>        “……”虽然明司寒的解题速度很快，又是完全正确的答案，教授却面不改色地应了声，“好。”
<br/>
<br/>        明司寒双手插在裤袋中，离开教室后，回了连星儿的一条信息。
<br/>
<br/>        你的寝室是几零几？——明司寒。
<br/>
<br/>        鸳鸯楼二幺零。——连星儿。
<br/>
<br/>        鸳鸯楼？210？
<br/>
<br/>        明司寒身心一怔，那她现在岂不是就住他寝室的隔壁！
<br/>
<br/>        这事不是我干的，总而言之，我会替你查清楚。——明司寒。
<br/>
<br/>        连星儿。
<br/>
<br/>        连星儿回复两个标点符号过来后，明司寒就未在回复她的消息了。
<br/>
<br/>        寝室里烂摊子总得收拾，连星儿和顾千凝，还有赵安彤和路娅玟不得不请开锁师傅来换锁，然后各自把各自的脏衣服重新清洗了一遍。
<br/>
<br/>        好在四个女生都很团结，没有因为这点小事而起内讧。
<br/>
<br/>        连星儿倒是有自知之明，觉得这事一定是因她而起，所以下午放学后，主动请三个好姐妹们去校外的馆子里吃饭。
<br/>
<br/>        四个人一起坐在包厢里，饭菜上齐后，只是吃到一半时，最先感到头晕的人是顾千凝。
<br/>
<br/>        “我……我头好晕。”顾千凝说着便晕倒在餐桌上。
<br/>
<br/>        随后，赵安彤和路娅玟也纷纷趴在餐桌上闭上了眼睛。
<br/>
<br/>        “你们怎么呢？”连星儿焦急地起身，刚想去查看她们三个的情况时，自己也开始泛起了头晕。
<br/>
<br/>        下一个瞬间，她闭眼前，隐约看到有两个穿着餐厅服务生制服的男人推开了包厢门，闯了进来。
<br/>
<br/>        其中一个男人大步上前，接住了连星儿欲要坠地的身子。
<br/>
<br/>        “把照片拿出来看看，是不是她？”男人命令道。
<br/>
<br/>        另外一个男人连忙从衣兜里掏出相片，跟连星儿比对了一下。
<br/>
<br/>        “没错，就是她！”
<br/>
<br/>        得到肯定的答案后，男人捞起连星儿的身子，直接扛在了肩上。
<br/>
<br/>        就在他们想带着连星儿离开，刚走到门口打开包厢门，就被一记大长腿给踢回了包厢。
<br/>
<br/>        两个男人不得不先将连星儿放在一旁的地上，看着门外的一男一女先后走进来。
<br/>
<br/>        包厢门一关，二对二，四个人在包厢里展开厮打。
<br/>
<br/>        明司寒和裴萱妍三下五除二地撂倒这两个男人后，两个男人见自己打不过他俩，连滚带爬地逃离了包厢。
<br/>
<br/>        餐桌上趴着的三个女生，他俩完全不放在眼里。
<br/>
<br/>        明司寒第一个冲上前去的，是将连星儿从地上抱了起来，眉头紧锁：“她怎么呢？”
<br/>
<br/>        “他们在饭菜里下了催眠药，所以，她们只是睡着了而已。”裴萱妍解释道。
<br/>
<br/>        “你怎么知道他们会在这里动手？”明司寒瞥了裴萱妍一眼，质疑道。
<br/>
<br/>        裴萱妍心知肚明，明司寒这话是在怀疑她，故意为之。
<br/>
<br/>        “傅夜沉派了不下十个保镖暗中保护他的这个宝贝女儿，但是，他们无法贴身保护对吧？只要在他们认定的是安全范围内，他们那几个保镖便会放松警惕。上次，那边的人没抓到你，必定会将目标锁定在你的未婚妻身上。所以，这几天，我变装近距离盯着这丫头即可。”
<br/>
<br/>        裴萱妍的话到这里，明司寒终于信了。
<br/>
<br/>        毕竟，裴萱妍此刻确实是一副餐厅服务生打扮。
<br/>
<br/>        更何况，她没必要自导自演，没有任何意义。
<br/>
<br/>        “你说你的任务是奉命保护我，难道你上司就没打我的主意？”明司寒反问。
<br/>
<br/>        裴萱妍微微耸肩：“我上司对你是什么心态，我就不知道了，我只是奉命行事。你现在最好去打个招呼，让后厨给她们换一轮菜，以免引起骚动。”
<br/>
<br/>        明司寒不得不照做，将连星儿抱回了餐桌旁，然后和裴萱妍一起离开了包厢。
<br/>
<br/>        连星儿和自己的三个室友醒过来后，全都有些一脸懵。
<br/>
<br/>        怎么吃个饭，都睡着了？
<br/>
<br/>        连星儿见饭菜都凉了，回眸看向顾千凝她们问道：“你们还吃吗？”
<br/>
<br/>        “不吃了，好想回去睡觉。”顾千凝说道。
<br/>
<br/>        赵安彤和路娅玟也纷纷点头。
<br/>
<br/>        连星儿结账后，和三个室友一起回到寝室，只见她们倒床便睡，而她自己却突然间睡意全无。
<br/>
<br/>        不知道是不是她因为没胃口，吃得不多的缘故，她记得自己在昏睡前好像看到了两个陌生人闯进来。
