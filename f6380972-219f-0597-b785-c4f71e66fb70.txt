　　紫晶翼狮王忐忑的将紫灵晶与苍狼王魔核收下。
<br/>　　随后，瞳孔盯着慕清霄，七八米长的身躯显得异常僵硬，若是仔细看就会发现，它时不时就会哆嗦一下。
<br/>　　要知道，站在它对面的可是传说中的斗圣，能够一句话搞死苍狼王的存在！
<br/>　　苍狼王的死，让它心脏狠狠的颤了一颤。
<br/>　　原来强如斗皇，在许多强者眼里，依旧是不起眼的蝼蚁，它打定主意以后一定要低调。
<br/>　　它现在极其的忐忑，生怕慕清霄要杀它，希望赶紧回到自己的老窝睡一觉，将今日的事情全部忘掉。
<br/>　　“您……您请问。”
<br/>　　不知不觉中，紫晶翼狮王语气中充满了敬畏，说起话来都有些结巴。
<br/>　　对于它的态度，慕清霄颇为满意，笑道：“第一个问题，距离我上次离开魔兽山脉，已经有五年了？”
<br/>　　“是，是的。”
<br/>　　紫晶翼狮王点着硕大的脑袋，道：“距离那个斗皇女人离开，确实已经有五年了。”
<br/>　　沉吟片刻，慕清霄总算是松了口气。
<br/>　　既然只有五年，担心就是多余的了，毕竟他当初设下的封印，可以封印厄难毒体六七年的时间。
<br/>　　“第二个问题，五年间，大陆上有没有发生某些大事情？”
<br/>　　闻言，紫晶翼狮王摇了摇头，带动着一丝劲风。
<br/>　　“本王……我一直生活在魔兽山脉中，未曾出过山脉，不过，倒是在很多同类口中听说，魔兽山脉外面的世界非常广阔……”
<br/>　　听紫晶翼狮王说了半天，慕清霄大概了解了一些信息，后者根本什么都不知道，只知道外面的世界很大。
<br/>　　虽说是斗皇级别，慕清霄觉得，它更像是一只困在鸟笼中的飞禽。
<br/>　　没有见识过外面的世界，永远都不知道天有多高，能够成为斗皇，都是因为紫晶翼狮王本身的血脉天赋。
<br/>　　“请……请问大人还有何吩咐？”
<br/>　　慕清霄瞥了一眼紫晶翼狮王，见后者目光躲闪，没好气的挥了挥手，道：“你去吧。”
<br/>　　“多谢，告辞。”
<br/>　　话语刚落，紫晶翼狮王就一溜烟的消失在视线的尽头。
<br/>　　“也不知道仙儿和彩鳞过的如何，先回迦南学院看看吧。”
<br/>　　待紫晶翼狮王离开，空间清微一阵波动，慕清霄的身影就失去了踪迹。
<br/>　　……
<br/>　　……
<br/>　　与此同时，迦南学院中热闹非凡。
<br/>　　今日，注定是迦南学院今年中最热闹的一天，因为今天正是内院选拔赛的最后一天。
<br/>　　偌大的广场上已经人满为患，这里的并非全是学院的学生，还有很多，是从学院外的迦南城慕名而来。
<br/>　　作为生活在迦南城中的一员，他们都清楚，今日才是整个迦南学院每年最热闹的时候，自然不会错过如此盛宴。
<br/>　　几乎所有的目光都汇聚在五道身影上。
<br/>　　琥嘉，白山，吴昊，熏儿，萧炎此刻都是迦南学院外院名气最高的学员。
<br/>　　萧炎身穿劲装，平凡的容貌却透着一抹坚毅，背后挂着一把黑色巨剑。
<br/>　　与其说剑，倒不如说是巨尺。
<br/>　　在前两日的比赛中，周围所有学员与迦南城来者都对于五人强横的实力有了一定的了解，纷纷惊叹不已。
<br/>　　就在这时，一道倩影走进偌大的广场。
<br/>　　来者是一名身穿白裙的女子，拥有三千银丝，薄纱掩面，看不清真容，身材却前凸/后翘，薄纱下的容颜引人遐想。
<br/>　　“快看，是小医仙，小医仙终于要上场了……”
<br/>　　“你们这群牲口，快将路给让开。”
<br/>　　女子进入广场的瞬间，拥挤在入口的学员顿时爆发出一阵的欢呼声，随后主动让出一条道路来。
<br/>　　迦南城中的来者颇为意外，没想到这名用薄纱遮面的‘小医仙’竟有丝毫不弱于其余五人的名气。
<br/>　　顺着道路，小医仙来到广场中央一块空地上，独自一人静静的站着，对于周围的欢呼声仿若未闻。
<br/>　　对面，坐在椅子上，萧炎眼中闪过一抹诧异。
<br/>　　“熏儿，没想到除了你，还隐藏着如此有名气的女子，学院中果然藏龙卧虎。”
<br/>　　目光看了一眼小医仙，古薰儿目光闪过一抹复杂：“萧炎哥哥，其实这名女子，倒是与你有些关系。”
<br/>　　“与我有关？”
<br/>　　萧炎顿时一愣，有些二丈摸不着头脑。
<br/>　　古薰儿点了点脑袋，道：“不错，确切的来说，应该是与你那名失踪五年的师兄有关。”
<br/>　　“你是说慕师兄。”
<br/>　　萧炎惊呼一声，目光凝了凝，时隔五年，没想到会在迦南学院中听到这个名字。
<br/>　　古薰儿平静道：“萧炎哥哥，此女修炼与我们有些不用，若是遇上，一定要小心，更不要小觑她，当然其他的对手也均是不弱。”
<br/>　　此时，小医仙瞥了眼古薰儿，随后摸了摸自己的袖口，道：“彩鳞姐姐，你说他什么时候会回来？”
<br/>　　“嘶嘶……”
<br/>　　袖口中响起一阵细微的嘶鸣，只见一条七彩色的美丽小蛇钻了出来。
<br/>　　旋即，妩媚清冷的声音在小医仙耳边响起。
<br/>　　“仙儿，随着你的修炼，封印已经在削弱，一年内他应该会回来的，现在将注意力放在比赛上，你的对手可都不弱，特别是对面的两个小家伙，都隐藏着不弱的气息。”
<br/>　　闻言，薄纱底下的绝美容颜上浮现一抹惊喜，急忙开口道：“彩鳞姐姐，你说他今年会回来吗？”
<br/>　　“仙儿，你到底有没有听本王说话？”
<br/>　　七彩小蛇将身躯缩进袖中，道：“那就要看他对你在不在意了，应该会回来吧……”
<br/>　　当年，在慕清霄离开后，不知为何，内院并没有追查，反倒是如潭水一样平静。
<br/>　　曾经，小医仙与彩鳞也查探过慕清霄的信息，但这五年间杳无音讯，仿佛整个人都蒸发了一样。
<br/>　　小医仙眸子中闪过一抹笑意。
<br/>　　“彩鳞姐姐，我相信他肯定会回来的，你不是也希望他早点回来吗？”
<br/>　　“哼，小丫头骗子，安心比赛吧，等你进入内院，才会有更好的资源提供，毕竟你修炼的东西与其他人都不同，内院估计会有。”
<br/>　　微微颔首，小医仙抬起头来，目光在广场中掠过，在萧炎身上顿了下，最后落在了古薰儿身上。
<br/>　　……
<br/>　　……
