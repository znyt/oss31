　　三个时辰的时间过去，楚风眠一行人便是随着人群，来到了城主府中。
<br/>　　一眼望去，楚风眠已经是足以看到了其中的一座巨大法阵了。
<br/>　　这传送法阵上的气息，古老无比，乃是万年前就已经存在的，是由上古时代的大能，缔造而出的。
<br/>　　如今的时代，这等手段早已消失，这泉城国度的强者，最多也只能是进行修缮，却是无法建造。
<br/>　　“哼，你们一群废物，都给我滚开！”
<br/>　　突然之间，就在楚风眠的背后，一声娇喝突然响起，是一尊女子的声音，
<br/>　　楚风眠回头望去，便是看见一名女子，一身蓝衣，看起来二十岁出头的样子，一身灵力散发出来，实力却是不弱，有着神力境七重的境界。
<br/>　　楚风眠感觉的到，这蓝衣女子的实力，还是要远超境界的，起码是比起拓跋鼎，都丝毫不弱，比起寻常的巅峰圣者还要强大许多。
<br/>　　也算是一尊资质极高的天才了。
<br/>　　“你们这些废物，别在这里挡路，统统给我滚开，别浪费本小姐的时间！”
<br/>　　那蓝衣女子，看着城主府中排队的人群，突然指着其中的无数武者，怒声喝道。
<br/>　　“什么？”
<br/>　　“该死！”
<br/>　　听到这蓝衣女子嚣张的话，无数武者都是看了过去，但是觉察到这蓝衣女子的实力一刻，这些武者却是全都不敢开口。
<br/>　　神力境七重的境界，足以抵得上一些小势力的老祖宗了，他们这些人，远远无法跟他抗衡。
<br/>　　一些武者，虽然脸上有着怒意，但是也只能是纷纷退让开来，让出了一条道路，让那蓝衣女子走了过去。
<br/>　　“这里那个势力出来的刁蛮女子？”
<br/>　　“该死，她的实力不错，我们不是她的对手。”
<br/>　　“算了，她实力强，我们就避一避，如此嚣张，早晚会有踢到铁板的一日。”
<br/>　　那些纷纷退让开的武者，都是脸色一怒到，口中交谈着。
<br/>　　“恩？”
<br/>　　这一道道议论声音，似乎是被那蓝衣女子听在耳中，令她眼神冰冷起来，扫视了那些刚刚退开的武者一眼，眼神中尽是蔑视。
<br/>　　“一群废物，也敢来议论本小姐？不服气的，都给我站出来！”
<br/>　　这蓝衣女子开口，便是一股威压，直接压迫了过去。
<br/>　　蓝衣女子的眼神中，尽是不屑，丝毫没有将在场的武者放在眼中。
<br/>　　这一股威压，肆无忌惮爆发出来，足以媲美巅峰圣者威压，令许多实力稍弱的武者，都是不由的吐出了几口鲜血。
<br/>　　“这位姑娘，似乎有些太过霸道了吧，我们也没有招惹你，你居然直接拿威压压迫过来？”
<br/>　　一尊年轻男子，突然走了出来，这年轻男子，一身实力也丝毫不弱，有着神力境八重的境界。
<br/>　　在境界上，还压过那蓝衣女子一头，不过楚风眠看的出来，这年轻男子的底蕴，却是远远不如那蓝衣女子。
<br/>　　真正交手，只怕这年轻男子，会轻易败在这蓝衣女子的手中。
<br/>　　“是郑家的公子。”
<br/>　　“这位可是神力境八重的强者，进入北邙学院，可以去争夺一下内门弟子的位置的。”
<br/>　　“看来这郑公子，忍不住要出手了。”
<br/>　　“教训一下这蓝衣女子也好，免得她太过嚣张！”
<br/>　　一些武者，看着那郑公子走出来开口，都是抱着一副看热闹的心态。
<br/>　　这蓝衣女子，的确是太过嚣张了些，他们也想要看看有人可以教训一下这蓝衣女子。
<br/>　　“怎么，你想要来当出头鸟？来做这个出头鸟，你还没有资格！”
<br/>　　蓝衣女子瞥了一眼那郑公子，冷哼一声道，言语之中的意思，丝毫没有将那郑公子放在眼中。
<br/>　　她的声音，极为尖锐，刺耳无比，冷笑连连。
<br/>　　这话，顿时是让那郑公子的脸色都不由的变了，心中无边怒火爆发出来。
<br/>　　这蓝衣女子，的确是太过戳戳逼人了些，他出来只是劝解一句，便是直接被如此针对。
<br/>　　郑公子也是一方天才，神力境八重的圣者，心高气傲，哪里经受得住如此挑衅。
<br/>　　“居然如此，那郑某人也就来领教一下姑娘高招了！”
<br/>　　郑公子冷哼一声，身形一动，便是一爪直接对着那蓝衣女子抓了过去。
<br/>　　他的一双手，都是化为了黑色，变的坚硬无比，一时间瞬间，便是向着蓝衣女子抓了过去。
<br/>　　这一招虽然不是杀招，但是含怒出手，威力不小，便是想要给那蓝衣女子，一个教训。
<br/>　　“黑天玄爪！这郑公子可是号称玄爪圣者，一手黑天玄爪厉害无比，足以比起地级灵器都要强大，巅峰圣者都不一定抵挡得住！”
<br/>　　“郑公子本来便是同境界几乎无敌的人物，这蓝衣女子的境界才不过神力境七重，这一次郑公子应该足以轻易取胜。”
<br/>　　郑公子在泉城国度中，也是有着不小名气的强者，看到他出手，纷纷一些他的支持者不由的开口道。
<br/>　　“一个偏远地区的废物，以为有些奇遇，就想要跟我抗衡？想要强出头？今日我就先让你吃点苦头！让你知道天高地厚！”
<br/>　　蓝衣少女看着郑公子出手，面对着黑天玄爪，她冷笑连连。
<br/>　　她大手一挥，顿时无数血气凝聚在她的手心中，这血气化为一道长鞭，直接打了过去。
<br/>　　这长鞭一打，顿时那郑公子的一双黑天玄爪，顿时都被打断，令他不由的惨叫一声，整个人都是飞了出去。
<br/>　　“什么！”
<br/>　　“怎么可能！”
<br/>　　“郑公子居然败了！”
<br/>　　一些围观众人，看到这一幕，都是近乎无法相信。
<br/>　　神力境八重，足以抗衡巅峰圣者的郑公子，居然就这样败了，在这蓝衣女子前，一招击败。
<br/>　　这简直是令无数人震惊，郑公子可不是无名之辈，谁也想不到，居然会一招败在这蓝衣女子的手中。
<br/>　　“这，这蓝衣女子的实力，只怕是有着三点战斗指数的力量！比起我都要强大！”
<br/>　　拓跋鼎看着那蓝衣女子，都是不由脸色大变，这蓝衣女子的实力，换算成战斗指数，足有三点。
<br/>　　比起寻常巅峰圣者的力量，要强大三倍。
<br/>　　她还仅仅是神力境七重的境界，这种实力，绝对是当之无愧的天才。
