　　“萧逸，又是你。”顾长风咬牙说着，脸上开始杀意密布。
<br/>　　一旁的赵不群疑惑道，“这是何人？长风公子认识？”
<br/>　　“哼。”顾长风冷哼道，“一个不知从哪冒出来的土包子罢了，北山榜上没有排名，北山郡内，也没听说过有哪个强大的家族是姓萧的。”
<br/>　　赵不群面露恍然之色，不屑道，“哦，难怪以本公子的见多识广，却不认识他。原来只是个无背景无势力的废物。”
<br/>　　顾长风冷冷地盯着萧逸，高声道，“半月前，你杀我手下，坏我大事，我不屑杀你，便饶你一命。今日，你还敢现身，拦我去路，真是找死。”
<br/>　　赵不群笑道，“原来还是长风公子曾经的手下败将吗，呵呵。”
<br/>　　周遭的武者以及团伙首领更是嗤笑一声，“一个手下败将，被饶了性命，如今竟还妄图挡下长风公子和不群少爷，不自量力。”
<br/>　　“长风公子，快杀了他来助我们。”
<br/>　　一众团伙叫嚣着。
<br/>　　“管好你们的嘴巴。”一道冷喝声响起，一把重若千钧的巨型大刀劈下，数十武者顷刻间重伤。
<br/>　　那叫嚣得最厉害的‘首领’，直接被劈飞数十米，嘴溢鲜血，噤声闭嘴。
<br/>　　“谁敢再污蔑我萧逸兄弟半句，我要他的命。”林劲脸色杀意凛然。
<br/>　　同时，一只大雕在高空中尖鸣一声，双翼一震，无数锐利羽毛如同一支支利剑，令得上百武者纷纷仓皇躲避，不敢说话。
<br/>　　“我秦飞扬的队友，也是你们敢乱嚼舌根的？”秦飞扬在空中摆了个自认为帅气的动作，高声说道。
<br/>　　花剑、遁地鼠、秃鹰等人，也是加大了攻势，发了狠似的攻击各个‘首领’。
<br/>　　“能在剑术上让我甘拜下风之人，你们有什么资格轻视。”何健一手剑术如花乱舞，一阵繁华落下，数十武者毙命。
<br/>　　一下子，大型团伙那边伤亡惨重。
<br/>　　而小型队伍这边，则士气大振。
<br/>　　赵不群和顾长风惊讶地看着，没想到自己随便的两句话，竟让自己这一边落入了下风。
<br/>　　“长风公子，那些家伙也不知怎么回事，竟如此帮这萧逸，我们先干掉他再说。”赵不群说道。
<br/>　　“好。”顾长风点点头。
<br/>　　两人联手而上。
<br/>　　而萧逸，自始至终都没有说过话，一把紫云精铁剑，冷冷地指着两人。
<br/>　　在他眼中，这两人，挥手可杀。
<br/>　　顾长风攻来，他上次已经吃过亏，早知单靠自己的修为不是萧逸的对手，已然用上了下品灵器风吟剑。
<br/>　　赵不群也不是傻瓜，虽说嘴上轻视萧逸，但见顾长风一开始就用上了风吟剑，也立刻拿出了自己的武器。
<br/>　　下品灵器，妖火长枪。
<br/>　　赵不群，竟是个擅使长枪的武者。
<br/>　　一剑一枪，同时攻来。
<br/>　　萧逸怡然不惧，仍旧是单手握剑，剑指两人。
<br/>　　锵的一声。
<br/>　　萧逸持剑微微一档，轻松将两人震退。
<br/>　　“有些本事。”赵不群微微一惊。
<br/>　　“不过如此罢了。”顾长风面露不屑。
<br/>　　萧逸皱了皱眉头，冷声道，“你们很弱，若没有灵器在手，比之一般的先天九重弱得多。”
<br/>　　“猖狂。”
<br/>　　“嚣张。”
<br/>　　顾长风和赵不群脸色一冷，再次全力攻来。
<br/>　　又是锵的一声，两人被震退。
<br/>　　只不过，这一次，紫云精铁是竟一阵颤抖，差点没断掉。
<br/>　　紫云精铁剑，终究只能算比较好的兵器，远远比不上灵器。
<br/>　　这样碰撞，要不了几下肯定会断掉。
<br/>　　顾长风显然也发现了这一点，冷笑道，“土包子就是土包子，也只用得起这种不入流的兵器了。”
<br/>　　“哼，以你的身份，一辈子都摸不上灵器吧。”
<br/>　　“待得你手中之剑断掉，我看你还拿什么跟我们战斗。”
<br/>　　两人对于自己拥有灵器，很是自豪。
<br/>　　说罢，两人再次攻来。
<br/>　　“哼。”萧逸冷哼一声，身影瞬间消失在原地。
<br/>　　“好快。”顾长风与赵不群瞳孔一缩。
<br/>　　下一秒，萧逸瞬间出现在两人身后，手中紫云精铁剑已然消失，取而代之的，是一拳轰出。
<br/>　　嘭的一声，两人竟瞬间吐血击飞。
<br/>　　“好强。”顾长风与赵不群同时吃了一惊。
<br/>　　还未等他们的惊讶之色褪去，身体再次如遭雷击。
<br/>　　嘭、嘭、嘭.....
<br/>　　萧逸的身影不断消失出现，两人不断被击飞，如同两个泄气的皮球般。
<br/>　　为免浪费时间，萧逸使用了肉体力量。
<br/>　　轰，最后一声轰鸣，直接将两人震成重伤，在半空中划过一道血线后，两人无力地跌落在地。
<br/>　　“怎么可能强到这个地步。”两人眼中尽是不可置信之色。
<br/>　　“手持灵器的我们，战力不亚于半步洞玄，竟毫无还手之力？”赵不群咬了咬牙。
<br/>　　忽然，顾长风眼睛紧紧地盯着萧逸的拳头。
<br/>　　萧逸的拳头上，一阵澎湃的真气覆盖着。
<br/>　　“真气如此浓郁，那家伙肯定用了极强的武技，刚才才有如此强的实力。”顾长风肯定地说道。
<br/>　　赵不群点点头，也道，“萧逸不过是先天五重修为，刚才的实力却比半步洞玄还要强数分，那肯定是地阶武技。”
<br/>　　“地阶武技？”顾长风眼中顿时闪过一丝狂热。
<br/>　　“哼。”萧逸将他们两人的话听入耳中，却没有反驳。
<br/>　　他自然是没有地阶武技的，只不过自己的肉体力量如此之强，为免引起其他人过多的猜测，或者和狂血玄君扯上关系。
<br/>　　他特地将自身真气凝聚到拳头上，以此来掩饰。
<br/>　　这样看起来，别人会以为他是使用了武技或者秘法。
<br/>　　而且，其实他并没有实力全出，否则，以他现在的实力，足以媲美洞玄四重的妖兽，等闲洞玄四重修为的武者都不是他的对手。
<br/>　　这时，顾长风与赵不群脸上的狂热之色已经极为浓烈。
<br/>　　“那家伙居然拥有地阶武技，怕是走了什么大运得到的。”
<br/>　　“一个无背景无势力的废物，不配拥有这么强的武技，一定要抢过来。”
<br/>　　两人心中狠狠地想着。
<br/>　　地阶武技，对于武者的吸引力太大了。
<br/>　　而且，地阶武技，太过珍贵了。
<br/>　　整个北山郡，只有寥寥几个庞然大物拥有。例如裂天剑派、暗影楼、北山慕容家，以及四季城白家。
<br/>　　拥有地阶武技的势力，无一不是拥有破玄境武者坐镇的势力，强大无比。
<br/>　　顾家和赵家，虽然是百武城的大家族之一，但家族中最强者不过洞玄九重，比不上那些真正的庞然大物。
<br/>　　家族中，最强武技也不过是玄阶巅峰。
<br/>　　“动用灵器内的力量，一定要杀了他，将武技夺过来。”顾长风和赵不群互相对视了一眼，心中杀意大增。
<br/>　　他们的想法，足够狠辣，也足够孤注一掷。
<br/>　　只是，在萧逸眼中，却苍白得无力。
<br/>　　当初，连钟无忧那等战力的武者，加之毅力过人，强行催动灵器，加上地阶武技，尚且打不过他。
<br/>　　更别说如今的顾长风和赵不群了。
<br/>　　嗖，萧逸瞬间消失在原地。
<br/>　　两人甚至还为反应过来，萧逸已经出现在他们眼前。
<br/>　　轰，萧逸一拳轰在顾长风肚子上。
<br/>　　‘噗’顾长风一口鲜血喷出，直接重伤。
<br/>　　“给我死。”一旁的赵不群以为找到机会了，一枪刺出，往萧逸要害刺去，显然想要了萧逸的命。
<br/>　　“滚。”萧逸暴喝一声。
<br/>　　赵不群毫无招架之力，直接被轰飞。
<br/>　　当他跌落地面时，已经面如金纸，再无战力。
<br/>　　萧逸与他们二人的战斗，早已引起了不少武者的注意。
<br/>　　此刻，不少武者都倒吸了一口凉气，愣愣地盯着萧逸。
<br/>　　“那...那家伙，是怪物吗？长风公子和不群少爷，联手之下，竟在他手中数招都接不住。”
<br/>　　忽然，顾长风竟高声喝道，“诸位，萧逸手上有地阶武技。”
<br/>　　“什么？”所有武者一瞬间定格，纷纷看向萧逸，面露狂热之色，以及贪婪目光。
