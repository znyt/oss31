　　回过神来，慕清霄瞥了眼屋内的学员，见他们都面露警惕的模样，收回目光，不再理会，拍拍小医仙的香肩。
<br/>　　“仙儿，时间快要到了，尽快选择一个吧。”
<br/>　　话语声刚落下，屋内的学员都回过神来，由于慕清霄的缘故，他们可什么都没有拿到呢...
<br/>　　眼看时间就快要到了，在不抓紧寻找自己想要的东西，恐怕真要空手而归了。
<br/>　　古薰儿顿时松了口气，刚才一股冰冷刺骨的杀意席卷而来，她莫以为后者要在这里动手，心弦一直紧绷着。
<br/>　　现在看来，完全是她想多了。
<br/>　　平复心中的情绪后，她也迅速加入寻宝大军中，古族底蕴虽然深厚，但屋内也有许多她想要的东西。
<br/>　　如此宝贵的机会，她可不想白白浪费掉。
<br/>　　小医仙则微不可查的点了点头，可却并没有离开他的身边，而是静静的守候着，美眸依旧充满了担忧之色。
<br/>　　至于屋内的宝物，对她来说，仿佛完全没有一丝的吸引力。
<br/>　　叹息一声，慕清霄眼底闪过一丝温柔。
<br/>　　视线回到屋内，慕清霄瞥了一眼几道能量波动比较庞大的光团上，无奈的摇了摇头。
<br/>　　因为伤势的缘故，这些恐怖的能量团，根本不是他现在能拿下的，真是有些可惜了，若是因此牵动伤势，可就得不偿失了。
<br/>　　约莫只有十分钟，寻宝就该结束，小医仙不想参与其中，慕清霄也不强求，带着她来到门口，查探起自己的伤势。
<br/>　　在他的神识中，体内的经脉乱作一团，五脏六腑皆是移位，甚至有一些受损，如此严重的伤势，换做别人，估计已经躺地上了。
<br/>　　四肢百骸中，黑色火焰遍布，渐渐的恢复着伤势，恢复内伤的速度，显然比恢复外伤的时候要慢上许多。
<br/>　　检查完身体，慕清霄视线落在阴阳湖泊上，阴阳玉依旧悬浮在其上高速旋转着，倒是没任何异样。
<br/>　　伤势比较严重，想要完全恢复，最起码也需要半个月的时间，若是别人，没有三四个月，估计都下不了床。
<br/>　　“咦...”
<br/>　　就在这时，慕清霄惊疑一声。
<br/>　　此刻，他的四肢百骸中，居然流淌着一股暖流，搭配原本就变态的恢复能力，使得他恢复的速度加快了许多。
<br/>　　“看来，并不需要半个月的时间，阴阳玄龙丹，效果滋生了吗？”
<br/>　　随着阴阳玄龙丹的药效滋生，慕清霄苍白无一丝血色的脸颊，也渐渐恢复了一丝光彩。
<br/>　　见他脸色恢复一丝血色，小医仙明显松了口气。
<br/>　　“时间到了，仙儿，我们走吧。”
<br/>　　说罢，慕清霄就率先朝着原路返回，小医仙瞥了一眼屋内的六道身影，快步追了上去。
<br/>　　见两人离开，古薰儿也拿着手中的卷轴走出房间，由于空间中一股无形的力量，以至于纳戒根本无法使用。
<br/>　　其余五名学员倒是想多待会，奈何时间不允许，只得苦着脸，跟在古薰儿身后，向外走去。
<br/>　　他们手中拿的皆是玄阶高级斗技，或者功法，但脸上依旧是不满足，但想到慕清霄和小医仙，什么东西都没有得到，心情瞬间变得舒畅无比。
<br/>　　随着他们的离开，房间周围，犹如漩涡状的能量罩，也是渐渐变小，化作一点，伴随着一声细微的波动，最后消失不见。
<br/>　　此时，偌大的房间里，已经变得空荡荡一片。
<br/>　　任谁也想不到，就在刚才，整个房间中都充斥着无尽的宝贝，任何一件拿到迦南学院外，都能掀起一阵腥风血雨。
<br/>　　“嘎吱...”
<br/>　　伴随着时间的结束，紧闭的古老大门，宛如被一只无形的大手轻轻推开。
<br/>　　下一秒，刺眼的阳光顺着门缝倾洒而进，在漆黑的通道中，映射出一条长长的光线，但很快又被黑暗吞噬。
<br/>　　大门开启，慕清霄几人陆续走出通道，分别站在三个方向，气氛有些尴尬。
<br/>　　见学员都离开藏书阁，大门诡异的自动关闭，灰袍老者袖袍动了动，随后便有一股浩瀚磅礴的精神力量，在他们的全身扫过。
<br/>　　恐怖的精神力量，停留在他们身上几秒后，最后如潮水一般消失，估计是在检查他们的收获。
<br/>　　“时间到了，你们都离开吧，谨记，今日这里的事情，包裹藏书阁内部的消息，你们不得向任何人泄露一丝一毫。”
<br/>　　说话的同时，老者诧异的看了一眼慕清霄，不仅没有带出任何东西，连自身的气息，都比原来消弱了一大半！
<br/>　　其中，到底发生了什么事情？
<br/>　　他们虽在这里看守藏书阁，但也不曾进去过，但其中有什么东西，他们也清楚，可在里面受重伤，他们就理解不了了。
<br/>　　慕清霄不说，老者也不会特地去问，衣袍停止颤动，仿佛睡着了一般。
<br/>　　“呵呵，都出来了。”
<br/>　　瞧见他们离开藏书阁，一直站在楼阁前的琥乾上前一步，笑道：“怎么样，都没有空手而归吧？”
<br/>　　话语刚落，其余学员都面露惊喜，随后目光齐齐落在了慕清霄与小医仙身上。
<br/>　　慕清霄面色如常，道：“让副院长失望了，我们并没有从中获得任何东西。”
<br/>　　“...究竟怎么回事？”
<br/>　　琥乾微微一愣，两人都是高阶斗师，除去古薰儿，其余五名学员都没有两人强，怎会没有获得东西？
<br/>　　叹息一声，慕清霄道：“其中有的能量团中的力量太大，受了点小伤...”
<br/>　　经过一番叙述，琥乾便了解了其中的原委，道：“哎，有的事情不可强求，你们不要气馁，进入内院后，机缘也不少。”
<br/>　　慕清霄微微点头，心中倒是一点不可惜，若是猜的不错，整个藏书阁中的东西加起来，价值估计都没有他所获得之物宝贵。
<br/>　　得知古薰儿几人皆是寻到了东西，琥乾松了口气，对着两位老者微微弯身，道：“既然这些小家伙已经出来，那我们就不打扰二老苦修了，告辞。”
<br/>　　对于琥乾的话，老者不为所动，后者也不介意，对着慕清霄几人挥了挥手，道：“都跟我走吧。”
<br/>　　众学员也微微行礼，随后跟在琥乾身后，离开青石阶梯，转身向空间门外走去。
