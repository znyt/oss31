<br/>　　而荒天王，已经是开始了凝聚血肉晶体了。
<br/>　　在不朽真身这一条道路上，荒天王比起少的走的要更远。

<br/>　　在外，楚风眠拿着荒天王，的确没有多少办法。

<br/>　　既然如此，那正好就让这荒天王第一个尝尝这剑源界的力量！

<br/>　　论起力量，只怕是要比起半步天人境的强者，还要强横。

<br/>　　可是今日这荒天王，正好是以本体，强行撕开了空间，闯入到了剑源界之中，正好是剑源界可以大放异彩的时候。
<br/>　　“神剑斩天！”
<br/>　　轰！
<br/>　　剑源界的力量，在楚风眠手心之中凝聚这，化为了一道巨大无比的剑光，冲天而起。
<br/>　　这一道巨大无比的剑光，跟楚风眠的力量，没有任何的关系，而是纯粹剑源界的力量。

<br/>　　单单论起力量，这剑源界的力量，甚至比起楚风眠的力量还要强大。
<br/>　　当这剑光轰击到那一道巨大无比手臂之上的一刻，那一道手臂之上的无数血肉晶体，都在疯狂的颤动这，甚至是接近破碎的迹象。
<br/>　　真正的不朽真身，都并非是完全无敌。
<br/>　　像是那青铜天魔，血邪魔帝，都是凝炼了不朽真身的人物，可依然会陨落。
<br/>　　楚风眠同样是凝炼了半步不朽真身，可在之前的战斗之中，还有数次差一点都要被打破。
<br/>　　论起肉身，这荒天王的肉身，还远远不如楚风眠，在承受了超越极限的力量之后，这上面的血肉晶体，都在疯狂的颤动这。
<br/>　　如果不是突然之间，在遥远的时空之中，这荒天王又是一股力量传递到了这一道手臂之中来。
<br/>　　只怕是刚刚那一剑之下，这一道手臂，就要被硬生生斩杀。
<br/>　　这就是剑源界的力量。
<br/>　　凝聚成了圣地的剑源界，已经不再像是之前那般，毫无还手之力了。
<br/>　　只要是在这剑源界之中，在强大的半步天人境强者，楚风眠都可以将他们逼退。
<br/>　　“短短的时间内居然凝炼圣地，好手段，小子，看来我还是小看了你，今日这一道化身，我就送给你了，不过早晚有一天，你也会连本带息的吐出来。”
<br/>　　这一道手臂差一点都被斩掉，这荒天王明显的谨慎了许多，在知晓了剑源界的厉害之后，他的这一道手臂，便是缓缓的消失了。
<br/>　　已经是放弃了那荒鎏金，还有他的那一道化身，选择主动退去了。
<br/>　　看着离开了荒天王，楚风眠也没有出手阻拦。
<br/>　　他也清楚，这荒天王要走，楚风眠不可能留得住他这一条手臂。
<br/>　　剑源界的力量虽然强横，可只有在剑源界之中，才可以动用，一旦是这一条手臂离开了剑源界，剑源界的力量便是就没有任何的意义了。
<br/>　　而在剑源界之中，楚风眠借助剑源界的力量，面对这荒天王的时候，才可以占据上风。
