　　看到火箭弹的飞行轨迹不是指向自己这边的时候，李然便意识到情况不妙。
<br/>　　在火箭弹击中冷却塔，发生爆炸导致冷却塔倾斜的瞬间，李然双眼瞳孔紧缩，对着身边几名战备一队队员大声吼道：“快跑！！”
<br/>　　李然和四名战备一队队员向后狂奔。
<br/>　　嘎吱——！
<br/>　　令人牙酸的金属断裂声中，十几米高的冷却塔轰然倒地，尘土飞扬而起，水泥地面也是被断裂倒下的冷却塔砸出道道裂纹。
<br/>　　冷却塔横卧在李然和四名战备一队队员面前，阻断了他们的追击。
<br/>　　海滩边，智天使和吴云两个人已经是爬上了靠在岸边快艇，快艇上，雇佣兵扔掉了手中那具已经完成发射的AT4火箭筒，将弹链装填进枪管已经冷却的M134机枪中，然后将M134机枪又提在了手上。
<br/>　　李然带着四名战备一队队员冲到倒塌的冷却塔边，发现已经无法再阻拦智天使和吴云逃跑了。
<br/>　　智天使站在快艇上，从口袋中拿出那部从黑色宝马车中带出来的手机，打开手机中的遥控程序，手指在上面按动了几下，随后对着两人晃了晃手机，甩手把手机扔进海中。
<br/>　　咕咚！
<br/>　　手机落入海水中，溅起一道水花，但是很快这道水花就被快艇溅起的水花给吞没、消失……
<br/>　　智天使的举动自然是没有逃过李然的眼睛，当看到智天使拿出手机在上面按动的时候，李然就意识到了不妙。
<br/>　　“该死！”李然咒骂一声，按住耳麦对指挥中心内的赵天宇说道：“龙头，油气厂中有炸弹，智天使刚才应该已经启动了炸弹，立刻派排爆小组进入油气厂，我们会尽快找到智天使安放的炸药！”
<br/>　　“注意安全。”赵天宇叮嘱了李然一句，随后语气严肃的说道：“这个合资油气厂是我们国家跟K国合作的重点项目之一，为了建造这个油气厂，我们国家付出了很多，所以绝对不容有失！”
<br/>　　“明白！”
<br/>　　“分散寻找，注意保持联系！”李然对身边四名战备一队队员吩咐道：“发现炸药后，不要轻举妄动！”
<br/>　　“明白！”
<br/>　　五个人分散开来，在油气厂中寻找智天使安防的炸药。油气厂面积可不算小，五个人想要展开地毯式搜索来寻找炸药显然是不可能的。
<br/>　　在其他四名战备一队队员分散开去寻找炸药后，李然站在原地陷入思考中，如果自己是智天使会把炸药放在什么地方？
<br/>　　智天使不可能在这座油气厂中放置太多的炸药，有限的炸药要造成巨大的破坏，只能把炸药放在最有破坏价值的地方。
<br/>　　油气管道！原油库！
<br/>　　李然通过竖在一旁的指示牌，找到油气管道和原油库位置后，立刻通过耳麦跟其他四名战备一队队员取得联系，“两个人去检查油气管道，两个人去检查原油库！”
<br/>　　李然多说什么，他知道自己这么说，四个人完全可以明白自己的想法。
<br/>　　这座油气厂中的油气管道总厂超过一千公里，原油在被运送到这里完成加工后，经由这根油气管道输送到某海港，再从某海港输送到国内西南方，满足那里的生产和生活需求。
<br/>　　因为受到国际形势的影响，这条油气管道从立项到彻底完成建造，总共花费了近十年时间，其中花费的人力物力财力更是难以计算，所以这条油气管道对于中方的重要性不言而喻。
<br/>　　智天使放置的炸药并没有做什么伪装，李然很快就在油气管道的一处重要位置上找到了已经进入倒计时的塑胶炸药。
<br/>　　这块塑胶炸药并不大，有两块板砖叠放在一起那么厚，李然初步估计一下这块炸药的重量在2~3公斤左右，如果爆炸，造成的破坏难以估量。
<br/>　　炸药的量虽然不大，但这是油气管道！油气管道的爆炸，就跟油库爆炸是一个道理，对管道的摧毁是毁灭性的，这根耗费大量人力物力财力和时间建造起来的油气管道可能就彻底毁了！
<br/>　　“龙头，我在油气管道上找到了炸药。”李然盯着炸药计时器上鲜红、不断跳动的数字，说道：“倒计时还有十分钟！”
<br/>　　指挥中心内，赵天宇通过李然身上的战场视频系统看到了那块固定在油气管道上的塑胶炸药，对站在自己身边的林晓阳问道：“蜂鸟，你有什么看法？”
<br/>　　林晓阳在通过战场视频系统看到油气管道上塑胶炸药的时候，就已经确定这块塑胶炸药跟写字楼走廊垃圾桶里塑胶炸药是出自同一个人的手笔。
<br/>　　“水银平衡装置，布线复杂，里面可能包含了不止一根火线。”林晓阳沉声说道：“制作这种炸药的人是个高手，如果给我足够的时间，我或许能研究透它，但是十分钟的时间……基本不可能。一旦水银平衡装置发生倾斜，或者剪错一根线就有可能引发爆炸。”
<br/>　　“有什么办法可以解决？”赵天宇问道。
<br/>　　林晓阳思考了一会儿，说道：“液氮！油气厂里面肯定有液氮瓶，用液氮对炸药进行冷却，利用液氮对水银平衡装置进行喷射，将水银管中的水银冻结，是它无法发生倾斜，液氮的低温也能让炸药的电路板失去功能，这样可以把炸药从油气管道上拆除下来。”
<br/>　　“液氮！”李然对着身后两名战备一队队员大声命令道：“立刻把液氮瓶弄来！快！”
<br/>　　那两名战备一队队员对油气厂中液氮瓶的位置并不了解，但是在这座油气厂中工作的中国籍员工却是对液氮瓶的存放位置非常了解，很快就带着两名战备一队队员送来了两瓶液氮和三套防寒服、三组氧气瓶和防寒手套。
<br/>　　李然快速穿上防寒服，戴上氧气面罩和手套，让无关人员离开后，打开液氮瓶盖子，将液氮倾倒在油气管道上的塑胶炸药上，白色气体中，有着透明的液体倾倒在塑胶炸药上，很快塑胶炸药上水银平衡装置中的水银被液氮的低温冻结，令得这个水银平衡装置失去了功能。
