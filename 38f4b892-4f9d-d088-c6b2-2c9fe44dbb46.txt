　　第442章
<br/>　　从柳轻名处得到消息后，程大雷三人便往杀虎口赶。两地相距大约有五百里，一路马不停蹄。
<br/>　　如此，就可以看出程大雷座下墨玉乌月兽的奇异之处，如此全力赶路，竟没露出丝毫疲惫之象。经常将徐神机与银眸甩在身后。至于他们二人，则是沿途抢马，保持赶路速度不被拖慢。
<br/>　　但马撑得住，马上的人却撑不住，尤其是徐神机，他的年纪的确大了，已不能适应这样的高强度赶路。为了照顾他，程大雷不得不中途停下来，吃些东西恢复体力。
<br/>　　三人在河边进食，程大雷忧心忡忡，盘算着接下来的路程。照之前的速度保持下去，大概再有三天就能赶到杀虎口，当然，这是在一切顺利的情况下。
<br/>　　所以，林少羽至少要能撑过三天，也不知他能不能撑得过去。
<br/>　　“看破阴晴圆缺，洞穿世态炎凉。”
<br/>　　正此时，远处突然传来一声长歌，程大雷扭头看过去，只见一个道士骑着头毛驴，手中晃悠着一杆白幡。
<br/>　　四下无人，他高声长歌，天高地阔，像是为他摆出的舞台。
<br/>　　“大当家，这老道看着有点古怪。”徐神机。
<br/>　　程大雷点点头：“装神弄鬼，却看看他耍什么花样。”
<br/>　　然后，在三人的视线中，这道士竟骑着毛驴向这边来了。在接近程大雷时，缓缓鞠了一躬，道：“敢问是凉州蛤蟆寨的程当家？”
<br/>　　徐神机心中一警，以程大雷的名声，实在不该被人道破行藏。却不知这老道什么来历，若消息走漏出去，怕会引来许多无法避免的麻烦。
<br/>　　不知不觉，他的手已经摁住刀柄。虽然徐神机没什么功夫，但真拼起命来，却也是凶得很。
<br/>　　程大雷以眼神示意他稍安勿躁，目光落到老道身上，笑着道：“阁下怎么称呼？”
<br/>　　道士翻身从毛驴背上跃下来，与程大雷一样，席地而坐。
<br/>　　“无名亦无姓，江湖人唤我捉鬼道人。”
<br/>　　“捉鬼道人？”程大雷玩味着这个名字，笑道：“不知阁下怎么认识我的？”
<br/>　　“江湖上，程当家的名字谁人不知谁人不晓，对于程当家的为人，江湖同道都是很佩服的。我这次来，是特意给程当家指一条路。”
<br/>　　“喔，不知我的路在哪里，还要请教？”
<br/>　　老道手指西方，道：“程当家不该往这里来，应该去西边。”
<br/>　　“怎么讲？”程大雷道：“去西边我可就回去了。”
<br/>　　“原本就该回去。”这道士裂开嘴，露出一口黄牙：“程当家有所不知，关于你来并州的消息已经传出去了。天下诸侯都要杀程当家立功，江湖匪道要杀程当家扬名。将军府的老兵组，相府的十杀士，还有扬州的四大才子……等等这些人，摆在江湖上，可都是以一敌百的高手。如今这草原，已是要杀程当家的一张大网，程大家还不该回去么？”
<br/>　　“老兵组，十杀士，四大才子？”程大雷微微皱眉。
<br/>　　“老兵组是将军府手中的死士，都是沙场上的百战老卒，从战场中摸索出一套杀人功夫。干脆直接，却是凶猛残忍。”
<br/>　　程大雷点点头：“相比那十杀士与四大才子，都是同样的组织了？”
<br/>　　老道点点头：“相府十杀士，身份极其神秘，只知道他们极其擅长刺杀，一旦出手，从来无人可避。而四大才子，扬州李星招揽的四位奇人异士。”
<br/>　　“听名字，就很气派呐。”程大雷赞道。
<br/>　　“所以我有一言相劝，程大家还是从哪里来回哪里去吧，毕竟性命要紧。”
<br/>　　“多谢道长赐教，不过，我还是有个问题想指教，还请道长与我解惑。”
<br/>　　“程大家请讲？”
<br/>　　程大雷点点头，道：扬州四大才子，据说有一位周，名忘川的，该不会就是道长您吧？”
<br/>　　捉鬼道人一惊，脸上的表情僵固住，眼神忽然一厉，袍袖扬起。
<br/>　　程大雷手摁绷簧，匹夫剑出鞘，一剑洞穿这道士的喉咙。
<br/>　　而这个时候，老道的手刚举在半空中，随着他的气息断绝，手臂无力垂下，从袖子中流出一种黑色粉末。
<br/>　　徐神机脸现凝重表情，道：“江湖传言，说这周忘川擅长用一种断魂烟，触者毙命，应该就是这东西了吧。多亏大当家反应及时，不然我们都得着了他的道。”
<br/>　　“好说，好说啦。”程大雷摆摆手。
<br/>　　将军府，相府甚至李乐天都在收集关于程大雷的情报。而程大雷本身就是个对情报工作极其重视的人，怎么会对天下高手一无所知。
<br/>　　事实上，将军府老兵组，相府十杀士，还有所谓江南的四大才子，程大雷都有一定了解。
<br/>　　四大才子分别是道士、和尚、书生、剑客。每人都有一种出奇的本事，阴损的紧，许多成名高手都折在他们手中。
<br/>　　这周忘川就是其中之一，他装神弄鬼，接近程大雷，就是在足够的距离释放出断魂烟。只不过，他没料到，程大雷的剑比他相信的快。
<br/>　　虽然度过一场危机，程大雷并没有掉以轻心。周忘川虽然心怀歹意，但程大雷觉得，他带来的消息未必是假的。从此地到杀虎口这条路上，定是危机重重，高手如云。
<br/>　　看着地上周忘川的尸体，程大雷微微皱眉。既然道士来了，那和尚、书生、剑客当也离得不远，或者，他们已经到了。
<br/>　　风吹，草动
<br/>　　显百名披甲士。
<br/>　　他们隐在芦苇丛中，有条不紊的推进，随着风吹草荡，显出强弓劲弩。
<br/>　　程大雷登高望远，道：“看身上的皮甲，像是扬州李星的手下呐。”
<br/>　　“啧。”徐神机手搭凉棚，感慨道：“本以为他们会派出四大高手，结果却派来了一百悍卒，这……有些不地道啊。”
<br/>　　声音落地时，一百悍卒已经抽弓搭箭，对准了远处张望的三人。
<br/>　　“大当家，现在该怎么办？”
<br/>　　“不怕，本大家已有妙计。”程大雷深吸一口气，忽然大叫一声：“跑呀！”
<br/>　　三人翻身上了坐骑，程大雷回身敲落两根箭，没命般向远处狂奔。
<br/>(本章完)
