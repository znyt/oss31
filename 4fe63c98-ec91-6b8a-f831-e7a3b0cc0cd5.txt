　　叶尘怎么也没有想到，昊天大帝的传承，居然就在迷罗星海之中。
<br/>　　记得前世，那神器月光宝鉴，便是叶尘在迷罗星海的第七重星域中的某一处所得。
<br/>　　哪怕是以他前世的修为，进入迷罗星海之后，也仅仅只是堪堪达到第七重而已，当时他还想继续深入，可是刚刚进入第八重，就险些丧命其中，然后便退了出去。
<br/>　　至于第九重星域，那可是连无数渡劫仙帝，都铩羽而归的地方！
<br/>　　百万年以来，还从来没听说过，哪个人进去过。
<br/>　　不过，这如果是回到地球唯一的办法，叶尘就算拼死，也要一试！
<br/>　　一念及此，叶尘坚定的点了点头，
<br/>　　“我一定要得到昊天大帝的传承！”
<br/>　　“好！”
<br/>　　老者脸上露出一抹赞许之色，
<br/>　　“其实如果你不愿意，本帝也不会强求，既然你有如此决心，那老夫就送你一件法宝，来助你一臂之力吧！”
<br/>　　说完这话，老者手掌一翻，取出了一颗散发着紫色妖异光芒的珠子，递到叶尘的跟前，道：
<br/>　　“此珠名为逍遥仙珠，乃是我主人亲手所炼制，你别看它只是一件低等仙器，有此物在手，便可以连通修真界所有的传送法阵，然后强行夺取通道，为自己所用！”
<br/>　　叶尘听完老者的介绍，顿时两眼一亮，
<br/>　　“逍遥仙珠吗？”
<br/>　　他前世也曾经听说过此仙物，这可是连渡劫仙帝都十分眼热的东西，可是前世八百年，叶尘却从来没有见过，一直以为这东西，只是一个传说，没想到居然真的有！
<br/>　　正如老者所说，这东西虽然还算不上神器，可是从某种角度来说，恐怕比神器的用处还要大呢！
<br/>　　只要一珠在手，便可以随意调用，整个修真界所有的传送法阵，这简直就像是地球上，网络中的黑客，今后他在整个修真界，都能来去自如了。
<br/>　　这对于实力相对较弱的叶尘来说，简直比神器还要实用。
<br/>　　“如此，就多谢前辈了！对了，还不知道前辈如何称呼？”
<br/>　　老者淡淡一笑，
<br/>　　“本帝的名字吗？连我自己都快记不清楚了，你叫我九老就行了！”
<br/>　　“你既然来自昊天界，将来若是能得到主人的传承，便是我的新主人，不过……”
<br/>　　九老说到这里，话锋一转，道：
<br/>　　“我还要提醒你一句，你并非是主人的唯一传人！”
<br/>　　叶尘微微一愣，旋即恍然，继而眼眸中闪过一抹担忧之色，
<br/>　　在他进入修真界之前，地球上神、仙、妖、魔的修真者，也曾经有不少，踏入通天路，进入修真界之中。
<br/>　　九老的意思很明显，但凡是从昊天界出来的，都有可能继承昊天大帝的传承。
<br/>　　只不过，并不是所有人的运气都这么好，会得知这其中的原委。
<br/>　　倘若前世，叶尘有缘遇到九老，知道了这一切，恐怕也就不会那么轻易，从迷罗星海中退出，甚至有可能，已经得到了昊天大帝的传承。
<br/>　　九老接着又道：
<br/>　　“这迷罗星海，每百年开启一次，下一次开启，便是在一年之后，这里距迷罗星海路途遥远，你若是没事的话，还是尽快赶过去吧！”
<br/>　　说到这里，九老又指了指一旁的幽冥炎蛇，道：
<br/>　　“至于你的这个灵宠，受伤太重，实力也不行，不如就把它先留在我这里，你意下如何？”
<br/>　　叶尘其实也正有此意，听到这话，自然无有不允。
<br/>　　紧跟着，
<br/>　　九老将逍遥仙珠的用法，向叶尘讲解了一遍，叶尘将法宝炼化之后，略微调息了一下体内的气息，便直接催动逍遥仙珠，
<br/>　　嗡~~
<br/>　　随着逍遥仙珠上发出一道妖异的光芒，前方虚空之中，顿时出现了一条诡异的通道！
<br/>　　而且通过逍遥仙珠，叶尘可以清晰的感受到，附近几个星域所有的传送阵，都可以为他所用。
<br/>　　很快，叶尘便找到了一条，前往中央大星域的路线，
<br/>　　“九老，幽冥就拜托你了，告辞！”
<br/>　　叶尘找到路线之后，便直接身体一晃，进入通道之中，迅速消失不见。
<br/>　　“哎……”
<br/>　　九老看到叶尘竟然说走就走，顿时脸色微微一变，想要阻拦却已经来不及了，不由得摇头叹息一阵，自言自语道：
<br/>　　“这小家伙，也忒心急了！本帝的话都还没有说完呢，这逍遥仙珠虽然是空间跳跃的神物，可如果一次跳跃次数过多，仙珠就会承受不住，还会引起空间崩塌，甚至有性命危险！”
<br/>　　“哎，罢了罢了，反正之前那么多传承者，也没有一人能够成功，真要是死了的话，那就只能再找下一个了！”
<br/>　　“只是本帝寿元将尽，也不知道能不能在有生之年，为主人寻得一位传人？唉！！”
<br/>　　……
<br/>　　却说叶尘进入空间通道之后，飞行了一阵，发现通道中还算稳定，于是便祭出赤羽飞舟，穿梭速度顿时便大大提升。
<br/>　　一连几次跳跃之后，便已经出了天玄星域的范围，来到了附近的苍炎星域。
<br/>　　虽然有了逍遥仙珠之后，叶尘的速度大大提升，可是中央大星域实在过于遥远，绝非一朝一夕可以抵达。
<br/>　　一转眼，
<br/>　　三天时间过去了，
<br/>　　因为着急赶路，叶尘没有停歇，动用逍遥仙珠，已经跳跃了上千次，横跨了近百个星域！
<br/>　　但是离中央大星域，依旧有着不短的距离。
<br/>　　如此又一连过去了五六天，
<br/>　　凭着前世的记忆，叶尘终于踏入中央星域的范畴了！
<br/>　　所谓的中央大星域，是修真界的核心地带，其中又包含无数小星域，疆域之广阔，即便是前世身为狂帝的叶尘，也只到过一小部分而已，比之前的天玄星域，大了何止亿万倍！
<br/>　　而且，修真界最顶尖的势力、宗门，也都在中央大星域之中。
<br/>　　或者说，只要进入中央大星域，才算得上是真正进入修真界，那些外围星域，不过是荒郊野地罢了。
<br/>　　“终于回来了！”
<br/>　　叶尘也微微有些激动，正准备继续寻找路线，直接一鼓作气，前往迷罗星海。
<br/>　　却不料，
<br/>　　不知为何，逍遥仙珠突然变得黯淡了许多，继而连周围的空间通道，也开始一阵剧烈晃动，似乎随时都要崩塌一般！
<br/>　　叶尘顿时脸色大变！
<br/>(本章完)
