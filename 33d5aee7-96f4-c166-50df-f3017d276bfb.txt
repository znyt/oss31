　　渊男健离开之后，不管是李毅恒还是黄浦都大大的松了一口气，他们在外面布置的人手虽然不多，但也足足有二十多人。
<br/>　　可是刚刚那个不知真假的渊男健进来如此长的时间，竟然一个自己人都没有出现，可见他们布置在外面的那些人手定然已经遭了不测，就算是没有生命危险，估计也是被人家给放翻了。
<br/>　　不过好在那渊男健并没有什么杀人的心思，如果他真的有杀人的心思，此时估计李毅恒两人早就已经身死魂消。
<br/>　　“黄浦，怎么办？真的要与他合作么？”李毅恒在渊男健离开之后，定了定神，找个地方坐了下来，有些担心的问道。
<br/>　　“我们还有别的选择的了么？他的计划是刺杀伪帝，一但他的计划开始实施，无论成功与否，我们都不会再有任何机会。”黄浦的的眼中闪过一抹杀机。
<br/>　　不过眼下他们处于弱势，暴露了自己的位置不说，甚至连对方的藏身之地在哪里，有多少人都不知道，这样的情况下别说报复，能不被别人坑就已经可以烧高香了。
<br/>　　李毅恒也明白其中的道理，之所以会问黄浦不过是想要确定一下，现在黄浦既然有了决定，他自然也就从善如流的点了点头：“那就这样决定好了。”
<br/>　　“这样决定是没有问题，不过我们却需要换一个地方待了。”黄浦眯着眼睛说道。
<br/>　　李毅恒顺着黄浦的话点点头：“不错，这渊男健也不知道是真是假，既然我们的藏身之处已经被他发现，这里怕是再也呆不了了。只是，眼下我们到底去什么地方好呢？敌暗我明的情况下我们就算是换了地方，只怕也还会被发现吧。”
<br/>　　对于李毅恒的担心，黄浦淡淡一笑：“我倒是有一个地方，只是不知道王子木到底有没有这个能力。”
<br/>　　“什么地方？”李毅恒问道。
<br/>　　“李府！”黄浦神秘的一笑：“那个李禾不是说与伪帝关系紧密么，我们不如混到他的身边去，借他的势力藏身。”
<br/>　　“这个主意倒是不错，只是王子木那里……”李毅恒想了想，同样觉得黄浦的这个主意很不错。
<br/>　　“让人通知他一下好了，若是连这点事情都办不到，他就再也没有什么用处了。”黄浦阴森森的笑着说道。
<br/>　　……
<br/>　　便是这样，隐太子李建成的私生子李毅恒与护卫黄浦在王子木的引荐下混进了称心的府邸，被随意安排了一个位置安顿了下来。
<br/>　　称心伪装的李禾是一个急于求成，但却没有什么城府的人，说白了就是一个败家子，所以对于府上多了什么人，少了什么人根本就不怎么上心。
<br/>　　现在的他正急于和那些异族做生意，将一些大唐淘汰下来的东西全都打包卖出去。
<br/>　　为了换钱，钢制弩基本上都是按斤来卖的，横刀、陌刀之类的按捆卖，至于说箭矢之类的那就什么都不用说，只要买足够数量的武器，那东西完全可以赠送。
<br/>　　对于称心如此粗旷的买卖方式，李毅恒看在眼里急在心上，数次想要让王子木找机会劝劝那个李禾，不要这么败家。
<br/>　　王子木同样有些着急，看着无数的武器被装车运走，终于还是耐不住性格找到了称心：“大人，异族不能信啊，他们买了这么多的武器，一定是别有用心，咱们可不能再这样卖武器给他们了。”
<br/>　　“为什么？这不是很好么？能赚钱为什么不能卖，你知不知道，这几天功夫，本官就赚了五十万两白银！”称心两只手上带着八颗宝石戒指，手中擎着玉石制成的酒杯，里面琥珀色的酒液散发着迷人的酒香。
<br/>　　“大人，这钱来的虽然快，可是后患无穷啊，那些异族买了这么多的武器，一定是不安好心，万一他们将来把武器运回国之后，再来进攻大唐，岂不是……”
<br/>　　还没等王子木把话说完，称心已经打断了他：“本官管不了那么多，戍边是朝中那些将军的事情，与本官有什么关系，本官只是完成陛下交待的任务。”
<br/>　　王子木无言的舔了舔嘴唇，心中对称心愈发鄙视起来。
<br/>　　这个眼中只有钱的家伙，简直就是一个卖国贼，为了钱估计就连他自己的爹都能卖。
<br/>　　可是尽管话是如此说，但想到李毅恒交待自己的任务，王子木还是再次开口劝道：“大人说的是，可是大人，您知道么，前来买武器的都是一些异族中的国公之类，他们买走的武器很可能都用到了自己私军身上，您若是再这样继续卖下去，很可能引起别国对我们的敌视。”
<br/>　　不得不说，王子木这话说的前后有些矛盾了，不过看在称心的废材属性上，他并不在乎这一点。
<br/>　　而称心果然也没有意识到王子木的自相矛盾，只等他一说完便瞪了他一眼：“你懂个屁，这是本官的计谋。只要本官现在拼了命的卖武器给那些有私兵的什么公爵、王爵，那些异族的国王如果受不了就一定会再来找本官继续买武器，到时候本官就可以卖出更多的武器。”
<br/>　　王子木眨眨眼睛，皱了皱眉头，虽然他认为称心这样的想法纯属白日做梦，但却并不是没有这种可能，若是那些异族公爵真的运送大批武器回国，估计那些异族的国王搞不好还真的会来找他买武器。
<br/>　　而且面对如此肆无忌惮的称心，王子木的心里也产生了另外的一个想法，于是便装出一副被称心说服的样子，乖乖的退出了他的书房。
<br/>　　而一刻钟之后，王子木出现在李毅恒和黄浦的院子里，恭恭敬敬行了礼之后，才小心的试探道：“皇子殿下，臣有一个想法，不知道当讲不当讲。”
<br/>　　李毅恒与黄浦对视一眼，笑着对王子木说道：“有什么话你就说，就算是说错了也没什么，眼下正是关键时刻，正是需要集思广益的时候”
<br/>　　“是，殿下”王子木肃然而立，正色说道：“臣以为我们可以找一个异族，从李禾那里购买一批武器出来，正好可以配合我们过段时间的那件大事！”
<br/>(本章完)
