　　想到此处，慕清霄脸上掀起一丝弧度，端起酒杯一饮而尽，随后起身离开。
<br/>　　脚下的城市，是广西省附近一座不知名的小城市，占地并不广，挨家挨户都有田地，以种植蔬果为生。
<br/>　　在城中转了一会，慕清霄也不打算多逗留，找到一家杂货商店，买了一张简洁的地图。
<br/>　　地图上的路线并不算详细，但比较有名的城市都有标记。
<br/>　　想来也是，在这种偏僻小城商店购买的地图，能够标注一些简单的路线就已经不错了。
<br/>　　如果记得不错，剧情主角景天所在的城市，似乎就是临安。
<br/>　　临安虽不是王城，却也比脚下的城市繁华数倍不止，沉思许久，慕清霄终于决定，先去一趟临安。
<br/>　　毕竟，临安是剧情的初始地点，确实该去看看，若是剧情已经开始，他就需要重新计划接下来的行程。
<br/>　　想到此处，纵身便化作一道长剑冲天而起，眨眼间就似流星，自天际一闪即逝。
<br/>　　……
<br/>　　……
<br/>　　临安，中原城市里的一座中等城市，城市周围山林风景秀丽，相比别处，显得格外繁华。
<br/>　　此时，城中来往行人，喧嚣声不断，繁荣景象尽显。
<br/>　　“咻咻……”
<br/>　　天际一道亮光闪过，眨眼间便坠入城外的山野中。
<br/>　　临安城外几里外，荒无人烟的竹林中，一道高达数仗的巨剑自天际纵贯而下，旋即潜入地面。
<br/>　　诡异的是，巨剑仅仅引起一阵微风，在地上留下一道平滑的剑痕，看似神威却并未破坏周围的秀丽景象。
<br/>　　陡然，巨剑闪烁几下，最后化作一名青年男子。
<br/>　　青年一袭白衣，腰间挂着龙型玉佩，剑眉星目，光泽流转，面如冠玉，气质出尘，风姿卓越。
<br/>　　“这里就是临安吗？一路上下来所见城市，相比临安，确实都远远不如。”
<br/>　　望着远处的城墙，慕清霄微微点头。
<br/>　　化剑术御空速度极快，在一路上，慕清霄也并未落下风景，半天的时间，总共穿过十几座城市，但繁荣都远不比临安。
<br/>　　望着几里外的城墙，周围空间轻微扭曲一下，慕清霄的身影就消失不见，下一秒就悄无声息的出现在临安城的城门口。
<br/>　　城门口有两名军人把守，毕竟临安在众多城市中，属于非常重要的运输通道。
<br/>　　站在城门口，望着城墙上两个清晰的古篆，面露微笑的迈起步伐向城内走去。
<br/>　　第一次来临安，慕清霄并没有急着寻找景天，反倒是游玩起来。
<br/>　　偌大的街道两旁挂着彩灯，杨柳随风摇摆，路边摆着各种摊位，商贩的吆喝声，行人的欢声笑语绕耳不绝。
<br/>　　收敛气息走在街头，倒也没有引起任何骚乱。
<br/>　　毕竟，他的容貌实在太过俊逸，穿着打扮完全不似普通人，气质更是万中无一，想不引人注意都难。
<br/>　　走了没多久，慕清霄来到一家酒店，酒店生意不错，一眼望去，大厅中数十张位置上坐满了客人。
<br/>　　在这个时代，生活就是如此简单。
<br/>　　特别是各派修士，或者有空的平民，平日里就喜欢拉着朋友到酒店，叫两壶好酒和下酒菜，聚在一起谈天说地。
<br/>　　刚走进酒店，慕清霄脚步一顿。
<br/>　　在酒店靠窗口的位置上，正坐着一名中年男子。
<br/>　　男子一身蓝袍，剑眉星目，容貌极其英俊，仙风道骨，气势恢宏，具有长者的威严，剑气浩然。
<br/>　　“没想到，在这小小的临安城，居然能遇到如此强者，真是没有白来一趟。”
<br/>　　后者的气息很强，同样也是合道期的修士。
<br/>　　然而，让慕清霄困惑的是，他明明是第一次见眼前的中年男子，后者却给他一种似曾相识的感觉，这种感觉很奇妙。
<br/>　　他打量中年男子的同时，男子同样也注意到了他，当后者察觉到他身上隐晦的气息后，目光不由一凝。
<br/>　　强者！
<br/>　　现今宗门林立，宗门强者无数，但合道期差不多已经是世上的巅峰强者，哪一个不是宗门的老怪物。
<br/>　　像这种老怪物一般都是待在宗门福地中静心修炼，一心感悟更高的境界，唯独他是特例。
<br/>　　而现在，眼前却出现一名与自己有些相似的青年，而且对方年纪不大，但修为却极其恐怖，简直令人匪夷所思。
<br/>　　注意到中年男子的视线，慕清霄微微一笑，迈步走上前去，道：“可以坐下喝一杯吗？”
<br/>　　“请坐。”
<br/>　　闻慕清霄也不客气，直接坐下，后者则提起酒壶，给他倒满一杯美酒，笑容和煦，道：“阁下也是前来游玩的？”
<br/>　　“一来是游玩，二来在临安有些琐事需要处理。”
<br/>　　说罢，慕清霄就拿起酒杯一饮而尽，微笑道：“好酒，在下慕清霄，不知道友尊姓大名？”
<br/>　　“慕容紫英。”
<br/>　　话语刚落，慕清霄捏着酒杯的手轻轻一抖，抬起头来看向中年男子的目光，充满了古怪之色。
<br/>　　见后者一直盯着自己的脸看，中年男子眼角微微一抽，身体情不自禁的向后挪了一下，看向慕清霄的目光中也多了一丝古怪。
<br/>　　“咳咳。”
<br/>　　干咳一声，慕清霄岂能不知，后者是误会了。
<br/>　　只是面前的中年男子，实在是令他有些震惊，没想到后者居然是慕容紫英。
<br/>　　说道慕容紫英，就不得不说魔剑！
<br/>　　西晋末年，姜国剑冢被发掘，魔剑落于江湖人手，引起武林腥风血雨，蜀山仙剑派第三代掌门人降服魔剑，并将其镇于锁妖塔顶。
<br/>　　然而，魔剑并不能被塔所制，后又出塔，被心怀不轨者带至不周山，最终被慕容紫英寻得并尝试净化。
<br/>　　慕容紫英居于剑冢，偶遇九州散人，将魔剑转赠于他，魔剑重归蜀山后，慕容紫英也随后离开剑冢，游历天下。
<br/>　　当然，让慕清霄震惊的不是魔剑，而是慕容紫英，在数千年后有一尊称“紫胤真人”！
<br/>　　没错，正是古剑奇谭中，天墉城执剑长老，紫胤真人！
<br/>　　这也是慕清霄听到他的名字后，有些震惊，目光古怪的原因。
<br/>　　他万万没想到，只是一时兴起，却遇到了中年时代的紫胤。
<br/>　　虽然，现在的紫胤未成仙身，与他一样都是合道期的修士。
<br/>　　……
<br/>　　……
