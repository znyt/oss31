　　“关于玛丽科娃这个女人，还得去问无常。”
<br/>　　罗昊想了想说道：“他比我们任何人都了解玛丽科娃，对她的研究也最深。”
<br/>　　云熙点了点头，没有反驳罗昊的说法，将这张照片保存了下来后，关闭了软件和文件夹，继续从硬盘中寻找有价值的东西；硬盘中的数据虽然被恢复了，但还是有一部分被彻底摧毁，有一部分损毁严重，没有能够彻底被恢复。
<br/>　　“这个文件夹我听老师提起过。”云熙把鼠标放在一个文件夹上，对罗昊说道：“这个文件夹因为是被隐藏加密的，所以没有被摧毁程序破坏，当时老师从里面找到了一张地图，但是那张地图很模糊而且也不完整，老师曾说，他会试着从其他隐藏分区中提取数据，看最终能不能将那张地图恢复，但那个时候我因为忙着照顾那两只小的，所以也没有参与到其中，只是偶然间听老师提起过。”
<br/>　　云熙深吸一口气，动手打开文件夹，但文件夹并没有被打开，桌面上弹出了一个输入密码的对话框，并且提示只有三次输入密码的机会。
<br/>　　仔细的想了想，云熙在键盘上按了一串数字，就是子午加入龙隐部队的年月日，但这次弹出的提示却是“密码错误”！
<br/>　　云熙皱眉，难道这个文件夹上的密码不是子午设置的？但随后云熙又是摇了摇头，这个硬盘被蛰龙寄回国后，就一直保存在子午手中，别人不可能接触到它。云熙看了眼文件夹的创建时间和最后修改时间，她可以很确定，这个文件夹就是子午创建的。
<br/>　　既然是自己老师创建的，为什么他经常使用的密码会打不开？云熙想了想，把刚才那组数字以逆序的方式重新进行输入，然后敲下回车键。
<br/>　　“SUCCESS！”
<br/>　　随着弹出的提示，云熙成功的解开了文件夹的密码。
<br/>　　文件夹中有好几张图片，逐一打开后，云熙发现这些图片全都是地图的一部分，全都是不完整的，尝试了一下，无法拼凑成完整的地图。
<br/>　　“把这些图片叠在一起试试看。”罗昊盯着电脑中的这些图片看了一会儿，对云熙说道。
<br/>　　云熙打开了图片处理软件，将文件夹里的图片逐一放入软件，叠层在一起，经过一些翻转调整，将地图显现了出来，可是整张地图给罗昊的感觉就是别扭。
<br/>　　“把这张地图复制一下，然后等分成四和八。”罗昊思考后，对云熙吩咐道。
<br/>　　云熙按照罗昊的要求，将这张别扭的地图备份之后，将两份地图分别分成四等份和八等份。
<br/>　　罗昊接过云熙手中的鼠标，把地图碎片逆时针进行旋转，以“空间组图”的方式将地图重新组合。最后那张被分成八等份的地图重新被组合成了一张地图。
<br/>　　“从地图上看，这应该是一处山脉，可惜最关键位置的那一块没了。”罗昊指着地图上空缺出来的位置，说道：“就凭这么一副残缺的地图根本不可能知道地图具体位置，即便是把残缺的地方补全了，在没有具体坐标的情况下，也不可能知道这是哪一处山脉。”
<br/>　　罗昊靠在椅子上，揉了揉自己的太阳穴，原本以为这次抓到了关键之处；可是现实却狠狠地给了罗昊一拳，说到底，这根本就是一张没有任何作用的废图！
<br/>　　云熙把硬盘全都找了一遍，但是并没有再找到相似的地图碎片，线索到了这里突然就断了。
<br/>　　地图线索没有再被找到的原因有两个。第一，缺失部分的地图被彻底摧毁了，没有恢复出来；第二，就是仙岛夫自己也没有得到完整的地图！
<br/>　　罗昊把地图保存起来后，看了眼外面的天，发现天色已经彻底暗了下来，自己跟云熙为了这个硬盘里的东西，居然花了一个下午的时间去研究。
<br/>　　很长时间没有回来，家里面也没有什么吃的东西了，两个人只能叫了外卖来吃。
<br/>　　“那两个小的不在家，忽然发现家里挺冷清的。”云熙用勺子把一次性饭盒里的饭菜搅拌了一下，叹了口气道。
<br/>　　“想他们了？”
<br/>　　罗昊抬起头，看了眼坐在自己对面的云熙，说道：“想他们的话，明天就去看他们，反正从龙穴过去也方便。”
<br/>　　“算了。”云熙摇头道：“既然进了红旗学校就要遵守那里的规矩，不然对他们两个造成的影响不好。”
<br/>　　本来罗战进“红旗”学校会在明年全国招生的时候，跟着那些通过考核的学员一起进去，可是老校长夏洋对罗战这小子实在是喜欢的不得了，不顾自己的年纪，多次来到龙穴要把罗战给带走，大有你们要是不答应让罗战跟着他走，他就要“三顾龙穴”的架势。
<br/>　　赵天宇实在不忍心老校长一大把年纪还一次又一次往龙穴跑，跟罗昊和云熙商量后，就答应了老校长的要求，让他把罗战带去红旗学校。
<br/>　　红旗学校那边有老校长的关照，再加上蓝若依在那里任教平时也方便关照。再有就是罗昊曾经在红旗学校的“赫赫威名”，他的儿子，别人多少也要给点面子。
<br/>　　罗战跟着夏洋进了红旗学校，云悠自然也跟着一起去了，于是这姐弟俩就作为插班生，一起去了红旗学校。罗战这小子就跟罗昊一样，也不是什么安分的主，别人不惹他还好，谁要惹了他，保准跟谁呲毛，这个时候，能管住罗战的，除了云熙和罗昊外，也就只有他的姐姐云悠了。
<br/>　　第二天早上，罗昊和云熙还在睡觉，一个电话就把他们从床上给叫了起来，电话是云岳打来的。
<br/>　　“几分钟前，学校那边给我打来了电话，说罗战在红旗学校跟人打架了。”云岳在电话那头对罗昊说道。
<br/>　　“他把别人打了？还是被别人打了？”罗昊拿着电话问道。
<br/>　　“那小子还能被别人打了？”云岳说道：“他把一个高年级的学员打了，一拳头把人的鼻梁骨都给打断了。”
