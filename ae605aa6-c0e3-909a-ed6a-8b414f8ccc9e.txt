　　轰！
<br/>　　他再一手向着虚空划去，这一次，召唤出来的是一条浩大的命运长河，在这长河之中，有无数星辰在闪烁，有几个人，那边有多少星辰，这是人的命星！代表着一个人的命运轨迹！
<br/>　　他伸出巨手，便向着属于自己的那颗命星抓去，无尽的规则锁链向他袭来，洞穿天地，威力巨大。
<br/>　　他怡然不惧，挥拳破碎这些锁链，大手径直抓住那颗属于自己的命星，将之收了回来。
<br/>　　洪荒之中，大罗金仙就可以超脱命运长河了，更何况他这个堪比准圣强者的转轮境武皇。
<br/>　　之后，那颗命星没入他的身体之中，化为一轮闪烁着妖异红色光芒的巨轮，散发着命运的气息。
<br/>　　这是命运轮！
<br/>　　至此，三轮已满，他成为转轮武皇后期，堪比准圣后期。每个人的转轮都会有所不同，这是他自己的选择。
<br/>　　“武道之路，生死之途！敢拼敢搏！敢做敢当！开窍境为武灵，金丹境称武宗，神通境称王，洞天境为武尊，法相境为武相，转轮境方可称武皇。”
<br/>　　“武道无极限，转轮之后，极道称帝！”
<br/>　　轰隆隆！
<br/>　　伴随着话音落下，天地轰鸣，大道隆隆，不仅宇宙内光华璀璨，照亮了整片黑暗星空，就是其他各地也都如此，因为此刻，其他的万族生灵的头颅在发光，像是被人点燃了某种道火，埋下一颗种子。
<br/>　　其中，尤其是人族最为明显。人族之中，每个人的头颅都在发光，在他们的都脑海之中，一篇篇最为契合他们自己的武道经书正在著成，因为这是玄天为人族创下的武道，所以人族对于这武道最为契合。
<br/>　　在玄天传道的过程之中，他们也是洪荒之中最为受益的种族。好多人族的修为都在猛然提高着，之前是因为没有适合他们的修炼法门，如今出现了，而且还是最为合适的法门，人类都是先天道体，修行无需化形，最为契合大道，他们的修为自然也都在蹭蹭蹭地提高着。
<br/>　　其中最为明显的是女娲用手捏出的这些人类，他们作为第一批人族，资质最好。此刻有了功法，修为更是提升猛速，好多人都直接来到武道第五境界——金丹境，更盛者进入到神通境，可以称武王了。
<br/>　　其中，人族三祖更是厉害，因为有着人族气运的庇护，他们得到的武道功法也是最为顶级的，如：燧人氏的《燧皇人道诀》，以拳炼体，以燧火炼神，以人道之气炼意，以人道文明炼气，可以凝聚出无上燧皇法相，威力无比。也是最为适合他的功法。
<br/>　　再如有巢氏，他的功法也很神奇，竟然不是凝聚出洞天，而是可以在身体之中凝聚出浩大的殿堂，犹如天堂，挥拳只见，带动莫大的威势。
<br/>　　缁衣氏虽然是三祖中的唯一女性，但她的功法也不弱，竟然可以凝聚出遮天布衣法相，布衣遮天，有遮掩天机之效，作战之时，更能将敌人包裹在一个漆黑空间之中，与洪荒隔绝。
<br/>　　他们三人的修为直接上升到武道第七境——洞天境，开辟出自己的洞天，成就武尊境界。
<br/>　　人族之中，每个人的脸上都洋溢着兴奋的笑容，感受到自身的强大，他们都觉得，人族不再如之前那般羸弱了，人族崛起之机，出现了。
<br/>　　“那是……人族第一人祖！”有人认出了虚空之中传法之人，惊呼出来。
<br/>　　“还真是！天啊！玄天老祖竟然没有死，还创出了人族的修炼之法！”
<br/>　　“是啊，我人族要雄起！”
<br/>　　……
<br/>　　人族三祖更是兴奋，既高兴人族第一老祖玄天没有死，更高兴人族终于有了修炼之法。
<br/>　　“那是……大哥，太好了，大哥没有死。”缁衣氏高兴道。
<br/>　　“是啊，从此以后，我人族一样可以像仙人一样，可以超凡入圣，可以飞天入地！”
<br/>　　“是啊，我人族日后也能与仙人媲美的强者了。”
<br/>　　“……”
<br/>　　几人大喜过望，惊喜激动，纷纷大叫起来，以此来发泄内心深处的喜悦、兴奋。
<br/>　　“不过，大哥不会有事吧？”
<br/>　　看着虚空之中还在经受天劫考验的玄天老祖，众人担心不已，可是，尽管担心，他们一样没有办法，毕竟，那雷劫不是他们可以应付的。
<br/>　　“应该……不会有事吧，大哥这么厉害！”燧人氏不确定道。
<br/>　　“希望大哥能够平安渡过此劫。”缁衣氏这样说道。
<br/>　　……
<br/>　　“太强了，这种法很可怕，还没传承下来，就让各族共鸣，等于在人们的心中埋下一颗道种！”
<br/>　　“因为，这种法出现，才会有这等恐怖异象！”
<br/>　　“这种法只修自身，注重肉身，倒是与巫族之法优秀类似，武道吗，果然厉害。”
<br/>　　洪荒之中全都不平静了，之前闭关的准圣强者们，也都被惊醒，看到此法后，觉得它十分不凡，很厉害。
<br/>　　“这种法太过注重战力，只尊自己，有伤天和，与魔道又有何区别？这与大道不符，迟早要遭天谴。”有的人冷笑，觉得这道不行。
<br/>　　但不管怎么样，这都说明了武道的不凡之处，这武道几乎包括了巫族的炼体之道、妖族的本命神通之道、仙道的元神之道，十分不凡。
<br/>　　说到底，万道殊途而同归，不管是什么道，都是参悟天地大道，最后超脱天地，证道称尊。
<br/>　　玄天终于创出了属于自己人族的法，一条本来就有，但却无人发现的道，从炼体开始！
<br/>　　玄天不是第一个，也不会是最后一个。
<br/>　　之所以说它本来就有，那是因为不管是仙道、巫道、妖道、佛道，还魔道，它们之间都会有争斗。而各种搏斗之术，作战之法，本就属于武道范畴。
<br/>　　有人的地方，就会有争斗，有争斗的地方，就有武！
<br/>　　那些将武道理解成仅仅是炼体之法，是一种狭隘的说法。真正的武道，无所不容，包容万道，万道共存，以武演之。
<br/>　　什么仙道、魔道，什么妖道、巫道，我自修一道，它名为武。以武演万法，仙武，道武、魔武、妖武、巫武……
<br/>　　以我武法，破万法！
<br/>　　这种法完成后，影响太大了，故此降下的天劫比成准圣还要恐怖。若不是因为天道之下，皆有一线生机，就恐怕直接降下毁灭天劫了。
<br/>　　在接下来的数十年中，玄天都在雷光之中度过，有许多次曾被劈的浑身焦黑，甚至武道法体都曾龟裂过，血溅天地。
<br/>　　看得众人都为他心急不已，同时也有许多人佩服他，一个人能够被雷劈数十年，也就没谁了，你这得做多大的伤天害理之事啊，才能让天劫如此待你？
<br/>　　不管众人如何想，但玄天却凭着自己的意志与武道，一次又一次的熬了下来。
<br/>　　他受伤的时间越來越少，那肌体逐渐发光，变得无比坚韧。
<br/>　　当然，到了后来，这天劫也在变化，变得更强了，又一次让他遭受重创，可是，他又一次次的恢复伤体。
<br/>　　雷霆之中，蕴含的不仅只有毁灭，还有新生。
<br/>　　毁灭的尽头，是重生！
