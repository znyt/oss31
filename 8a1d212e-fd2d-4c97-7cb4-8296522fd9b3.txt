　　“您的意思是我们要去南美洲吗？”
<br/>　　罗昊抬头看着詹姆斯。
<br/>　　“我想是这样的。”詹姆斯点了点头，点开一张地图，在三维电子沙盘上放大，指着上面一个红点，说道：“这里就是西伦科特最后出现的位置。”
<br/>　　“‘******’王国！？”
<br/>　　罗昊看着在三维电子沙盘上放大的地图，诧异地看向詹姆斯。
<br/>　　“没错。”詹姆斯点头，回答道：“就是‘******’王国！全美国毒品市场上的******，至少有百分之八十来自于那里。国际刑警组织和特种部队多次联合对那里采取行动，但是收效甚微，就算打击掉一个制毒贩毒组织，很快就会有新的制毒贩毒组织冒出来。”
<br/>　　罗昊和詹姆斯所说的“******”王国是南美洲中部的内陆国家，是一个多民族民主共和制国家。全国以高原地形为主，平均海拔超过3000米，是世界平均海拔最高的国家，因为有着丰富的自然资源，所以这个国家又被称为“坐在金矿上的驴”，但是这个国家更为人所熟知的就是******，全世界绝大部分高纯度******都是来自于这里，也正是因为这个原因，这个国家又被称为“******”王国！
<br/>　　“我们已经在那里建立了指挥部，代号‘鹰巢’。”詹姆斯说道：“所有特种部队今天下午搭乘C-130前往‘鹰巢’，准备对西伦科特进行斩首行动，这次绝对不能再让西伦科特从我们的眼皮子底下逃脱！”
<br/>　　傍晚时分，两架C-130运输机从美军印度洋军事基地起飞，飞往建立在“******”王国东南部丛林中的行动指挥部——鹰巢！
<br/>　　“鹰巢”所在的地方，原来是一个炼油厂，后来这个炼油厂被制毒贩毒组织控制，变成了当地最大的制毒工厂之一，并且依托地形的优势，盘踞在丛林中多年，直到美军特种部队、国际刑警采取联合行动，才将这个制毒工厂彻底摧毁。
<br/>　　吉普车进入厂区之后，罗昊还能看到当时行动中被武装直升机所发射的导弹所摧毁，只剩下半截的烟囱。
<br/>　　厂区中长满了杂草，最高的杂草有罗昊腰那么高，地面上散落着很多的建筑垃圾和杂物。
<br/>　　行动指挥部“鹰巢”设立在炼油厂最大的那间厂房中，厂房门口站着几个荷枪实弹、全副武装的士兵，在对汽车中人员的身份进行审核后，才把几个人放入了厂房。
<br/>　　行动指挥部中，竖立着十几台服务器，正前方是一块巨型的液晶屏幕，指挥部一侧摆放着一排武器架子。十几名穿着目前主流的MC迷彩军人正在忙碌，当罗昊、麦克米兰这些人走进厂房的时候，他们也只是抬头看了眼，然后就继续专注于自己手上的工作。
<br/>　　整个行动指挥部中整齐有序，很难想象在昨天之前，这里还只是一个荒废了好几年的破旧厂房，这不得不让人惊叹于美国人的工作效率。
<br/>　　主屏幕前站着一个中校，他是目前行动指挥部中军衔最高的，他不时跟身边的技术人员小声说着话，提出要求，注意到从身后传来的动静，中校转身朝着走进厂房的多国特种部队走来。
<br/>　　中校站在麦克米兰面前，对着他敬了个礼，说道：“我是‘鹰巢’临时负责人Damien，在詹姆斯将军抵达前，这里由我负责协调。”
<br/>　　麦克米兰点了点头，对Damien问道：“方便介绍一下目前的情况吗？”
<br/>　　“当然。”Damien回答道：“我们这次动用了四颗近地轨道卫星进行侦察和监控，而且我们目前也掌握了西伦科特的下落。”
<br/>　　Damien带着麦克米兰、罗昊，还有其他几个国家的特种部队队长走到三维电子沙盘前，指着上面投影出来的三维影像，说道：“几个有红点的位置就是西伦科特这几天出入的地方，但是他很警觉，不在一个地方连续停留，位置经常变动。”
<br/>　　罗昊对这个情况倒是并不意外，毕竟在中东，西伦科特准备的藏身处，还有在抓捕到Parker前，联合行动司令部所没有掌握的秘密安全屋，总共加起来的数量可要比现在眼前看到的多得多。
<br/>　　“另外还有一个情况。”Damien关闭了三维影像，手指在三维沙盘上轻轻一滑，将一张黑人军官的照片放大在三维沙盘上，说道：“这个人叫拉蒙，联合军第3装甲部队上校，跟这里多个制毒贩毒组织都有联系，根据我们的监控，发现西伦科特这几天跟拉蒙至少有三次会面。根据我们推测，西伦科特很有可能是在寻求拉蒙的帮助。如果这次拉蒙也被卷入到其中，对我们的行动可能会造成很大的阻碍。”
<br/>　　“我认识他。”麦克米兰开口说道：“三年前，那次代号‘焚化’的缉毒行动中，我的小队是斩首部队，我们跟拉蒙手下的部队交过手，他们很强，不仅配备了先进的武器，战斗力也很强。如果不是后续部队的抵达，逼迫他们撤退，胜负还很难说。”
<br/>　　“拉蒙跟当地的制毒贩毒组织有联系，难道他的上级不做处理吗？”罗昊不解地问道。
<br/>　　“你把这里想的太简单了。”麦克米兰拍了拍罗昊的肩膀，说道：“一个制毒贩毒组织能在这里生存下来，如果没有关系怎么可能做到？拉蒙只是表面上我们所能看到的，在背后还有很多我们看不到的，其中涉及到的军队高层比你想象的要恐怖的多，不过他们也不敢做的太过分，他们是跟制毒贩毒组织有合作，但也有合作的前提，如果威胁到他们自身的安全，他们会立刻脱身。说白了，有钱赚的时候是自己人，但是遇到了解决不了的麻烦，就不再是同一阵线的战友。这些家伙狡猾着呢，如果没有被后人的授意，你觉得就靠拉蒙这个上校能活到现在？拉蒙不过是上面那些人的白手套，洗钱机器而已，随时可以一脚踹掉的。‘******’王国不仅是指这里的******产量，还有这里负责的关系网络。”
