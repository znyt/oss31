　　第269章
<br/>　　程大雷不知该如何回答李行哉的问题，该和你扯经济基础决定上层建筑，还是既得利益与下层民众的先天矛盾，或者是生产力和社会关系……关键是，这些东西程大雷自己也整不明白。
<br/>　　他只能尝试去阐述，一间屋子立得时间久了，又赶上狂风狂雨，总有一天要倒塌的。这是谁也没有办法改变的事情。
<br/>　　“难道破屋子就不能再修了么？”李行哉问：“给漏雨的地方补瓦，生白蚁就驱虫，该立柱子就立柱子？”
<br/>　　李行哉问出和明帝同样的问题，答案还是一样的，简单得只需要三个字。不过，很显然李行哉的眼光比明帝看得远，明帝只是问：帝国能否击败戎族，李行哉思考的是帝国的问题出在何处。
<br/>　　帝国能否击败戎族？能。但问题依旧存在。该被风雨吹倒的屋子，最终还是会倒的。
<br/>　　“一间屋子，打了很多补丁，立了很多柱子，如果去研究前人怎么修怎么补的，一定会觉得哇这块瓦补得好，这根柱子立得妙。可不管更好更妙的立柱补瓦，依旧无法改变这是一栋破屋子的事实。倒不如一把火烧掉，重新搭建，倒是更轻松些。”
<br/>　　程大雷说这些时，李行哉听得很认真，福德勒同样在倾听。听罢之后，李行哉表情更颓废，程大雷倒不觉得如何，反正键盘侠么，又不用为自己的话负责人，而程大雷也不是这破屋子的主人。
<br/>　　还是专心做好山贼这份有前途的工作。
<br/>　　晚上，出去打探消息的兄弟渐渐回来，大家聚在一起，交流得到的情报。
<br/>　　琴川关附近有五座城池，而距离此地最近的一座城池名叫浩甲城。想要驱除蛇岛上盘踞的蛇群，需要大批量的雄黄，或许只有浩甲城内才能凑齐。
<br/>　　“大当家，还有一个问题，咱们现在没钱了？”徐神机提出一个问题。
<br/>　　离开长安城时，程大雷散尽家财，如今真是一穷二白，从零做起。
<br/>　　“我们是山贼么，缺什么就去抢，要钱做什么。”程大雷意气风发的挥挥手：“从明天开始，大家什么都不要做，都去抢雄黄。周围的小村小镇，有多少抢多少，一定要快，快。”
<br/>　　“就算这样抢，想要短时间凑齐大量雄黄也不太容易。”李行哉问。
<br/>　　“那就要靠你了。”程大雷一脸色眯眯的笑。
<br/>　　“靠我，靠我什么？”李行哉忽然脊背发凉：“姓程的，我告诉你，我是来考察学习的，你可别拉着我和你一起做山贼。”
<br/>　　……
<br/>　　第二天，程大雷，徐神机，李行哉，福德勒一行四人进入浩甲城。
<br/>　　先找了一间成衣铺子，给李行哉与福德勒量身定做了两套衣服，接着又寻到汤池，给二人洗澡洁面。
<br/>　　真不知二人经历了怎样的生活，浑身上下一股馊味，和他们一比，程大雷恍然觉得自己竟也是个精致的小男生。
<br/>　　等二人从汤池出来的时候，全身上下已经是焕然一新。李行哉身着紫纹锦袍，两鬓垂黑，手中一把折扇，当真是浊世佳公子，翩翩美少年。
<br/>　　这时候，徐神机已经打探消息回来，城内最大的生药铺子是一间叫杏林堂的店，东家复姓西门，是浩甲城有名的大财主，家里呼奴唤俾，骡马成群。
<br/>　　“程大雷你究竟要我去干嘛？”李行哉现在还没琢磨明白。
<br/>　　“嘘，噤声，现在我可是上了通缉令的人，你应该不想又被关起来吧。”程大雷道。
<br/>　　“那我们去干嘛？”李行哉放低声音。
<br/>　　“去买雄黄。”
<br/>　　“我们不是没钱了么？”
<br/>　　一行四人来到杏林堂前，见联排的铺子，是一条街上最气派的地方。
<br/>　　四人来到门前，还没进去，就看到一中年男人被冲里面推了出来。
<br/>　　中年男人拱手哀求着门里：“求求林掌柜可怜可怜我，我家娘子病了三个月了，等着药救命呢。”
<br/>　　“你当这是什么地方，慈善堂么，打开门做生意，没钱滚蛋。”
<br/>　　一个留着两撇胡须的男人站在屋檐下，在看到李行哉时，顿时眼前一亮，道：“这位公子，是前来买药么？”
<br/>　　程大雷掏出一吊铜板丢给男人，随着李行哉跨入杏林堂。
<br/>　　“你们东家在哪里，我来和他谈些生意。”李行哉扫视着杏林堂内的环境。
<br/>　　“我们东家前街还有绸缎庄，不常来药铺，这里我做主，您和我说就行。”林掌柜道。
<br/>　　“和你谈，我和你有得谈么？”李行哉摆摆手，大手一挥道：“走啦。”
<br/>　　“公子息怒，公子息怒。”程大雷忙把他拦住，冲林掌柜道：“我们是从琴川关来的，想要采购大批的雄黄，最好还是把你家东家请出来。”
<br/>　　琴川关的莫鸣米半兵半匪，在此地属于半公开的秘密，一听这伙人来自琴川，林掌柜顿时重视起来。
<br/>　　“我家东家确实不在……”林掌柜压低声音：“东家最近和西市的潘娘子要好，怕是在一起吃酒呢，我们也不敢去打扰。您要多少雄黄，和我谈是一样的。”
<br/>　　“潘娘子，西门大官人？”程大雷眼睛一亮：“你们东家可是复姓西门，单字名庆？”
<br/>　　“不是啊，我们东家名叫春长。”林掌柜一脸懵圈：“怎么了？”
<br/>　　“没什么，想到一个熟人，大家谈生意吧？”
<br/>　　所以说居移体，养移气，李行哉一举一动都是王者做派，简单来说，他就是蹲在地上挖鼻孔，也是个挖鼻孔的帝国六王子，这一点不是谁想冒充就能冒充的。
<br/>　　这一幕还真把林掌柜唬住了，于是双方很顺利谈好。程大雷出高价采购一船雄黄，杏林堂负责送到琴川，到时候财货两清。
<br/>　　“三天，最多三天，把公子要的货送到。”林掌柜拍着胸脯保证。
<br/>　　事情如此顺利，程大雷四人也松了口气。
<br/>　　“只不过，公子是不是先支付一笔订金，我们也好安排手下人干活？”
<br/>　　李行哉一愣，抬起头看到林掌柜一双三角眼内，闪烁着狡黠的光芒。
<br/>　　“订金么？”李行哉顿了顿，从怀里掏出一块玉佩：“你看这够么？”
<br/>
