　　三年后，依旧是原来的地方，还是熟悉的两人。
<br/>　　元始天尊盘仰坐于青石之上，看着下方那已经修炼有成的太玄，开口说道：
<br/>　　“三年时期已过，为师要走了，你需努力修行，不得怠慢。”
<br/>　　“是，老师，弟子定然不会辜负老师的期待。只是弟子何时才能去寻找老师呢？”太玄恭敬弟应了一声，之后又询问道。
<br/>　　“哈哈，为师云游四方，天地之广阔，为师若是不想让人找到，你就算再如何寻找也寻不到。未来随缘吧，你也不必强求。”元始哈哈一笑，接着又对他劝谏道：“这皇宫内院，是温柔乡，北方极地，是贫瘠地，难以修炼有成。”
<br/>　　太玄听了，毅然决然地对着元始说道：“老师，弟子知这非长久之计，早有归隐之心，只待弟子除去周边妖魔，荡尽群邪，便向父王母后说明，云游四方，寻访仙山。”
<br/>　　元始见其归道之心果然似磐石般坚韧，向道之心还比广成子要强，大感欣慰，于是指点其说：“西方为金，与你心性不合，你应越海东游，寻访一座仙山，那山自乾兑起，水出震宫，山势广阔，自由太极，名为武当山，是你修行的最佳去处。”
<br/>　　顿了顿，元始继续说道：“你若寻得此山，便可隐居于此，择众峰之中高耸如云者居之，如此便可上得天河。”
<br/>　　语罢，元始便不再多语，道路已经指明，今后是成是败，便要看他自己的造化了。
<br/>　　“多谢老师指点，弟子谨记。”太玄道。
<br/>　　“嗯。”元始点了点头，沉默片刻，道：
<br/>　　“为师去也，你好自为之。”
<br/>　　语罢，元始不再多说，脚步踏出，便消失于山林之间，却是继续云游四方去了。
<br/>　　看到这一幕，太玄喃喃道：“老师真乃神人也。”
<br/>　　接着，他自己也转身离开了。
<br/>　　……
<br/>　　夏禹百年修道有成，便告别玉玲珑等四人，下山帮人族治理水患。
<br/>　　其实，最开始是大禹的父亲鲧在治水，可是鲧治水多年，大水依旧没有消退，鲧不但没有好的办法，还消极怠工，采用堵的方法，希望堵住这些洪水，可是，这些洪水是水妖在兴风作浪，哪有那么容易停止的，故而，最终他失败了。
<br/>　　大禹归来之后，见天下百姓处在水深火热之中，还要害人，乱水淹世，深感心痛，他觉得自己身为一个人族，一定要处理好这件事情。
<br/>　　帝舜将治水任务交给了夏禹，夏禹自然知晓这是一项艰巨的人物，同时也是一份重大的责任，但他依旧毫不犹豫地接受了。
<br/>　　大禹的父亲鲧因治水不利，被发配到了羽山，后来死在那里。大禹非但没有因此而怀恨在心，而更激起了他的雄心，他觉得自己的父亲没有治好水，而给人民带来了苦痛与灾难，而他自己更应该承担这份责任，努力去治理好这水患问题。
<br/>　　夏禹带领伯益、后稷和一批助手，跋山涉水，风餐露宿，走遍神州大地的山山水水，穷乡僻壤。各处人迹罕至的地方，都留下了他们的足迹。他沿途看到了无数的子民在洪水之中挣扎，也看到了无数水妖在吞噬着落谁的群众，他一次次地在那些流离失所的人民面前流泪发誓，定要治好水患，还人间一分清净。
<br/>　　夏禹吸收了父亲采用堵截的方法治水的教训，而是发明了一种疏导河水的新方法，其要点便是疏通河道，使水能顺利东流入海。
<br/>　　为此，他走遍神州大地，做出最详尽的计划部署，勾勒出所需要疏通的河道，一路之上，妖魔鬼怪无数，但依旧挡不住夏禹的脚步，开通河道有时会触碰到一些妖魔的利益，但夏禹依然没有害怕，一路上斩妖除魔，开通河道，沟通内外，在神州大地之上绘画河流，指点江山，泼墨山河社稷。
<br/>　　一次，他途经青丘，遇到一美丽女子，一段邂逅，女子听闻他的志向，愿意帮助他治理水患。大禹闻言大喜，一路上，两人相互扶持，最终夏禹也娶了这名女子为妻子。
<br/>　　原来这名女子名为涂山氏，是青丘一族族长的女儿，为九尾白狐，刚开始是为了接近大禹，谋取功德，但后来她却是真正爱上了大禹，大禹知道后，也并没有过多责怪于她。
<br/>　　大禹治水，三过家门而不入，一次治水路过家门，听到家中传来小孩的哭声，那是他的妻子涂山氏给他生下的一个儿子，他多希望进去看他儿子与妻子一眼，但为了治水大任，他只能向着屋内行了一礼，心中对妻子与儿子愧疚不已，却又不得不快速离开。
<br/>　　大禹根据神州山川地理情况，将天下划分为九州。九州依次是冀州、青州、徐州、雍州、荆州、豫州、梁州、扬州、兖州。一举定下华夏疆域范围分布，后世沿用千万年，人们又称华夏为九州。
<br/>　　大禹治水方法便是将华夏的山山水水当成一个整体来治理，先治理九州徒弟，该疏通的便疏通，该平整的便平整，使得大量土地变成肥沃土地。
<br/>　　之后再治理山脉，其中经过他治理的山脉有：岐山、荆山、雷首山、太岳山、太行山、砥柱山、碣石山、大别山等等。
<br/>　　山路治好之后，他便开始疏通水脉，长江以北的大江大河，都有他梳理过的痕迹。
<br/>　　一路梳理，一路杀妖，一路开道，一路前行。
<br/>　　一路之上，他的修为也是突飞猛进。
<br/>　　九鼎镇世诀，聚九鼎，方可镇世。九鼎分别为：天鼎、地鼎、阴鼎、阳鼎、风鼎、雷鼎、山鼎、泽鼎、水鼎。每一鼎都需要无尽资源，都需要凝聚天地元素之力，每一鼎，都相当于一个洞天，最后甚至都能演化出一方世界。
<br/>　　大禹一路上降妖除魔，治理山河，疏通脉络，靠的便是他的九鼎，九鼎包容天地间一切最基本元素，无物不吞，无物不聚，所以，这治水虽然是他的责任，但又何尝不是一种机缘呢。
<br/>　　最后，百年时间，终于让他疏通了天下的水道，将河流引进东海。今后不管有多少水患，都会经过河道而进入东海。
<br/>　　治水工程，至此完美竣工。
<br/>　　当然，也不算是很完美，他还没有找出最后发动无数水妖，祸乱天下的幕后人物。但即便如此，治水之事也完成了。大禹靠此次治水之功，赢得了天下民心，大禹治水，大仁大义，举世无双。
<br/>　　最后帝舜证道大帝，去了火云洞，将帝位禅让于大禹。
<br/>　　至此，人族最后一位大帝，北方夏禹大帝，终于开启了他的帝王生涯。
