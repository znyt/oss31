　　“这是？”
<br/>　　看着水镜破碎，那任务殿长老，猛然瞪大眼睛，脸上显露出了几分震惊。
<br/>　　“居然就连我，都推算不出当时的情况，这一股力量，可并而寻常龙族武技，其中的种种，居然是让我感觉到了远古的气息。”
<br/>　　任务殿长老，目光闪烁，似乎是在思考这些什么，突然猛然对着楚风眠开口道。
<br/>　　“这龙族武技，是怎么回事？”
<br/>　　“是这样的，这龙族武技，是我曾经得到的奇遇，从中一枚玉符中爆发出来的力量。”
<br/>　　楚风眠一边说着，一边拿出了一枚玉符，递给了任务殿长老。
<br/>　　“就是这一股力量，击退了地魔门的魔头，但是还是没能救了雷绝师兄。”
<br/>　　开口的时刻，楚风眠的语气，便是装出来的无比悲伤，这等小小演技，对楚风眠来说自然手到擒来。
<br/>　　演的天衣无缝，令人根本无法看穿楚风眠话中的破绽。
<br/>　　“这是，护身玉符？其中的力量，倒是的确跟水镜中出现的力量，一模一样。”
<br/>　　任务殿长老，打量了一眼手中的护身玉符，便是将楚风眠的话，信了七八分。
<br/>　　这一股龙族武技，显然不可能是楚风眠一行人谁可以打出来的，而且那气息，古老无比，甚至是来自远古时代的龙族武技。
<br/>　　也就只有这种曾经远古时代龙族强者炼化的护身玉符，才会有这种力量。
<br/>　　“这是远古龙族炼制的护身玉符，你应该是触发了其中的力量，才让这一股力量，爆发了出来。”
<br/>　　任务殿长老，打量了两眼，便是将那玉符，扔给了楚风眠。
<br/>　　“这里面已经没有多少力量了，可惜这等宝物，居然是用来对付一些小小魔头。”
<br/>　　任务殿长老转过头看向楚风眠开口道。
<br/>　　“雷绝的事，就这么过去了，算了，你们将那任务所要的东西，交出来吧。”
<br/>　　“是。”
<br/>　　楚风眠早已准备好，将一枚空戒递了上去。
<br/>　　在这空戒之中，便是盛放着的那祭坛的碎片。
<br/>　　任务殿长老接过这空戒，打开一看，不由眼前一亮道。
<br/>　　“不错，正是那地魔门的祭坛碎片，你们果然捣毁了其中一处祭坛。”
<br/>　　“不过。”
<br/>　　任务殿长老伸出手，向着那空戒之中，似乎是在寻找这什么东西，随后他脸上的喜色，便是消失了许多。
<br/>　　“这祭坛，的确是被捣毁了，但是你们三人可知道，这祭坛之中的东西，去了哪里？”
<br/>　　“这祭坛里面的东西，也属于任务的范畴之一，你们若是私吞，这任务你们依然没有完成，速速将其中的东西交出来！”
<br/>　　说着，这任务殿长老的声音中，都蕴含这几分杀意，简直就是在威胁楚风眠三人。
<br/>　　“该死！”
<br/>　　楚风眠的眉头不由一皱，心中都掀起了几分怒火。
<br/>　　按照雷绝的记忆，这任务，应该就是捣毁祭坛便是，就算是完成了任务了。
<br/>　　可是现在这任务殿长老，居然是要所要祭坛中的东西。
<br/>　　显然是这任务殿长老，知晓这种祭坛之中，肯定会有着一些支撑力量消耗的奇物，想要直接将这些奇物直接占为己有。
<br/>　　这简直就是在明抢。
<br/>　　这一枚龙丹，价值珍贵，为了这样一个所谓的任务交上去，简直可笑。
<br/>　　更何况这考核任务，本就没有任何奖励，楚风眠将龙丹交上去，也没有任何的好处。
<br/>　　楚风眠不卑不亢的开口道。
<br/>　　“长老，这祭坛中的所有东西，都在这空戒里面了，长老所说的东西到底是什么，我不清楚。”
<br/>　　任务殿长老听到楚风眠的话，也是眼神中露出几分冷笑道。
<br/>　　“真的不知？那你们就将你们的空戒，全部交出来吧，让我来查一查。”
<br/>　　任务殿长老的话，令楚风眠的脸色，也变得冰冷许多。
<br/>　　探察空戒，这简直是要将楚风眠的一切底蕴，都给查个清楚。
<br/>　　这等要求，简直不可理喻，一尊武者，若是没有底牌，那就是一只待宰的绵羊。
<br/>　　别说是任务殿长老，就算是北邙学院的刑法殿，无缘无故也不可能敢这样做。
<br/>　　楚风眠虽然是刚刚加入到北邙学院来，不愿意招惹是非，但是也不代表可是人人来在他一道。
<br/>　　楚风眠冷冷的开口道。
<br/>　　“长老，这事做的过了吧。”
<br/>　　“放肆！谁给你的资格，跟我这样说话！”
<br/>　　任务殿长老，猛然站了起来，一股威压，直接冲着楚风眠四人，压迫而来。
<br/>　　“你们现在，甚至都还不是北邙学院的弟子，就连考核都没有完成，也敢在我面前放肆！”
<br/>　　“速速将空戒什么都交出来！你敢对长老不敬，已经是触犯门规！你们的所有财富，都要交出来！就当是你们的赔偿！”
<br/>　　任务殿长老，怒喝道。
<br/>　　“再不快点，我就算你们没有完成任务，那你们便是擅自返回学院，临阵脱逃，便是死罪！知道吗？”
<br/>　　“恩？”
<br/>　　楚风眠的心中，杀意涌起，这任务殿长老的话，简直是戳戳逼人。
<br/>　　令楚风眠，都是动了杀心。

<br/>　　在北邙学院中的地位，已经不再一些小长老之上了，起码跟这陆鸣长老相比，都丝毫不差。
