　　萧逸的手，紧紧地握住了毒袍的手杖。
<br/>　　毒袍，怡然不惧，反倒阴翳的脸色愈加阴冷。
<br/>　　“论玩火，你是比我在行。”
<br/>　　“可这是荒影蛛之骨凝聚而出的苍炎，在我的荒影之骨下绝对控制。”
<br/>　　“论荒影蛛之骨的控制，你比得过我的荒影蛛武魂？”
<br/>　　毒袍的话语中，同样充斥着无比的自信。
<br/>　　对，说到控火，紫炎易霄确实极强。
<br/>　　可这太荒苍炎，说白了，乃是‘荒骨苍炎’，以骨化火。
<br/>　　以骨化火，亦由此骨绝对控制。
<br/>　　荒影蛛之骨，自也是荒影蛛武魂的掌控力最强。
<br/>　　萧逸，拿什么抢夺此骨的控制？
<br/>　　“桀桀。”毒袍冷笑着，手杖上的火焰，愈发澎湃，愈加快速增长着。
<br/>　　同一时间，毒袍身后，一具巨大的白骨之影凝聚。
<br/>　　这一景象，与当日萧逸在火牙宗和那位太上长老一战时，一模一样。
<br/>　　只是，此刻毒袍身后的白骨之影，更加巨大。
<br/>　　手杖上凝聚的火焰，也更加温度可怕。
<br/>　　“易霄，你拦不了本护法。”
<br/>　　手杖上的火焰，增长速度愈加惊人，从原本的一小型火球，渐成一数丈大小的火团。
<br/>　　火团之上，幽冷白异的火光，让人心惊至今。
<br/>　　毒袍，冷眼凝视着萧逸，准确来说，是凝视着萧逸的眼眸。
<br/>　　“易霄，你可知道，本护法异常讨厌你的自信，不，更准确来说，是你讨厌你这双眼睛。”
<br/>　　萧逸皱了皱眉，不语。
<br/>　　毒袍阴蛰笑着，凝视着萧逸的眼眸，愈发阴狠。
<br/>　　“你这双眼睛，清明、冷酷，又总是充满着自信。”
<br/>　　“在你眼中，似乎所有事都如你算计的那般，一切尽在掌控。”
<br/>　　“你凭什么？你又算什么？”
<br/>　　“本护法成名之时，你还只是个穿开裆裤的黄毛小儿。”
<br/>　　“就因为你和本护法鏖战多日，便能踩着本护法的名声扶摇直上，一跃成为中域炙手可热的妖孽？”
<br/>　　毒袍接连说着，语气愈发阴寒，愈发显得有些咬牙切齿。
<br/>　　对，他最讨厌的，便是易霄那充斥着无比自信的眼眸。
<br/>　　他毒袍，才应该是无比自信之人。
<br/>　　自他成为邪君府护法以来，多少年了，他记不得了。
<br/>　　但但凡他出手，要处理之事，从来都是完美成功。
<br/>　　但凡他出手，追杀之人，同样无人能活命；哦，不，唯一一次的失败，便只是那位惊天动地的传奇。
<br/>　　可那是一位传奇，一位大陆传奇。
<br/>　　而不是易霄这样的毛头小子。
<br/>　　而且，即便他失败了，那位传奇也奈他不何，硬生生让他逃了去。
<br/>　　可现在呢，这一切，仅仅是从一月多前，便悉数颠覆。
<br/>　　他要追杀的这个毛头小子，竟反倒是他奈何不得，甚至被压得屡落下风。
<br/>　　即便他要逃，这个疯狂的小子，不惜与龙家硬碰硬，也要搅得整个六龙地域风云变幻，硬生生要逼他出来。
<br/>　　若非他在六龙地域早有后手，想必他早在前些天便被逼得现身了。
<br/>　　还有武道冰晶之事，火牙宗也罢，那附近的数百地域也罢，本归他掌管。
<br/>　　可这一切，同样是在短短一月多的时间内，一个个分部，接连被毁；一个个得力手下，接连被杀。
<br/>　　他准备多年，存储多年，打算送回总府的大量武道冰晶，一朝被毁。
<br/>　　即便是在六龙地域这里，他谋划多时，仅差一时三刻，便能得到六龙六煞，得到整个六龙地域。
<br/>　　而这一切，又是在这个毛头小子身上，一朝尽丧。
<br/>　　原本，拥有于万物皆可掌控之自信的，是他毒袍温易。
<br/>　　而不是面前这个毛头小子。
<br/>　　毒袍的脸庞，抽搐着。
<br/>　　天知道他这一个月来，压抑着怎样惊人的杀意。
<br/>　　天知道他对易霄的愤恨，达到了何等地步。
<br/>　　毒袍身后，那具白骨之影，已达千丈大小。
<br/>　　白骨森森，面容狰狞，气息邪恶到极点。
<br/>　　手杖之上，火焰，已达百丈。
<br/>　　“快了，就快了。”毒袍咬紧了牙，直视着萧逸。
<br/>　　“你不是很厉害？不是很自信？”
<br/>　　“刚才那句‘论玩火，你比我在行’的自信呢？”
<br/>　　“来，拿你最自以为傲的控火手段，试试能否阻挠本护法的太荒之炎。”
<br/>　　他是咬着牙，但不是他在承受着什么，而是他在压抑着什么。
<br/>　　他在压抑着心头的痛快。
<br/>　　“桀桀，若你阻挠不了，由我的荒影蛛之骨所凝聚而出的太荒之炎，就会焚尽大半个六龙地域。”
<br/>　　“还有你，你这个外人中声名赫赫的控火妖孽；若是死在本护法的火焰之下，该是多么讽刺的事。”
<br/>　　“你猎妖殿武者，不是以击杀邪修为己任？但，今日，你阻挡不了。”
<br/>　　“今日，也是你，紫炎易霄，惹怒了本护法，非要阻挠本护法，才逼得本护法不得不以太荒之炎焚尽大半个六龙地域。”
<br/>　　“大半个六龙地域，过亿生灵，将因你而死。”
<br/>　　“六龙地域，也是唯一一个因你紫炎易霄，而被我邪君府怒火焚烧殆尽的闻名地域。”
<br/>　　“本护法不仅要将你烧至虚无，还要你身败名裂，桀桀桀桀，桀桀桀桀。”
<br/>　　毒袍阴冷大笑。
<br/>　　这一刻，他的脸上，再无压抑。
<br/>　　他的痛快，在这一瞬间，展露无遗。
<br/>　　因为，他手杖之上的冰冷白异苍炎，已凝聚至数百丈大小，要不了多久就能彻底爆发。
<br/>　　而这时，一直沉默的萧逸，缓缓张开了嘴巴。
<br/>　　“身败名裂？”萧逸的语气，淡漠如斯，“你觉得易某，是个在乎名声的人？”
<br/>　　毒袍眼眸阴蛰，“无论你是否在乎，今日你必死无疑。”
<br/>　　“呵。”萧逸这时玩味一笑，“毒袍，我说，论玩火，易某比你在行。”
<br/>　　“可我有说过，易某要用控火手段阻你吗？”
<br/>　　“你什么意思？”毒袍脸色一惊，心头闪过一丝不妙的念头。
<br/>　　萧逸淡漠一笑，“易某的手，握在你这手杖上许久了。”
<br/>　　“若非你得意洋洋地说那么多话，易某还真没把握。”
<br/>　　“嗯？”毒袍蓦地一惊，随后脸色大变，“紫炎，你诈我？”
<br/>　　萧逸淡漠一笑，“我说了，你看我像是在乎名声的人吗？”
<br/>　　话音落下的瞬间，萧逸眼眸一冷，“给我断。”
<br/>　　咔…
<br/>　　恰在此时，白骨手杖，应声而断。
