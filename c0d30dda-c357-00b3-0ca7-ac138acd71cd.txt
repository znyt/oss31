
<br/>　　<br/>　　    十几分钟后，附近的乡镇派出所到了，除了同样震惊意外，很快就封锁了现场，这个案子明显不是派出所能查得了的，紧接着没到半小时，就是区里的大案队过来接手案子了。<br/>　　<br/>　　    范成梁肯定也来了，他是这里的主管领导，园区也是他牵头建起来的，自前次死了三个工人后，今天又发现两具尸体，这里已经死掉五个人了，作为主管的直接领导，他于情于理都得来。<br/>　　<br/>　　    在现场，范成梁的脸色很阴，沉的吓人，听完了已经被控制起来的许主任的汇报后，他脑袋里首先就冒出了两个念头。<br/>　　<br/>　　    第一，这个破庙确实有问题，第二，则是有没有人在针对自己。<br/>　　<br/>　　    像范成梁这种被当做接班人培养的后备人员，在他们成长的路上，永远都不是一帆风顺的，所以每次遇到莫名的麻烦，他马上就会往阴谋论上联想了。<br/>　　<br/>　　    他是后备，但并不是唯一的，同时还有别的后备都在相同的起跑线上，没有个三四十年，是看不出最后谁能接班的。<br/>　　<br/>　　    而在这段时间里，搞不好就有人会阴他一把，毕竟把范成梁给干掉了，竞争的人就会少一个。<br/>　　<br/>　　    刑警队长姓潘，叫潘利明是个四十多岁的老刑警了，办案很有经验，他勘探完现场之后，就跟范成梁实话实说了，案子非常棘手，现场除了许主任和司机的痕迹以外，没有任何的痕迹，也就是说嫌疑人处理的非常干净。<br/>　　<br/>　　    “领导，我干了三十年的刑警，还是第一次碰到这种案子，凶手很有可能经验非常丰富，想要破案难度很大，所以我想请您给我最大的权利”<br/>　　<br/>　　    “保护好现场，你们慢慢搜集线索……”范成梁扔下一句话后就回到了自己的车里。<br/>　　<br/>　　    潘利明有点奇怪，按理来说碰到这种案子，在这种紧要关头，上面领导肯定都是雷霆暴怒然后并且要限期破案的，但是范成梁表现的太平静了。<br/>　　<br/>　　    回到车里的范成梁，拿出手机后找出了一个号码，犹豫了半天摇头叹了口气。<br/>　　<br/>　　    这时候正在拉sa修整完，准备前往大草原的王惊蛰接到了范成梁的电话，不得不改变了定好的行程。<br/>　　<br/>　　    一天后，三清山机场，王惊蛰从航站楼通道里出来的时候，一辆帕萨特前站着个三十岁左右的青年就快步迎了过来：“是王先生吧？”<br/>　　<br/>　　    对方说话时，就接过了王惊蛰手里的行李，然后笑道：“领导不方便过来，我是他的秘书，您交我小李就行了”<br/>　　<br/>　　    “范二哥忙着呢？”<br/>　　<br/>　　    “在区里开会，案子有些麻烦……”<br/>　　<br/>　　    上车后，帕萨特往前岗区方向开，李秘书很客气和礼敬，他来的时候范成梁就交代他一句话，你怎么对我的，就这么对待我让你接的这个人。<br/>　　<br/>　　    范成梁被破庙的事给整的有点焦头烂额了，他认为这是个麻烦的同时也觉得没准是个阴谋，所以想要解决这件事别人他是信不到的，也觉得不太托底，唯一能信得过的人，就是对他无欲无求的王惊蛰了。<br/>　　<br/>　　    帕萨特到了区宾馆，李秘书领着王惊蛰来到一间开好的房间，说道：“领导还有一个小时才能开完会，您先休息一下，房间里有水果……”<br/>　　<br/>　　    下午四点钟左右，下班前范成梁开完了会，会议是由一把手组织的，区公安和刑警队部门参与，主要探讨的就是破庙的案子，两次没了五条人命，这种人命案在前岗区至少都有三十年没发生过了，在市里面也是挂了号的，主要领导都已经发话过问了，范成梁很头疼，他刚上任没多久就出了这种事，如此棘手的问题他要是处理不明白的话，履历上可能就会有个黑点了，到时候他想往上走一步，都有可能成为竞争对手发力的针对点。<br/>　　<br/>　　    开完会以后，范成梁赶紧就找上了王惊蛰，两人在一个农家小院里见的面，打算边吃边聊。<br/>　　<br/>　　    “我虽然不太懂你们这些道道，但也觉得很麻烦，麻烦你过来一趟，不好意思了”<br/>　　<br/>　　    王惊蛰端着酒杯，叹了口气，说道：“以后不好意思的事少干吧，你正好搅合了我一件正事，就只好压后在处理了，说说吧，你那是什么问题？”<br/>　　<br/>　　    范成梁将前后事都叙述了一遍，王惊蛰一直听着也没有插话，对方说的无非就是一些表面上的过程，真正的东西都藏在表面下呢，他要了解的话，得亲自去现场看看，然后再看下尸体才会有什么结论。<br/>　　<br/>　　    “我最怕的是碰到针对我的阴谋，从京城外放到地方，意味着我要走上培养的道路了，有很多人不想看到我一步步的顺利走下去……”<br/>　　<br/>　　    王惊蛰皱眉说道：“你知道，我最讨厌和反感掺和进庙堂这种争斗里的”<br/>　　<br/>　　    范成梁两手一摊，笑了：“那没办法，谁让我把你当朋友了呢，不麻烦你麻烦谁啊？”<br/>　　<br/>　　    王惊蛰嘴角抽搐了两下，无言以对。<br/>　　<br/>　　    吃过饭以后，范成梁跟他说道：“稍后我给你介绍个人，是刑警队的潘队长，他会带你了解一些详情，案子方面我没办法给你了解，他肯定最合适，不过惊蛰……”<br/>　　<br/>　　    范成梁欲言又止的看着他，王惊蛰摊着手说道：“有话直说呗，我人都被你找来了，还有什么不能说的”<br/>　　<br/>　　    “如果查出，是针对我的阴谋，我希望你能帮我顺藤摸瓜把后面的人给揪出来，毕竟一个藏在黑暗角落里的对手，是很让人头疼的”<br/>　　<br/>　　    “你知不知道，你这个要求挺让人为难的……”<br/>　　<br/>　　    范成梁苦笑道：“从你的角度看，你为难，但在我的角度，我却一定要查下去，对方在我一外放的时候就动手，明显是不想看见我走的太远，这有点太狠了”<br/>　　<br/>　　    “不能拒绝呗？”<br/>　　<br/>　　    范成梁拍了下他的肩膀，笑道：“来都来了……”
