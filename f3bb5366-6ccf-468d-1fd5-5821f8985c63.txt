　　第九七一章
<br/>　　李二的想法是常人所不能理解的，就连李承乾在收到老头子的诏书之后也有一种懵圈的感觉。
<br/>　　急吼吼的找来孔颖达和苏檀，把诏书往桌上一拍：“您二位看看，父皇到底是啥意思，帮我参谋参谋？”
<br/>　　人老奸，马老滑，兔子老了鹰难拿。
<br/>　　李承乾虽然自认聪明，但是官场上的事情他还是差了一些经验，所以在陷处懵圈的状态之后，他第一个想到的就是长孙无忌，不过那老货此时正跟着老头子混辽东呢，根本没办法问他，所以只能把自己的老丈人和老婆的爷爷找来。
<br/>　　整个大唐，能帮他出主意，而且不能太过坑他的，算起来也就这两个人比较靠谱一点了，毕竟这两家老家伙就算是为了自己家的闺女考虑，也不可能置身事外。
<br/>　　但是李承乾明显的高估了孔颖达和苏檀，屁股决定脑袋这句话虽然不是真理，可是在很多时间却可以当成真理来用，孔、苏两人根本就没有达到那个位置，所以对这封诏的判读上，竟然比李承乾还要懵，竟然要给李二写奏折，建议不要让太子去辽东战场。
<br/>　　这特么不是扯淡么？老头子诏书都下了，怎么可能不去，就算是现在立刻自己打折双腿，估计到了最后也得背个抗旨不遵的罪名。
<br/>　　无奈之下李承乾将孔、苏二人打发走，一个在书房里面兜圈子，琢磨这件事情的利弊。
<br/>　　若是不想去，现在的办法只有一个，那就是去找老老李，让重病之下的老李渊把他留在长安，这样的话李承乾就不用担上抗旨不遵的罪名。但同样，李承乾若真是这样做了，那么无疑会在老头子那里失分不少，弄不好后果比去辽东还要严重。
<br/>　　可是若去辽东呢？
<br/>　　凭心而论，李承乾的心里还是很想去辽东看看的，身为一个男人，怎么可能不向往那种金戈铁马、沙场扬威的生活。
<br/>　　但是去了的后果他也很清楚，那与对自己早就已经心生不满的世家一定会在背后搞东搞西，弄不好就会给他扣上一个拥兵自重，阴谋造反的罪名。
<br/>　　所以他手下的军事力量一定不会很大，最多也不可能超过十万之数，因为这是老头子留给他的。
<br/>　　可是用十万人的军队去打高句丽……，这事儿可能么？就算是打下来的城池不用去驻守，光是战损也足以把这支军队拖垮。
<br/>　　去或者不去，李承乾考虑了整整一个下午的时间，最后不经意间看向远处的‘兰若寺’脑中突然灵光一闪，狠狠拍了一下自己的脑袋：该死的，怎么把这么重要的一个人物给忘了！
<br/>　　李纲，老李纲，这个曾经李承乾的老师，虽然这老家伙已经在前几年故去，但是在地府‘有人’的李承乾会在乎这一点点的困难？扯淡呢不是。
<br/>　　老崔那可是地府的判官，想提个人出来又有什么难的，再说地府和阳间可是有时差的，李承乾可没有忘了上一次他昏迷半个月，地府过了才不到一天的事实。
<br/>　　所以按这个来考虑的话，估计李纲那老头子应该是还没有投胎，找老崔把李纲好好照顾照顾，留他在地府多待上几年，这岂不是等于自己多了一个智囊？
<br/>　　于是乎，可怜的崔钰崔判官不得不再一次利用职务之便，假公济私了一回，查了生死薄，将老李纲找了出来，并且安排人带到了他时常办公的大殿。
<br/>　　“老师！可还记得承乾？”地府之中，老崔的公堂之上，李承乾看着被带进来的老李纲，笑着迎了上去。
<br/>　　“太子殿下？！”李纲看到李承乾脸色瞬间大变，看上去似乎有一种天要塌了的感觉。
<br/>　　本来听说崔判官要见自己，老李纲以为是要受审之类，结果没想到竟然在地府之中看到了绝对不应该出现在这里的一个人。
<br/>　　李承乾那可是大唐的继承人啊，此刻出现在地府之中，这说明了什么？大唐要变天了么？太子都特么挂了，大唐的未来还怎么办？
<br/>　　“老师，老师您怎么了？”李承乾扯着李纲问东问西，完全没有想到李纲的表现就是被他给吓的。而且如果不是老人家已经死过一次，估计被他这么一整直接就得背过气去。
<br/>　　过了好半晌，李纲缓过劲来，迟疑的问道：“殿下，大唐……，大唐发生什么事了么？”
<br/>　　“没啥啊，我就是想您了，下来看看！”李承乾摇摇头。
<br/>　　“想我了？下来看看？”李纲眉头皱了皱，脑门上有黑线浮现。
<br/>　　“对啊。就是看看您，一会儿我还得回去呢！”李承乾接着说。
<br/>　　“回去？你是说……，你能自由往来于……”李纲的脸色变的愈发古怪，伸手向上面指了指。
<br/>　　“对啊！”李承乾一边答应着，一边扯着李纲来到一个角落：“老师，我这次来实际上是有些事情搞不定了，打算找您给出出主意。”
<br/>　　夜猫子进宅，无事不来！就这就李纲现在心中唯一的想法。
<br/>　　不过老头子实在很喜欢自己最后收的这个弟子，所以尽管被他吓到了，最后想想还是原谅了李承乾，轻哼一声表示了一下自己的不满，然后问道：“到底出了什么事情，竟然让您费了这么大的心思来找为师？”
<br/>　　“老师，事情是这样的……”李承乾尽量用最精炼的语言把这段时间发生的事情说了一下，并且着重说了一下李二要调他去辽东的事情，最后问道：“老师，您认为弟子到底该不该去？”
<br/>　　“承乾呐，想必你应该知道两害相较取其轻这句话吧？”李纲默默的听完李承乾的讲述之后，缓缓问道。
<br/>　　“弟子知道。”李承乾尽管心中着急，恨不得老李纲马上给他拿个主意，然后好回去，但是最后他还是耐着性子认真的请教着。毕竟他自己的事情虽然重要，但是为了今后考虑，多陪陪这个小老头儿，还是很有必要的。
<br/>　　“既然知道，为何还要问为师呢。”李纲反问道。
<br/>　　“啊？”李承乾有些疑惑不解的说道：“老师的意思是……弟子回去装病？”
<br/>　　结果，李纲把眼睛一瞪：“朽木不可雕也！对于一般人来说‘两害相较取其轻’，但你是大唐太子，你的标准和一般人能一样么？你就不想想，如果你不能表现出迎难而上的魄力，将来如何能让陛下将这大唐放心的交给你……”
<br/>　　不知道老头子是不是很长时间没有说过话，总之李承乾在恢复自己意识的时候发现，外面的世界竟然已经过了两天！
<br/>　　就连一直在大安宫伺候老老李的长孙皇后都被惊动了，放下李渊那边的事情，跑来了东宫！
