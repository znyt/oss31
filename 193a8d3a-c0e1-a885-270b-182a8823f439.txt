　　第514章
<br/>　　第二天早晨，程大雷又拎着剑回了蛤蟆城。结果，当然是一无所获。他不得不承认，自己还是小看了这白衣尼姑。
<br/>　　拖着重伤之躯，却还有极好的脚力。前半夜程大雷大概还能跟上她的脚步，但后来，程大雷就彻底看不见她的影子了。这一路，委实是将程大雷累得不轻，他的体力快被耗尽，再追下去，程大雷估计就算追上了，死的也有可能是自己。
<br/>　　聂隐娘犯过的错误，自己可是不能再犯。所以程大雷半路见追不上，也就识趣的回来了。
<br/>　　这次和上次相比，蛤蟆城平静很多。根本没有发动大兵去寻找，只是在城门口放了几个眼睛，一有程大雷的消息，立刻回城禀报。
<br/>　　“大当家您回来啦。”城门卫兵招呼道。
<br/>　　“回来啦，回来啦。”程大雷摆摆手：“都忙着，都忙着。”
<br/>　　“杀了那老尼姑没有？”
<br/>　　程大雷瞥了他一眼，根本没有理他，这是谁带出来的兵呐，这么不会说话。
<br/>　　程大雷提着剑回了城主府，心情自然谈不上有多开心。被那白衣尼姑逃脱，没有完成任务还不太重要，关键是折损了十几个弟兄的性命，连人身上一根毛都没有留下。
<br/>　　这口气，程大雷委实咽不下去。
<br/>　　程大雷不开心，秦蛮却要比他更郁闷，出动了上千自以为的精兵强将，结果却还是人走脱了，蛮字军这次人丢得委实有些大。
<br/>　　秦蛮如今正加紧训练手下的士兵，将一腔怨气都发泄在他们身上。知道程大雷回来的消息，他也没有过来见上一面，估计是觉得每脸见人。
<br/>　　不过，这也让程大雷明白一件事，对于真正的高手，凭人海战术却是起不到太大作用。如白衣老尼这样的人物，弓箭无法对她形成威胁，近战的话，又牺牲太大。用一百条命杀一个人，这笔账不用想也知道，委实有些不值得。
<br/>　　如今，果然是江湖人施展本领的天下啊。
<br/>　　程大雷发出一声感慨，习得文武艺，货卖帝王家，真如果有一身出神入化的武艺，在这个世界是不愁没有市场的。
<br/>　　有了聂隐娘，又有了白衣尼，程大雷估计，天下要杀自己的人绝对不少，以后类似今天的刺杀还会层出不穷，自己必须想个办法应对。
<br/>　　程大雷想出的主意，是专门训练一支有江湖游侠组成的队伍，当特种兵来用。再碰到白衣老尼这样的人物，直接群起而攻之，将其灭口。
<br/>　　不过，这件事还有个麻烦，就是人手的问题。如果程大雷通过点将台召唤一批江湖高手，也确实不是太大的困难。但因为暴恶的属性，也相当于变相给自己召唤一批强大的对手。聂隐娘和白衣老尼不都是这样搞出来的么。
<br/>　　一时事情卡在这里，程大雷不知该如何进行下去。不过，这件事也并不着急，程大雷只要窝在蛤蟆城不出去，任谁有天大本事，也威胁不到自己安全。
<br/>　　不过，最近山寨还需加强防卫，免得那白衣老尼杀个回马枪。
<br/>　　……
<br/>　　白衣尼现在的情况很糟糕。本以为以自己的本事，要了程大雷的脑袋，并没有太大的困难。谁料想隐居山谷十几年今朝是头一次出山，痛痛快快的踹出了出山第一脚，结果出山第一脚就结结实实踹在铁板上。
<br/>　　摆脱程大雷的追击后，她躲在一个山洞中，原先山洞中的野兽，自然被她杀了裹腹。
<br/>　　饿了就猎山中的小兽填饱肚子，采山中的草药疗伤，如此在山洞中住了七天，七天之后方才离开大山。
<br/>　　到现在其实还没有脱离蛤蟆城的控制范围，不过回头杀程大雷一个回马枪，她实在没有这个勇气。第一次已经失利，再第二次找上门去，估计直接得把命丢了。
<br/>　　仔细回想这次行动，委实有些小看了程大雷，本以为程大雷是个酒囊饭袋，却没想到他手上的功夫竟这么硬。现在她衣衫破烂，身上挂着血污和草屑，与那些流离失所的难民没有任何区别，身上的高人风范丢得一干二净。
<br/>　　想起这件事就委实有些郁闷，然后郁闷转化成要杀程大雷的决心。可以自己现在的情况，当真也杀不了程大雷。
<br/>　　怕……还要找几个帮手。
<br/>　　她走出大山，在一座小镇上偷了一套衣服和一匹马，在偏僻的郊外收拾一番，如此，才找回一些高人风范。
<br/>　　骑着马一路向西，穿过了程大雷控制的凉州一地，路上本想着碰到几个蛤蟆城巡逻的士兵，自己好杀了泄愤，可惜，蛤蟆城好像根本没有把自己当回事，并没有派出大军搜捕自己。
<br/>　　如此，倒是对程大雷有些好奇了。这样一个山贼，又是从哪里学来一身惊人功夫，那连绵如雨的凌厉剑法，现在想起来仍然是心有余悸。
<br/>　　她一路向西，穿过草原后便是崇山峻岭，这里是人迹罕至之所在。或许几千年都未有人类踏足在此地，山上积雪终年不化，云雾绕着雪峰，宛若从古至今，不曾有一粒灰尘飘到这里。
<br/>　　来到大雪峰前，她便弃马步行，沿着山坡一路向上攀登。行不知多久，眼前不见半点绿色，只有一眼寒透心底的冰川。
<br/>　　她踏着冰雪继续前进，已快要登上雪峰，眼前峰回路转，看到半山腰的平台上立着一座茅草屋。
<br/>　　茅草屋前蹲着一个老者，老者身上裹着一件冻僵的兽皮大衣，满头白发披散下来，不知已活了多少年纪。
<br/>　　白衣老尼迈步靠近，那老者嚯地抬起头来，眼睛盯着白衣尼姑，脸上出现不可思议的惧意。
<br/>　　他忽地站起身，啊呀怪叫一声，转身就向后逃，来到悬崖边，一跃而起跳了下去。
<br/>　　白衣尼姑脸上没有太大反应，似乎很了解这人的秉性，她立在茅屋前等了一会儿。
<br/>　　过了很久，只见一个人影又从半山坡上爬了下来，一瘸一拐来到白衣老尼面前，眼神不善的盯着对方。
<br/>　　过了很久，他长长的叹了口气：“都过去这么久了，你怎么还要找我麻烦。”
<br/>(本章完)
