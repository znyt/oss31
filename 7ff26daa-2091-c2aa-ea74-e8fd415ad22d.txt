        傅夜沉仅仅是一个电话，一句命令的话，今夜通往中心医院的那条主路，响起了警车的鸣笛。
<br/>
<br/>        警车给一辆豪车开道，这样的事情，有多方面的内容可以让媒体报道撰写，甚至还能上实时社会热点新闻的头条。
<br/>
<br/>        傅夜沉在服务员推开包厢大门走进来时，季董事长连忙从座位上起身，焦虑不安地问：“傅总，我女儿她……”
<br/>
<br/>        “她平安无事，很快就会回来的。季董耐心等待便是。”傅夜沉语气温和了一些，比起刚刚那冰冷的态度，他似乎对这对父女要刮目相看了。
<br/>
<br/>        郑忠威看着自家大少爷对季家父女态度上的大转变，心里难免匪夷所思。
<br/>
<br/>        也不知道那位季小姐这是唱的哪出戏，玩个“失踪”，都能让大少爷改变对她的看法。
<br/>
<br/>        一个小时后……
<br/>
<br/>        季思妍回来了……
<br/>
<br/>        作为父亲的季董事长却被她吓傻眼了！
<br/>
<br/>        季思妍出现在包厢门口的那一刻，“狼狈”这两个字真的无法忍心用在她的身上。
<br/>
<br/>        她脸上的汗水，让精致的妆容稍稍的花了些，身上那件昂贵的绛紫色礼裙也沾满了血渍。
<br/>
<br/>        季董事长从惊愕中回过神来，连忙大步上前，就连膝盖不小心磕碰到椅子角上，他都不顾自己，直接来到女儿跟前，握住女儿季思妍的手臂，心疼不已，支吾地问：“女儿啊！你这是、你这是……”
<br/>
<br/>        “爸，我没事！”季思妍抬起手来，覆在了自己的父亲季董事长那粗糙的手背上，安慰完父亲后，抬眸看向傅夜沉，一脸抱歉地接着说，“对不起，傅总，我失态了。”
<br/>
<br/>        “你去了这么久才回来，到底是做什么去了？还有，你身上，这血渍是怎么一回事？”季董事长这才按捺不住好奇地问。
<br/>
<br/>        季思妍抿了抿唇，抬眸看向坐在那边姿态高贵又疏离的傅夜沉，赧然地微笑，又不失礼貌地说道：“爸，我真的没事！对了，傅总，我就先回去了。我这个样子，真的很失礼。”
<br/>
<br/>        比起刚刚她一口一个“夜沉哥哥”，此刻的她倒是要生疏客气了许多。
<br/>
<br/>        她是一个有自知之明的女人，会察言观色，懂得如何把握分寸。
<br/>
<br/>        季董事长无奈地瞥了自家这女儿季思妍一眼后，不得不回到席位上继续陪着傅夜沉。
<br/>
<br/>        毕竟，今天的贵宾是这位大金主。
<br/>
<br/>        傅夜沉虽然仍旧并未多看季思妍一眼，但比起先前的拒人于千里之外，此刻的他，要稍稍缓和了这包厢里的冷酷气氛：“你学什么专业的？”
<br/>
<br/>        “建筑行业，但在国外学得比较杂，学业不精。”季思妍虽然说得含糊不清，但是她说到了点上。
<br/>
<br/>        傅夜沉黑眸微眯，淡淡地接着说道：“你明天去傅氏集团旗下地利置业人事部报到。我会安排我弟弟，带你入行。”
<br/>
<br/>        “你弟弟？！”季思妍眨巴着眼睛，露出一副受宠若惊的样子来。
<br/>
<br/>        她明知故问，却又装得天真无邪的样子。
<br/>
<br/>        “他是地利置业的总经理，也是学建筑的。他虽然只有两年的工作经验，但是他的工作能力很强。你跟着他，一定能学有所成！”傅夜沉慢条斯理地说。
<br/>
<br/>        季思妍立即高兴得像个天真的小姑娘一样跺脚，双手一拍即合，笑容甜美，声音宛若抹了蜜糖一般软绵地谢道：“夜沉哥哥，太谢谢你了！”
<br/>
<br/>        傅夜沉不再吭声，而是从餐桌前站起身来，礼貌性地向季董事长欠了欠身：“季董，很谢谢您的热情款待，我先回去了。”
<br/>
<br/>        “我送你！”季董事长顿时眉开眼笑，连忙跟着从椅子上起身，向傅夜沉迎了迎手。
<br/>
<br/>        傅夜沉点了下头，温和地谢绝：“季董留步。”
<br/>
<br/>        他语毕，只身离开，特助郑忠威毫不停留片刻，一丝不苟地跟了出去。
<br/>
<br/>        季思妍目送傅夜沉离去的背影，嘴角勾起一抹意味不明的浅笑。
<br/>
<br/>        季董事长连忙走到季思妍的身边，按捺不住内心的好奇，迫不及待地问道：“小妍，你这到底是怎么一回事？”
<br/>
<br/>        季思妍看向自己的父亲，柳眉灵活自如地轻轻一挑：“傅夜沉是什么性格的人，我来之前就已经摸透了。”
<br/>
<br/>        “这跟你现在这个狼狈不堪的样子有什么关系？”季董事长怔愣地抬起手来，摸了摸自己那两鬓寒霜，一脸茫然。
<br/>
<br/>        季思妍冷不丁地白了自己的父亲一眼，嗤之以鼻地解释道：“我特意安排了一个快要临盆的孕妇和装扮成客人的产科男医生，以及两三个路人甲乙丙在酒店里待产，我算好时间后，让产科男医生给那孕妇打了催产针，然后那孕妇、医生、路人便在同一段时间内跟我一前一后断断续续地进入卫生间。我出去那么久未回来，爸爸你一定会心急地打我电话。我电话打通了没人接，你一定会急得向傅夜沉求助。傅夜沉是个头脑聪明的男人，找不着人的话，第一时间会想到要去调看酒店里的监控视频。”
<br/>
<br/>        而傅夜沉在监控视频里所看到的是，一个孕妇从卫生间里出来后，就突然面部难看地蹲在地上，从先后从卫生间里出来的路人见状，一个个怕惹是生非，疾步离开。
<br/>
<br/>        关键时刻，轮到她季思妍上场。
<br/>
<br/>        季思妍从卫生间里出来后，看到孕妇蹲坐在地上表情痛苦，于是主动上前去询问。得知孕妇腹部阵痛快要生了的时候，季思妍一边从裙兜里掏出手机拨打电话，一边左看右看寻求路人的帮助，并陪在孕妇身边寸步不离。
<br/>
<br/>        直到她安排的那个产科男医生也从卫生间里出来，她立即拦住那个产科男医生，请求他的帮助。
<br/>
<br/>        随后的监控里，傅夜沉肯定会一路锁定，最后会看到她和那个产科男医生一起搀扶着孕妇坐上了她的豪车离开酒店。
<br/>
<br/>        在季思妍看来，这一切都是她精心策划，就连时间她都精算到秒上来计数。
<br/>
<br/>        但是，在傅夜沉的眼中，这一切都是偶遇，几乎没有一丝破绽。
<br/>
<br/>        每一处细节，都体现出了她季思妍是在热心助人。
