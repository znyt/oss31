        &lt;h3&gt;第三卷第一百五十二章  冯睿的算计&lt;/h3&gt;
<br/>
<br/>        冯睿和陈子建离婚之后性情大变。
<br/>
<br/>        她变得易怒暴躁，在工作上也是频频失误，可是她不思悔改，反而变本加厉，三天打鱼两天晒网，惹得同事们敢怒不敢言，毕竟她还是领导。
<br/>
<br/>        陈子建将她收藏的东西都上缴了之后，她手头上基本没有可用的东西，她还一直拿着陈子建的工资卡，如果说她一个人的生活是够了，可是她还要隔三差五的送送人，要维持他们的关系，对于蔡少芬和弥青云来说，弥青云还好说话一点，这一次又要去北京，可是又没有什么值当的东西可拿。
<br/>
<br/>        她找出来陈子昂送她的项链，这条项链其实她很喜欢的，只是要见蔡少芬他们，之前都是大手大脚惯了，如果不拿，冯睿自己都说不过去。
<br/>
<br/>        她想到了夏颜，夏颜不是想要她引见蔡少芬她们吗，夏颜也得要出出力吧，世界上哪里有那么安逸的事情，说说话动动嘴皮子就能将事情办了？
<br/>
<br/>        冯睿给夏颜打了一个电话，约夏颜在上次南门那个商场见。
<br/>
<br/>        夏颜在洛城的项目中，她伸手了。
<br/>
<br/>        事情都按照她的想法在进行着，江俞轩果然废标了，和她站在了同一条线上，洛城的俩个项目呼延临庆都顺利中标。
<br/>
<br/>        陈子昂，你活该被背叛，你二嫂背叛了你，江俞轩、傅海生他们都背叛了你。
<br/>
<br/>        今后的西城信息项目不会再是你陈子昂的天下，而是我夏颜的天下，以前你干不过我，现在你也不是我的对手。
<br/>
<br/>        夏颜有时候会想，她此生大概就是陈子昂的克星，她注定就是陈子昂的对头，要不然她的母亲怎么会在那么多的人中选择陈子昂呢。
<br/>
<br/>        因为洛城的项目都拿下来了，呼延临庆非常开心，中午打电话请夏颜吃饭，他们两家关系一直都很好，呼延临庆上了个三本，回来上了几天班，然后开始折腾着做生意。卖过衣服、开过饭店、也随着他爸爸搞过建筑，最后都感觉不顺心，他想要找个高大上的事情，阴差阳错的在电子一条街租了个门面，折腾起电脑了。
<br/>
<br/>        要说呼延临庆也是大胆有闯劲的人，而且也是很会来事的人，要不然也不会在西城能创下一片天地，当初他和别人合伙做了一个项目，那个项目做了很多年，经过几年的努力，他们也从小白鼠便成了业界数一数二的人，因为有了那次失败，他们就更加努力，融资开始做起了研发，但是和陈子昂比起来他们那真是小巫见大巫了。
<br/>
<br/>        他和陈子昂也是同学，每次在竞争中都争不过陈子昂他们公司，当然他也是在那次聚会的时候才知道陈子昂的，陈子昂在西城是以陈子霖的名义合伙开的公司。
<br/>
<br/>        后来夏颜调到西城，夏颜手中对口的项目也很多，随着年龄和社会阅历的增长，夏颜也越来越有能力了。
<br/>
<br/>        呼延临庆找到夏颜，请夏颜帮忙，这些年夏颜经手的项目基本都是呼延临庆在做，而呼延临庆也不会让夏颜白忙活的，每个项目下来都会给夏颜算一股。
<br/>
<br/>        “颜颜，这次项目还真是按照你的布局在进行。”呼延临庆和夏颜坐在临窗的雅间里吃午饭。
<br/>
<br/>        “陈子昂时期已经过去了，你争点气，好好的把事情做起来，西城就是我们说了算。”夏颜有些恨铁不成钢的说。
<br/>
<br/>        “放心，这次项目我一定要做得漂漂亮亮的。”呼延临庆说。
<br/>
<br/>        “说得倒是好听，方案大部分还不是沿用陈子昂他们的。”夏颜对呼延临庆有些不满。
<br/>
<br/>        “这不是你害怕拿不下来吗，再说了借鉴一下他们公司的方案也好，毕竟人家还是业界大佬。”呼延临庆嬉皮笑脸的说。
<br/>
<br/>        “我发现你这脸比外面的城墙还厚了，一点也不脸红？”夏颜笑骂了一句呼延临庆。
<br/>
<br/>        “我还是脸不厚，要是脸厚就早将你追到手了。”呼延临庆看着夏颜一本正经。
<br/>
<br/>        “滚，看你猥琐的样子我都吃不下去饭。”夏颜敲了一下呼延临庆的碗。
<br/>
<br/>        “我要是从这十楼滚下去就粉身碎骨了，好歹我还是你亲亲的同学吧。你忍心啊？”呼延临庆和夏颜开着玩笑。
<br/>
<br/>        “我亲亲的同学多了去了，陈子昂也是。”夏颜的思绪飘了。
<br/>
<br/>        “得了，我算是明白了，你这一生怕是和陈子昂不死不休了。说真的，当初我们谁都没有想到穷鬼陈子昂会走到今天这个份上。你知不知道，我们现在坐的这个地方是谁开发的吗？”呼延临庆问夏颜。
<br/>
<br/>        夏颜沉默不语。
<br/>
<br/>        “陈子昂的五姐，听说也有陈子昂的股份在，光这一个商场都肥的流油，何况他们的超市还遍地生花，我，靠，陈家不知道怎么了祖上冒青烟了。”呼延临庆摇动着他那肥肥的脑袋。
<br/>
<br/>        “你还吃不吃饭？”夏颜心里不舒服极了，陈子昂走到今天也算是她促成的吧。
<br/>
<br/>        “其实，陈子昂应该感谢你，她还说她不计较，真的是没有良心。”呼延临庆又添了一把火。
<br/>
<br/>        “没良心的人多了去了，也不光光是陈子昂一个人，难道我都要去生气？”夏颜知道呼延临庆故意激将她。
<br/>
<br/>        俩个人正聊得欢，冯睿的电话打过来了。
<br/>
<br/>        “颜颜，我准备去北京了。”冯睿看着手中的项链说。
<br/>
<br/>        “冯姨是想我和你一起去吗？”夏颜望着楼下来来往往的人群。
<br/>
<br/>        “不，我这次先去给她们提前说一下，等过年期间我带你一起去。”冯睿稳了稳心神。
<br/>
<br/>        “好的，冯姨，那辛苦你了。”夏颜听说冯睿过年带着她一起去见陆家和张家的人，心里很高兴。
<br/>
<br/>        “不过，颜颜，这次去，怕是要带些东西，你在哪里？陪我一起去商场看看？”冯睿点拨着夏颜。
<br/>
<br/>        夏颜立马就明白了冯睿的意思，她看了一眼呼延临庆，对冯睿说：“我在中心街安阁大厦，要不你过来，我在这里等你。”
<br/>
<br/>        “等一会要来一个人，比较重要，后期我们还得让她帮着，我工作上的事情也是她帮了一把。你今天大概要出出血了。”夏颜看着呼延临庆说。
<br/>
<br/>        “说什么话呢，咱俩谁跟谁，别说是出点血，我这一身肉你全拿去都行。”呼延临庆大气的说。
<br/>
<br/>        冯睿到达商场的时候，夏颜和呼延临庆正在喝茶等她，夏颜看到冯睿到了便带着呼延临庆和冯睿去了三楼专门卖珠宝地方。
<br/>
<br/>        俩个小时下来，冯睿不但是拿到了去北京的上门费还给自己赚了一把，心满意足的回去了。
<br/>
<br/>        嗯，这个夏颜还不错，能知恩图报，要好好的利用这层关系。
