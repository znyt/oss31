　　伴随着慵懒的呻吟声，慕清霄眼角抽了抽，指尖的动作，却没有一丝停止的迹象，反倒是加快了一些。
<br/>　　小医仙想要忍耐，可就是压抑不住。
<br/>　　指尖移动的速度越来越快，在她敏感的部位......
<br/>　　小医仙感觉太过愉悦，羞涩的同时，俏脸涨红的犹如一颗红苹果。
<br/>　　随着封印渐渐成型，小医仙体内的毒气，终于是冲破了原本封印，浓郁的紫色毒雾，自玉体上冉冉升起。
<br/>　　随着毒雾越发的浓郁，甚至扑面而来，慕清霄脸色慎重，指尖依旧没有停止。
<br/>　　毒气虽强烈，但对他来说，根本无关紧要，现在最优先的是要封印厄难毒体，否则，后果不堪设想。
<br/>　　片刻后，慕清霄指尖陡然加速，一道道黑色火痕，在其指下，迅速形成，手指舞动，几乎划出一片残影。
<br/>　　旋即，指尖顿住，火苗一窜，火痕两头连接在一起，形成一个完美的封印。
<br/>　　在封印形成的同时，小医仙体内的毒气也停止涌出。
<br/>　　至于早就离开她体内毒气，则弥漫在屋里每一个角落，哪怕斗灵进来，恐怕都撑不多久，便会毒发生亡。
<br/>　　不理会屋内弥漫的毒气，指尖神火大盛，轻轻落在小医仙平坦没有一丝赘肉的小腹上。
<br/>　　腹部正是封印的阵眼，只要将能量灌入其中，就能瞬间激活封印。
<br/>　　旋即，炽热的能量顺着他的指尖，涌入阵眼，最后顺着火痕，向四面八方涌去。
<br/>　　直到最后，炽热的能量将玉体覆盖，小医仙身上的神秘封印，发出淡淡的黑色光晕，极其绚丽。
<br/>　　“天为形，火为灵，天都火印，封！”轻唱声落下，慕清霄掌心，黑芒大盛。
<br/>　　待黑芒消散，小医仙玉体上的火痕也随之消失，但腹部却出现一副黑色火焰图案，图案妖异，摄人心弦。
<br/>　　成功封印厄难毒体后，小医仙美眸也是在一瞬间睁开，粉唇轻启......
<br/>　　旋即，弥漫屋内每一个角落，浓郁的紫色毒雾，如同受到牵引一般，顺着气流，尽是涌进粉唇中。
<br/>　　当最后一丝毒雾也被小医仙吞入腹中，她的俏脸上顿时泛起一抹奇异的神采，美眸扑闪扑闪的。
<br/>　　同时，她的气息也在吞噬毒雾后，开始攀升，仅仅两分钟的时间，便直接从八星斗师，步入四星大斗师。
<br/>　　如此速度，若是让迦南学院中，那些所谓的天才看见，恐怕会惊得下巴都掉在地上吧。
<br/>　　此次，厄难毒体完美封印，并且肆意的毒雾，更是令得小医仙修为暴涨，当真是祸福相依。
<br/>　　气息停止，感受到体内温热的能量，就像泡在温泉中一般，完全没有一丝的难受，小医仙俏脸上浮现惊喜之色。
<br/>　　“真......真的封印了，好舒服的感觉。”
<br/>　　说着，还伸出纤纤玉指，摸了摸小腹上阵眼所在，浑身都暖洋洋的，厄难毒体就像消失了一般。
<br/>　　没等她在愉悦感中回过神来，便感觉被一股阳刚的气息压在了身上，美眸望着近在咫尺的俊逸脸庞，含情脉脉。
<br/>　　“仙儿，你是想诱惑死为夫吗？”慕清霄坏笑一声，俯下身子，就吻住那诱人的红唇。
<br/>　　“唔唔...!”
<br/>　　感受到他霸道的亲吻，小医仙捏着粉拳，在他胸膛上轻轻敲了几下，但很快就沦陷在其中。
<br/>　　指尖下滑，落在挺翘的玉女峰上，轻轻揉捏......
<br/>　　最后，向着最为神秘的地带滑去。
<br/>　　“唔！”
<br/>　　小医仙玉体一僵，美丽的睫毛颤了颤，却感觉玉体上手指停止了动作，渐渐睁开美眸。
<br/>　　望着上面的青年，小医仙抿着粉唇，眨了眨大眼睛，忍不住开口问道：“怎么停下了？”
<br/>　　慕清霄能察觉到她的柔情，俯下身子，在她玉额上轻轻一吻，语气温柔。
<br/>　　“等你将厄难毒体完全掌控后，到时候，我们在继续刚才没有做完的事情，相信不会太久的。”
<br/>　　“......嗯。”
<br/>　　小医仙嫣然一笑，乖巧的点了点头，莲藕般的玉臂，轻轻的环着他，美眸中尽是幸福之色。
<br/>　　她本就聪慧，自然清楚，慕清霄现在忍得极其幸苦，恐怕就是担心，继续下去，会影响厄难毒体的封印。
<br/>　　躺在床榻上，慕清霄温柔的抚摸着她的玉体，思绪万千，最终神色变得坚定起来。
<br/>　　“仙儿，我有一件事情要与你商量。”
<br/>　　“嗯。”
<br/>　　见她乖巧的模样，慕清霄有些无奈。
<br/>　　“厄难毒体暂时封印了，封印期少则五年，多则八年，但是，你想要完全掌控厄难毒体，必须要前往中州。”
<br/>　　说着，慕清霄叹了口气，道：“除了厄难毒体，中州也有我需要的东西，但以现在的实力，却是寸步难行。”
<br/>　　中州，那片偌大的地域中，斗皇多如狗，斗宗满地走，斗尊也是数不胜数，斗圣级别的超级强者，也不是没有。
<br/>　　而他现在的实力，顶多也就能抗衡斗宗强者，遇到斗尊，恐怕就只有逃遁的份，更别说取得异火，身边还要带着小医仙。
<br/>　　闻言，小医仙紧紧依偎在他的怀中，似乎猜到了些什么，却是乖乖的听着。
<br/>　　“所以，我暂时需要离开一段时间。”望着怀中闭目的佳人，慕清霄认真的说道。
<br/>　　许久，小医仙睁开美眸，其中泪光闪烁，声音柔弱，道：“需要离开多久？”
<br/>　　“少则两三年，多则......三五年。”
<br/>　　慕清霄也是于心不忍，但必须狠下心来，他的实力需要提升，而小医仙也需要经历一番磨练。
<br/>　　贝齿紧咬着粉唇，玉体深深的埋在温暖的怀抱中，恨不得融进他的身体里。
<br/>　　想到三五年都无法见面，小医仙心中就觉得有些委屈，但她不是无理取闹的女人。
<br/>　　“迦南学院中很安全，在这里等我回来。”
<br/>　　“嗯，我会等你回来。”
<br/>　　纵然有千言万语，她现在此刻都只能装在心中，因为她清楚，现在的自己，只会成为拖累。
<br/>　　两人紧紧相拥，彼此都没有再开口说话。
