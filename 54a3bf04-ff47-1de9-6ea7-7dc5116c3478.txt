　　第61章
<br/>　　“凡从蛤蟆岭下经过的人，都须交二成过路费。”程大雷敲着立在身边的木牌：“童叟无欺。”
<br/>　　“程当家，我劝你莫要欺人太甚！”陆哼已咬紧牙关。
<br/>　　“喂，搞搞清楚好不好，这是有薛城主签字盖章的。欺人太甚这个大帽子请不要乱扣。”程大雷没有生气。
<br/>　　“没钱。”陆哼口中迸出两个字，他想明白了，对付程大雷这样无赖的人，就要用同样无赖的态度对待他。
<br/>　　“这样说就不对了，钱，就像海绵——毛巾里的水，挤挤总是有的。陆将这么大身份，总不会一块铜板都没有。”程大雷眨眨眼睛：“何况，以你们现在的状态，有把握平安走出青牛山么，我们蛤蟆寨拿了钱，可是有护送服务的喔。”
<br/>　　陆哼心里咯噔一下，他这次的经历十分悲惨，连兵器都被幽州王扣下，包括陆哼的铁棍。所以现在陆哼这一队人是手无寸铁，如果青牛山的群贼要痛打落水狗，他们未必有把握活着离开青牛山。
<br/>　　如果有蛤蟆寨保驾护航，这自然不算什么，不管怎样，程大雷现在的面子是有的。
<br/>　　可问题是，陆哼现在真的没钱。
<br/>　　“不如，留两个人给我。”程大雷说着，目光扫过陆哼身后衣衫褴褛的士兵。蛤蟆寨现在缺人，程大雷不得不把主意打在陆哼身上，不管是乌龟还是甲鱼统统来者不拒。
<br/>　　小不忍则乱大谋，小不忍则乱大谋……陆哼不停用这两句话安慰自己，目光看着自己身后的士兵，忽然双手一抱拳道：“抱歉，他们是我的兵，我带着他们出来，也会带着他们回家。”
<br/>　　说着，陆哼从身上掏出一块玉佩，递到程大雷面前。
<br/>　　“这枚古玉……权当是我的过路费。”
<br/>　　程大雷一怔，忽然笑笑，没有接那枚玉佩道：“请，秦蛮，送陆将军过山。”
<br/>　　陆哼眼神严肃起来，抱拳道：“程当家请了，山水有相逢，咱们下次再回。”
<br/>　　看着陆哼落魄离开的背影，程大雷明白，下次见面，怕就是要用兵器说话了。
<br/>　　这一幕，对柳家众人的冲击不小。早听说蛤蟆大王在青牛山如何如何，只以为是传言以讹传讹，眼前却看到，就连官兵过山，都要给程大雷交保护费，那这传言还能是假的么。
<br/>　　嘟，收到恐惧值10
<br/>　　嘟，收到恐惧值9
<br/>　　嘟，收到恐惧值……
<br/>　　程大雷怔了怔，却是没想到这样也能收到恐惧值。他看来山下众人一眼，上山后继续自己的招聘工作。
<br/>　　这一幕又令几个人动摇，上山准备加入蛤蟆寨。等到天黑的时候，大概有五十几人加入蛤蟆寨，剩下三十几人，一些是被程大雷淘汰掉的，另外一些是以柳慕云为首的柳家血亲，柳芷和孟子云都在其中。
<br/>　　程大雷站在山上看着他们，越过初起的山雾双方目光对视，宛若一场对峙。
<br/>　　程大雷没有和他们对峙，他令人搬起桌子，将寨门关闭。
<br/>　　有一个夜来临，相比之下，这一夜要比昨天更加难熬。饥饿折磨着他们的心灵，更痛苦的是，明明不远的距离就能得到热腾腾的饭菜。
<br/>　　蛤蟆寨的待遇还是相当不错的，晚饭的饭菜香味已飘到这里，令所有人垂涎欲滴。
<br/>　　人群里想起纷纷的抱怨声，一开始这声音极小，但越来越大。
<br/>　　已经有人开始埋怨柳慕云当时为何怂恿大家要发难程大雷，难道荣誉能当饭吃么。
<br/>　　“当初人家待我们差么，把我们从奴隶营赎出来，给我们准备食宿。是我们好心当成驴肝肺，对不起程当家。”
<br/>　　“要我说，程当家还是很仗义的……”
<br/>　　“嗯，程当家人不错。”
<br/>　　程当家不错，那谁是错的，自然便是怂恿大家，坚守所谓荣誉的柳慕云。
<br/>　　权威在被瓦解，君君臣臣父父子子，与生俱来的道德观正在崩溃。
<br/>　　柳慕云孤坐在火堆前，一夜无话。
<br/>　　次日天明，程大雷又一次摆出桌子，今天又有几个人上山，但以柳慕云为首的人凝固得更强。
<br/>　　终于，在下午时，柳芷缓缓上山。
<br/>　　“程当家，您能不能下山与柳爷爷他们谈谈，他们心里其实还是很感激程当家的。”
<br/>　　程大雷其实明白，这些人的坚守已经动摇，现在他们只是需要一个台阶，方便能舒舒服服走下来。
<br/>　　但这个台阶程大雷没准备给，你爬得多高，就结结实实给我摔下来，高，是你自己爬上去的。
<br/>　　“柳家姑娘，我实在也很想和柳老先生谈谈，可你看我这里这么忙，实在走不开啊，改天吧，改天吧。”
<br/>　　柳芷心中一凉，不得不承认，当程大雷从天而降救了她时，她是有那么一两刻动心的。然而，可这种冷冰冰的对决，并没有柔情的存身之地。
<br/>　　她黯然下山，形单影只。
<br/>　　到下午时分，已经再无人上山，山下留着的，只剩下二十几人。
<br/>　　一方在山上，一方在山下，谁都看得见彼此，目光冰冷。
<br/>　　在天将黑时，柳芷再一次上山，面对她楚楚可怜的神情，程大雷展现了自己从未有过的冰冷。可柳芷开口的第一句，却令他目瞪口呆。
<br/>　　“柳家愿意将我献给程当家，只求程当家能够开恩，收留柳家老小，给他们一条生路。”
<br/>　　“献给……”程大雷睁大眼睛：“什么意思？”
<br/>　　柳芷递过来一张纸，上面大意是将柳芷送给程大雷，以换取程大雷能收留柳家老小，下面还有柳慕云为首几个老人的画押。
<br/>　　『献给』的意思是，只要程大雷点头，他将拥有柳芷的所有权。
<br/>　　这份权力不是以人的方式，柳芷做程大雷妻子、妾室之类的，而是以物的方式。程大雷将完全拥有柳芷的身体乃至心灵，他可以令对方为他铺床叠被，暖床洗脚，也可以转手将她送给别的人。
<br/>　　在这件事里，柳芷完全没有自己的自主权，她的生命以及余后的人生都是属于程大雷的。
<br/>　　程大雷领悟到这一点后，愣了很久，半晌冒出一句稀里糊涂的话。
<br/>　　“这么简单！”
<br/>　　
<br/>(本章完)
