        “咔”地一声，实验室的这张厚重的防弹钢门突然向两边缩进了门缝里。
<br/>
<br/>        连华生吓得往后退了一步，回神之际，一只银色的麻醉枪抵在了他的额头上。
<br/>
<br/>        定睛后，他才看清楚，这个持枪的人，竟然是童筱颖。
<br/>
<br/>        童筱颖面色惨白，眸子里黯淡无光，长长的秀发像瀑布一样从双颊两边垂至腰间，她身上穿着白色宽大的睡裙，整个人比以前要消瘦了许多。
<br/>
<br/>        “童、童、童筱颖？”连华生难以置信地瞪大了眼睛。
<br/>
<br/>        虽然裴金澈已经事先告知过他童筱颖并未死的时候，但是若不是他连华生现在亲眼所见，还真的会怀疑是自己见鬼了。
<br/>
<br/>        毕竟，半年前，他找到她的时候，她已经断气了。
<br/>
<br/>        看来，裴金澈说得一点都没错，他连华生的父亲，这么多年来的呕心沥血，已经成功了。
<br/>
<br/>        怪不得，他们要继续拿她童筱颖做实验。
<br/>
<br/>        连华生暗自思忖。
<br/>
<br/>        “你认识我？”童筱颖面无表情地问，语调平缓，没有一丝感情。
<br/>
<br/>        连华生将她上下打量了一番，见她左手上还提着一个银色的药箱，下意识地举起手来：“对！我认识你！我们以前是最要好的朋友。”
<br/>
<br/>        “连、连少，快拦住她，不要放她走！”实验室门内，突然传来一个女人吃力的呼喊声。
<br/>
<br/>        连华生脸色一僵，一种不祥的预感漫上心头，耳边只听到“嗖”地一声，额头上便传来一阵刺痛，下一秒，他便倒在了地上。
<br/>
<br/>        合眼前，他看到童筱颖往一旁扔了手上的麻醉枪，转身离去。
<br/>
<br/>        她……
<br/>
<br/>        好像……
<br/>
<br/>        真的已经不是以前的那个童筱颖了。
<br/>
<br/>        另一边，傅氏集团旗下的高管公寓里。
<br/>
<br/>        裴金澈掐住夏玄灵的腰肢，不停地起起伏伏。直到门铃响起，两人不约而同地从意乱情迷中清醒过来。
<br/>
<br/>        “这么晚了，会是谁？”夏玄灵不禁感到纳闷。
<br/>
<br/>        裴金澈抽身下了床，捡起地上的浴袍穿上，踏上拖鞋走出了房间。
<br/>
<br/>        来到玄关的大门前，通过门边的可视电话，看清楚门外的来访者后，他立即将大门给打开了。
<br/>
<br/>        “你是怎么找来的？”裴金澈伸手将童筱颖拉进了屋，立即关上了大门，见她手上还提着一个银色的药箱，又疑惑不解地问道，“你手里拿的是什么？”
<br/>
<br/>        童筱颖二话不说，直接往客厅走去。
<br/>
<br/>        卧室里，听到外面的动静，夏玄灵将衣服穿好后，也走了出来，去了裴金澈的跟前：“这么晚了，是谁？”
<br/>
<br/>        “喂！你是怎么离开实验室的？”裴金澈不管夏玄灵的问话，直接走到了童筱颖的面前。
<br/>
<br/>        夏玄灵也跟了过去，看到童筱颖的脸时，足足地吓了一跳：“她、她不是死了吗？”
<br/>
<br/>        童筱颖完全不理会这两个人的“十万个为什么”而是十分淡定地打开药箱，从里面拿出了一盒药剂递给裴金澈。
<br/>
<br/>        “这是你的解药，每隔一个星期注射一次，三次一个疗程。你的病就能痊愈，以后，你就是正常人了。”童筱颖波澜不惊地说。
<br/>
<br/>        裴金澈接过药仔细地看了看后，难以置信地反问：“你居然偷了连叔叔的药！还有，你是怎么逃出来的？”
<br/>
<br/>        这盒药剂，就是连华生的父亲他们团队的最后研究成果。
<br/>
<br/>        童筱颖于他们而言，即是小白鼠，也是药引，也叫解药。
<br/>
<br/>        第一个实验对象就是童筱颖，自我修复成功。而第二个实验对象就是得了癌症的小白鼠，其体内的癌细胞也得到了抑制。
<br/>
<br/>        等到下个月，他们就正式投入市场，以高价卖给那些得了不治之症的富豪，这就是他们想要把童筱颖变成植物人的原因。
<br/>
<br/>        “实验室里有人帮我。”童筱颖定了定神，接着说道，“她告诉我，只要把你治好了，没有了你这个辅药，他们就无法再继续下去。”
<br/>
<br/>        “你确定，你带给我的药剂，能治好我？”裴金澈狐疑。
<br/>
<br/>        童筱颖合上药箱，从沙发上站起身来：“我不确定，但是你可以试试。试了，就算是失败了，你也不会死。不试，我两都得死。”
<br/>
<br/>        “好！我试试。”裴金澈点了点头。
<br/>
<br/>        夏玄灵一直躲在裴金澈的身后，畏畏缩缩地看着童筱颖。
<br/>
<br/>        这个女人的皮肤有些病态白，披头散发，又穿着宽松的白色睡裙，脚上还没穿鞋子，真的就像女鬼一样吓人。
<br/>
<br/>        童筱颖走之前，忽然间想到了什么，于是从裙子口袋里掏出了一张纸递给裴金澈，问道：“你认识我画的这个男人吗？”
<br/>
<br/>        裴金澈放下手中的药剂，又接过童筱颖递过来的纸，展开来看了看。
