<br/>　　“是什么东西？”曾书书瞪大了眼睛，向四周望去，不忘向法相和李洵问道。
<br/>　　法相喘了口气，道：“毒虫，估计要来了。”
<br/>　　话语刚落，众人凝神戒备，只见在火光和轮回珠金色光芒照耀下，周围森林深邃的黑暗中，响起窸窸窣窣的声音，渐渐的黑影攒动，自四面八方包围了过来。
<br/>　　等黑影到了近处，张小凡等人仔细一看，顿时都倒吸了一口凉气。
<br/>　　只见周围不断从黑暗中爬出各种毒蛇，色彩缤纷，最大的只有婴儿胳膊粗细，但胜在数量，与地面摩擦，吐着芯子，发出毛骨悚然的声音。
<br/>　　饶是张小凡等人修为不俗，此刻脸色也是苍白了几分，但毒蛇对火堆有些畏惧，虽然靠的近了，却也只是在两丈外，并没有靠近内圈。
<br/>　　斩龙剑出鞘，林惊羽低喝一声，道：“何方妖孽，在此装神弄鬼。”
<br/>　　陡然，黑暗中响起一声轻笑，道：“几位便是正道门派弟子吧，刚才可是你们向我先动的手......”

<br/>　　亿年神木上，慕清霄站在粗壮的树干上，向四周看了看，目光顺着奇异的藤蔓，最后定格在色彩缤纷的花海上。
<br/>　　他隐隐感觉，面前的这些花朵不普通，生长在神木上，难道是某种灵药不成？
<br/>　　“系统，这些花朵是灵药吗？”
<br/>　　“叮，回答此问题，宿主需支付5000掠夺积分。”
<br/>　　系统的提示声响起，慕清霄脸上肌肉狠狠的抽了抽，险些一句‘马卖枇’脱口而出，同时心中也有些喜悦，看来是某种灵药没错。
<br/>　　想到这里，慕清霄取出封印用的容器，灵气一卷，周围色彩缤纷的不知名灵药，全部化作长河，不断莫入容器中，既然是灵药，自然没有留给别人的道理。
<br/>　　待天帝宝库周围的灵药全部封入容器，慕清霄微笑道：“别看这些花朵普通，很有可能是罕见的灵药，下面还有不少，我们一并摘走。”
<br/>　　闻言，陆雪琪点了点头，道：“好。”
<br/>　　话语刚落，慕清霄就搂着她化作一道剑光，纵身一跃，向下方疾驰而去，向下的速度自然比向上要快许多，几分钟的时间，就来到神木的中央地段。
<br/>　　此处，奇异的藤蔓上，生有许多灵药，虽不及天帝宝库外面的多，但数量也着实不少。
<br/>　　将灵药全部搜刮干净后，整颗神木上就剩下光秃秃的奇异藤蔓，慕清霄满意的点了点头，道：“各门各派都是冲着天帝宝库而来，我们就此离开吧。”
<br/>　　“嗯，不过，我想先回一趟青云门，与师尊道别，到时候就随你浪迹天涯。”陆雪琪温柔的说道。
<br/>　　慕清霄轻轻将她搂入怀中，道：“不用浪迹天涯，我会带你去一个温馨的家。”

<br/>　　......
<br/>　　半个月后的清晨，一道剑光落在青云山脚下，剑光散去，两道身影显现出来。
<br/>　　慕清霄望着熟悉的青云山，心中一番感慨，转身看向陆雪琪，道：“我就在这山脚下等你，速去速回。”
<br/>　　“嗯，我很快就回来。”陆雪琪露出一丝温柔的笑容，随即化作一道蓝光向青云山上御空而去。
<br/>　　待陆雪琪离开，慕清霄深深的望了一眼大竹峰方向，狠下心道：“系统，消除本世界，任何与我有关的记忆。”
<br/>　　“叮，消除本存在于本世界中的记忆，需要1000掠夺积分。”
<br/>　　慕清霄深吸了口气，青云门终究是他人生中的过客，在大竹峰六年，其中点点滴滴，生活的记忆，他会永远记在内心深处。
<br/>　　“扣除！”
<br/>　　“叮，掠夺积分扣除成功，记忆开始清除......清除成功，关于宿主的信息，记忆已全部从世界中消除。”
<br/>　　待记忆清楚，慕清霄总觉得自己轻松了许多，道：“系统，我要待雪琪离开本世界。”
<br/>　　“让剧情女主脱离本世界，需扣除4000掠夺积分。”
<br/>　　没有丝毫犹豫，慕清霄语气平淡，道：“扣除。”
<br/>　　“叮，宿主随时可携带剧情女主脱离本位面，检测到宿主完成所有任务，开始统计位面奖励，请稍等......”
<br/>　　闻言，慕清霄怔了怔，真是久违，记得当初统计奖励的时候，给自己带来了两本功法，御剑篇章，段体篇章，不知道这次会获得什么奖励。
<br/>　　而且，不知道是不是错觉，他总感觉，系统的声音，比以往多出了一丝人性化，而不是冰冷的机器。
<br/>　　“难道，系统真的有人性化的情绪，只是没有表露出来？”慕清霄暗自想道，但这种想法刚刚升起，便被他掐灭，估计是自己想多了吧。
<br/>　　“叮，统计结束，宿主掠夺积分达标十万，掠夺商店开启，另外，宿主可从以下三件物品中，任选一件成为奖励。”
<br/>　　系统提示声刚落，慕清霄脑海空间中就出现三样东西：
<br/>　　第一件物品，是一颗红色的丹药，丹药表面上有复杂的银色纹路，（圣灵丹）效果：可提升宿主资质，修为提升至化神六重中期！
<br/>　　第二件物品，是一块乳白色的玉石，平淡无奇，（物品未知）效果：未知。
<br/>　　第三件物品，是一柄长剑，长剑锻造的堪称完美，其散发的气息，比魂剑强上不知道多少倍，品级：（玄品）！
<br/>　　望着脑海空间中的三件物品，慕清霄暗自吞咽了口唾沫，心情可想而知！
<br/>　　第一件物品，圣灵丹，看到它的效果后，慕清霄险些将舌头都给咬掉，能直接让自己的修为，提升一整个层次，若非他心智坚定，恐怕想也不想的选择了圣灵丹。
<br/>　　接着，目光落在第二件物品上，这是一卷玉简，没有记载来历，其中内容也是未知，但不知为何，慕清霄却自玉简上，感受到一股熟悉的气息。
<br/>　　微微摇头，目光落在第三件物品上，玄品长剑，慕清霄虽不清楚玄品代表什么，但从上面的气息来看，绝对比魂剑高出好几个等级。
<br/>　　慕清霄沉吟片刻，最终心神落在了中间的玉简上，魂剑的等级，迟早会甩开这柄玄品长剑，魂剑可是他的灵魂锻造的，根本不是这柄钒铁能比的！
