夜滢浑身一震，那些过往现在被厉老夫人给无情的揭开，她依然觉得难堪。
<br/>
<br/>        当年她是夜家的千金大小姐，就因为她在人群里多看了这位帝都之子厉君墨一眼，她的一颗芳心就都掉在了他的身上，几乎到了入魔的地步。
<br/>
<br/>        为了得到他，夜滢找了一个机会在厉君墨的杯子里下了药，她本来打算趁此机会将生米煮成熟饭，让自己变成厉君墨的女人。
<br/>
<br/>        可是…
<br/>
<br/>        可是半途发生了意外。
<br/>
<br/>        厉君墨这个人生性无比的警觉，他很快就察觉到了身体的异常，所以他并没有回自己的房间，从而让她扑了一个空。
<br/>
<br/>        那中间有一段空白，夜滢也不知道发生了什么事情，她在酒店里四处寻找厉君墨，最后在一个房间里找到了！
<br/>
<br/>        夜滢现在还可以清晰回忆起当时的一幕，她走到那个房间门口，房门正好打开，里面跑出来一个女人。
<br/>
<br/>        那个女人撞到了她的身上，因为速度太快了，她并没有看清那个女人的脸，只看到那个女人衣衫不整的露出了半个莹润的肩头，她那漂亮的蝴蝶扇骨下方有着一朵鲜红的花儿。
<br/>
<br/>        那花儿并不像是纹身，而像是从肌肤里自然生长出来的，花儿鲜活到妖冶，让人看的勾魂夺魄。
<br/>
<br/>        夜滢就看了那么匆匆的一眼，却再也没有忘记过。
<br/>
<br/>        当时她觉得很奇怪，等那个女人跑走了，她就站在门口向房间里面看去，这一看她直接忘记了呼吸。
<br/>
<br/>        她在房间里看到了厉君墨，厉君墨躺在柔软的大床上，那地毯上都丢的衣服，床单凌乱，一看就是激情过后的战场。
<br/>
<br/>        她僵住了，一个可怕的认知在脑海里盘旋，那就是她为他人做了嫁衣裳，厉君墨进错了房间。
<br/>
<br/>        那个跑走的女人成了厉君墨的女人！
<br/>
<br/>        夜滢不知道那个女人是谁，因为那个女人再也没有出现过，也没有回来找厉君墨，帝都之子厉君墨无人不知，恰值风华绝代的年纪，有人爬上了厉君墨的床，竟然没有回来找厉君墨，这一直是夜滢这些年深藏在心底的秘密。
<br/>
<br/>        不管怎样，错误已经造成了，她最应该考虑的是如何自救。
<br/>
<br/>        她等这个机会等了太久了，对于厉君墨而言，发生过一次的事情，绝对不会再发生第二次，她错过了这个机会，就永远无法得到厉君墨了。
<br/>
<br/>        于是当时的她做了一个重大的决定，她毅然的走进了那个房间里，趁厉君墨还没有苏醒的时候脱了衣服，躺在了他的身边。
<br/>
<br/>        后来厉君墨睁开眼时，第一眼看到的就是她！
<br/>
<br/>        后来的后来，她就怀孕了，厉君墨为了她肚子里的这个孩子娶了她，她借腹上位，生下了厉嫣然。
<br/>
<br/>        厉君墨根本没有碰她，无论是在婚前还是在婚后，所以她的女儿厉嫣然…根本不是厉君墨的种！

<br/>        当然了，厉嫣然的身世没有人知道，她也不会让任何人知道的。

<br/>        一旦厉嫣然的身世曝光，那一切都会完了。
