　　　　
<br/>　　慕清霄右手抚摸着她雪白嫩滑的脸颊，陆雪琪美眸注视着他的一举一动，知道这流氓又不正经了，但心里却甜滋滋的，有些享受。
<br/>　　他虽然喜欢美色，却也不会因为美色耽搁了正事，道：“我的小美人，等离开万蝠古窟就吃了你。”
<br/>　　闻言，陆雪琪芳心‘砰砰砰’的跳动起来，美眸娇羞无限，只是风情万种的白了他一眼，随后不再说话。
<br/>　　七颗红色石子倒影慢慢浮现出来，像是一个手掌，静静的漂浮在水面，慕清霄对准位置，伸出右手，在手掌的位置按了下去。
<br/>　　水潭很浅，右手很快就接触到了潭底，有一层沙石薄薄的铺在水底，很快，慕清霄就感觉到手下有五个稍稍凸起的地方，正是对应着手指的五个指尖。
<br/>　　慕清霄知道这就是开启滴血洞的机关，也不再犹豫，直接按了下去，下一秒，一阵刺耳且沉重的“咯吱”声在山洞中响了起来。
<br/>　　两人同时向声音源头看去，只见在水帘背后，曾经天衣无缝，坚硬至极的石壁，居然整块的向后退了进去，虽然缓慢，却终于露出一个新的洞口。
<br/>　　陆雪琪来到岸上，站在慕清霄的身旁，脸蛋上更有水珠滑落，清丽无双，美眸中尽是笑意与温柔。
<br/>　　慕清霄牵起她的芊芊玉手，道：“我们进去吧。”
<br/>　　陆雪琪微微点头，再次踏入水中，穿过水帘，随他走进新的洞穴中。
<br/>　　这是一条幽深的隧道，洞侧石壁上发光的萤石明显比外边通道少了许多，光线暗淡，可对慕清霄来说，却犹如白昼。
<br/>　　两人走的都挺悠哉，陆雪琪觉得安全感十足，慕清霄则用神识探路，毕竟八百年来这是第一次有人进入滴血洞，黑心老人又是元婴修士，谁也不知道他会不会在洞内布置厉害的禁制。
<br/>　　走了片刻，路上倒也太平，并没有发生任何意外，只是这条通道颇为曲折，又深又长，而且慢慢向上，慕清霄心里估算，只怕他们已经深处山腹的中心地带。
<br/>　　脚步声消散，慕清霄停下脚步，轻声道：“到了。”
<br/>　　抬头向前看去，只见在前方隧道尽头，一丝明亮的光线照了过来，那里隐隐看见是一个偌大的石室，两人对望一眼，迈步前进。
<br/>　　随着两人接近，最终也看清了石室的情况，整个石室呈圆形形状，隧道正在石室的中中间，而在它的对面，还有一条通道向里延伸，并不是唯一的尽头。
<br/>　　在石室的左边，放着两尊巨大的石刻雕像，一尊慈眉善目，微笑而立，一身衣裳被刻的如风中吹拂栩栩如生，到有点像是佛门的观音菩萨。
<br/>　　另一尊却完全是不同的模样，狰狞凶恶，黑脸鬼角，八手四头，甚至在嘴角处还有一丝鲜血流下，令人看了不寒而栗。
<br/>　　此外在这两尊的雕像面前，还有一张石桌，上边一个香炉，旁边放着几包香烛，都是灰尘遍布，显然这八百年来从未有过香火。
<br/>　　至于这石室的另一头，却只有几个蒲团，随意地仍在地上，没有任何其余的东西。
<br/>　　陆雪琪望着两尊雕像，道：“不用祭拜吗？”
<br/>　　慕清霄眼中闪过一丝不屑，轻声道：“我连斗战胜佛都不拜，他们有什么资格让我祭拜。”
<br/>　　“斗战胜佛？”
<br/>　　“是一只猴子。”
<br/>　　陆雪琪微微一笑，不明所以，便紧随其后，走向洞穴深处，没过多久，两人又进入一处宽敞的地方。
<br/>　　但这里却不像外面的石室装修过，而是一个钟乳倒悬怪石突兀的山洞，洞里各色钟乳石千奇百怪，颜色也是异彩纷呈。
<br/>　　而在两人的面前，洞口处立着一块巨大的石碑，上边龙飞凤舞地刻着十个大字：“天地不仁，以万物为刍狗！”
<br/>　　这十个大字，每一字几乎都有半人大小，笔意古拙，笔势苍劲，直走龙蛇，竟有迎面而出，呼啸苍穹之势。
<br/>　　慕清霄嘴角勾起一丝笑意，陆雪琪也察觉到了他脸上的笑容，温柔道：“这是你需要的东西吗？”
<br/>　　“没错，我们先去别处看看，待会在来此处。”
<br/>　　“好。”
<br/>　　跃过石碑，洞底是一面光滑的石壁，石壁两侧各有一条隧道，通往不知名处，但在这石壁下面，却是一块青石平台，上面竟有一具骷髅成端坐形状，安静的坐在那里。
<br/>　　慕清霄只是随意的瞥了一眼，上面蜘蛛网遍布，灰尘浓厚，道：“想必他就是黑心老人，我们走吧。”
<br/>　　随后，慕清霄就牵着陆雪琪向左侧的隧道内走去，这条隧道与外面来时的路线一般无二，但却幽深静谧的多，往深远处看去，几乎便是一片黑暗。
<br/>　　慕清霄也暗暗咂舌，其中道路似乎也比较长，真是想不通，当年那些魔教炼血堂的人是怎么开辟出这么浩大工程的，若是他用魂剑来开辟，没有几天几夜也不行啊。
<br/>　　走了好一会，前面逐渐明亮起来，只见道路的尽头散发出柔和的光线，在黑暗中分外清晰，如温柔的触手，诱惑着世间凡人。
<br/>　　来到尽头，这里也是一个中等大小的石室，一侧摆着许多架子，一侧却堆积着一堆垃圾，大多是些铁器，诸如刀剑，枪弓等残损不堪。
<br/>　　慕清霄只是瞥了一眼就失去了兴趣，这些东西对他来说，本就与破铜烂铁无异，更何况本身就残损不堪。
<br/>　　在其中走了一圈，到处都是锈迹斑斑，唯有最后一个架子空空如也，只是比较干净，而且上面还有一个小铁盒，没有标签，谁也不知道其中放着什么。
<br/>　　慕清霄将铁盒拿在手中，随意看了一眼，用灵气包裹，直接打开铁盒，打开铁盒的瞬间，只听见“喀”的一声轻响，一股黑气瞬间冒了出来。
<br/>　　陆雪琪一惊，慕清霄拍拍她的香肩，示意没有大碍，这团黑气估计就是古尸毒，不禁让他对盒子里的东西有些期待起来。
<br/>　　瞥了眼阴阳灵气罩中的古尸毒，慕清霄一击将之击散，随后目光落在铁盒中。
