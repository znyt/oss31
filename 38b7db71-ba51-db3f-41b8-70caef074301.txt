　　“糜稽让你找人？”桀诺揍敌客背着手，走在阴暗的走廊内。
<br/>　　梧桐右手托着一个硕大的银质托盘，半圆罩子卡在托盘上，透过缝隙散溢出丝丝缕缕的烤肉香气。
<br/>　　“是的。三天前，糜稽少爷回来后，让我发动家族情报系统，帮他找两个人。一个叫比杨德，身份不明；另一个是帕里斯通，猎人十二支中的子鼠。”梧桐恭敬道。
<br/>　　“找到了吗？”桀诺淡淡问。
<br/>　　“根据情报显示，比杨德似乎和尼特罗会长有些关系，现在下落不明。而帕里斯通，据说有人在卡金帝国皇室成员身边见过他。”
<br/>　　“席巴怎么说？”
<br/>　　“族长大人允许调查，但吩咐不许将结果告诉糜稽少爷。”
<br/>　　“嗯，就这么办吧。”桀诺沉吟片刻，做了同样的决断。他越来越看不懂自己这个孙子了。每次观察，就好像在看一团迷雾。
<br/>　　但有一点他看得很清楚，这个实力大进的孙子，是真的毫无畏惧。甚至可以说肆无忌惮。
<br/>　　比杨德找不到还好说，如果让他知道帕里斯通在卡金帝国。桀诺非常肯定，自己这个孙子，甚至敢把卡级帝国皇室给掀个底朝天！
<br/>　　如果有报酬，他不介意让苏重出手。但这种任意妄为，明显的亏本买卖，万万是做不得的。
<br/>　　“是。”梧桐微微躬身，站在一旁恭敬送走桀诺。然后转身从岔道出走向餐厅。
<br/>　　“糜稽少爷，新鲜烤制的魔狼后腿。按照您的要求，选取实力强大魔狼，肉质坚韧非常。”梧桐面不改色的介绍，心里却直翻白眼。
<br/>　　这根本就不是坚韧了，而是坚硬！简单的切割都需要运用念力加持。烤制更是专门用念进行渗透加热。寻常人一口下去，甚至能把牙给崩下来！
<br/>　　“有比杨德的下落了吗？”苏重伸手拿起足有半米长的魔狼大腿。
<br/>　　梧桐刚想开口，就看到苏重猛然咧开大嘴，嘴唇直接裂开到耳根，上下颚瞬间胀大，一下就把整个魔狼腿吞进嘴里。
<br/>　　尖利的牙齿猛然合并。咔嚓！白色骨头碎屑崩飞，堪比钢铁的魔狼腿骨，被苏重好似脆骨般咬碎。
<br/>　　梧桐浑身一紧，额头瞬间深处细密汗珠。有那么一瞬间，他怀疑眼前之人，根本就是一个人形魔兽！还是凶悍异常的那种。
<br/>　　作为揍敌客家大管家，梧桐见多识广。可饶是如此，苏重狂野的进食方式，仍然把他给吓到了。
<br/>　　站在苏重身边，就感觉好似绵羊面对魔狼一般。那是一种来自食物链顶端的碾压。
<br/>　　“咳……还没有什么消息。”梧桐尽量让自己平静。他巧妙的避过帕里斯通不谈。而比杨德，他确实没得到什么消息。
<br/>　　“嗯！”苏重眯眼看向梧桐，他感觉梧桐的情绪有些不对。可又不像是谎言。
<br/>　　这种压迫感，怎么比席巴族长的还要大！梧桐心里大吼。甚至有种逃离的冲动。眼角狂跳，那种面对食物链顶端的战栗感，越发强烈了！
<br/>　　苏重眉头微皱，感觉自己情绪有些不对。努力把自主散发的精神冲击，收摄起来。
<br/>　　没了气势压制，梧桐明显松一口气。
<br/>　　“你走吧。”苏重脸色不太好看。
<br/>　　他发觉，自从魔龙七段进入第六段巅峰。魔龙法相再也无法增长。而随着时间推移，他的身体开始发生变化。
<br/>　　比如那满嘴的尖牙，比如能够裂到耳根的大嘴。他正在变得越来越不像人。
<br/>　　而且，自从全力释放了一次精神冲击后。这种极具压迫力的精神威严，就一直环绕在四周。
<br/>　　走在旷野，甚至不用他刻意发作。所有动物都会毫不犹豫的逃跑。而撞到他面前的动物，则会趴在地上，一副任君食用毫不反抗的模样。
<br/>　　这是一种高层次生命对低层生命的威慑，是魔龙的威严！
<br/>　　这些变化，让苏重更加强大。但却不受苏重控制的发生。不受控制的力量，向来不被苏重所喜。
<br/>　　而且这种变化，最终也不知道是好是坏。就像他的脾气，越发的暴躁。刚才梧桐的丝丝异样，放在往常根本就不算什么。但他竟产生一股杀掉对方，吃掉对方的冲动！
<br/>　　“算了不想了。”苏重压下心中暴虐，旋即开始思考比杨德的问题。
<br/>　　“几次三番的算计，找到这家伙，一定要杀了他！不不不，这不够。还要吞了他一身念力，让他在痛苦中衰老而死！”苏重咧嘴一笑，黑色瞳孔弥漫起死死红色纹路。黑红参杂，极似岩浆！
<br/>　　……
<br/>　　“你为什么派遣念能力者，前往东拓国？难道不知道，那里已经被嵌合蚁秘密控制了吗？！”比杨德脸色难看，死死盯着帕里斯通。
<br/>　　“不要在意。”帕里斯通整理这西服领带，一脸的随意道。
<br/>　　比杨德脸色一阴，瞬间来到帕里斯通身边。抓住对方衣领，一下就把他高高举起。轰然一声，狠狠按在墙壁之上，比杨德高大身躯凑近帕里斯通。
<br/>　　“嵌合蚁来自黑暗大陆，你永远不会明白黑暗大陆的可怕。派遣更多的念能力者，只会增强嵌合蚁的实力。一旦超出掌控，立刻会造成恐怖级别的灾难！你难道不清楚！”比杨德在帕里斯通耳边吼道：“老夫警告你，不要玩火！”
<br/>　　“哈！暗谋弑杀生父的家伙，竟然会在意他人，会在意什么灾难。”帕里斯通毫不客气的嘲讽，丝毫不在乎自己被比杨德攥在手里。
<br/>　　比杨德眼睛一眯，把帕里斯通狠狠甩在地上：“在意他人？灾难？我从不在意死人，但那会延迟探索黑暗大陆计划。已经走到如今这个地步，我绝不容许出现任何错误！谁挡路，谁就得死！包括你！”
<br/>　　帕里斯通脸颊一抽，突然间意识到，眼前这个长相硬朗，貌似磊落的家伙，是个十足十的偏执狂！
<br/>　　这家伙可是连生父比尼特罗都敢算计谋害，惹急了他，真有可能会动手！
<br/>　　“我明白了。”帕里斯通收起脸上嬉笑肃然道：“你放心，不会出问题的。糜稽揍敌客的实力太强了，他加上尼特罗，嵌合蚁王已经无法起到原有的作用。我们必须设法加强蚁王的战斗力。”
<br/>　　“这不但能顺利解决两人，还能进一步保证兵蚁的战斗力。你知道的，蚁王越强，培育的兵蚁就越厉害。扫清障碍，还能加强探索队的战斗力，这是一举两得的事情。”帕里斯通冷静的分析。
<br/>　　比杨德脸色缓和：“希望能你说的是真的。”
<br/>　　“绝不会超出掌控。”帕里斯通整理好褶皱的西装肃然道。
