　　第475章
<br/>　　程大雷手指落下的位置，出现三个墨字：饮马坡。
<br/>　　并州的地形大部分由草原构成，一眼望不到边界，几乎是无险可守。饮马坡的位置在黄沙城城北一百里处，由一片散落的湖泊组成。从前牧民常常在此地饮马歇脚，故而有了饮马坡这个名字。
<br/>　　“饮马坡地形开阔，敌人的兵力十倍于我们，若是正面冲突，怕是我们不占什么优势。”刘悲说出了心中的疑问。
<br/>　　“地形上，大家都占不到什么便宜。”程大雷道：“但在开阔的地形上，我们的火器可以发挥出最大威力，这也是将决战地点选在饮马坡的原因。”
<br/>　　刘悲沉默思考着程大雷的回答，暂时没有说话。徐神机问道：“他们会从饮马坡过么？要是从其他地方绕路过来攻城，我们在饮马坡岂不是白准备了。”
<br/>　　“现在的情况，帝国军希望我们集结在一起，将我们一网打尽。而我们何尝不是这样想的，大家都在寻找决战的机会。”程大雷坚定道：“向他们下战书，邀天下诸侯，在饮马坡一战，看他们是否有这个胆量。”
<br/>　　之后，大厅的诸人便商议着这次具体行动计划，看怎样才能将蛤蟆军的战斗力发挥到极致。
<br/>　　同时，程大雷派出喜字队的人，给帝国军递上了一封战书。
<br/>　　程大雷亲自执笔，在战书中慷慨陈词，说你们都是乱臣贼子，鱼肉百姓十恶不赦之类的。而我程大雷，是天上降下的圣主，以正义之名，用光明做武器，将你们这些黑暗的渣渣清除干净。
<br/>　　这封战书送进了帝国军的联营。
<br/>　　因为程大雷一直在草原上搞事情，帝国军各路诸侯已悄悄聚集在一起。如今各路诸侯的大营扎在一起，在草原上浩浩荡荡摆开一座联营。如此做，一是为了将程大雷一网打尽，二，也是避免那支部队落单后，再被程大雷趁机消灭掉。
<br/>　　收到程大雷的战书后，帝国军立刻召开了一次军事会议，主要目的，就是商议如何应对程大雷的挑战。
<br/>　　出席会议的，有李乐天、李行哉、李星等帝国皇族，以及崔相、尉迟离等朝廷重臣，还有青州、豫州、冀州、凉州各地的诸侯。
<br/>　　程大雷亲笔所写的战书被大家传着看了一遍，会议还未开始，大家便对程大雷的文笔和字迹做出深深的嘲笑。
<br/>　　直言程大雷果然是一个不通教化的草莽，文笔粗浅鄙俗，字迹潦草丑陋。甚至有人说：蛤蟆军难道就没有一个会写字的么，引得众人哄堂大笑。
<br/>　　李乐天轻咳一声，止住众人的笑声，开口问：“敌人的战书已送到这里，至于要不要应战，大家如何看？”
<br/>　　一片静默，在李乐天的声音落地后，诸人一时间都没有说话。
<br/>　　无论诸人愿不愿意承认，李乐天都从这场战斗中收获了不错的名声。手持天子剑，明帝的长子，带着孤军力守长安……无论如何，李乐天都是此间场合说了算的那个。
<br/>　　当然，有些人心底还是很不把他当会事的。
<br/>　　见没有接李乐天的话茬，尉迟离清了清嗓子道：“程贼不除，终究是帝国的心腹大患，此间他要决战，我们便与他决战，难道我堂堂帝国军人，还怕了一帮山贼不成。”
<br/>　　“咳咳。”
<br/>　　尉迟离的声音刚落地，一道熟悉的咳嗽声便响起。尉迟离在心中冷哼一声，骂了一句老狐狸。
<br/>　　这咳声自然而然是来源于崔相，他扶须笑道：“这程大雷果然是一介草莽，将两军对垒，当做江湖草莽的逞凶斗狠，决战？呵，传出去怕是会被天下有识之士笑掉大牙。”
<br/>　　“那么以崔相之意，这件事该如何办，难道我们就不应战，看着程大雷在我们面前耀武扬威不成。”尉迟离。
<br/>　　“喔……呵呵，这个老夫就不敢妄加论断，领兵打仗的事本不是我专长，这件事还要请教尉迟将军。”
<br/>　　请教你妈……
<br/>　　尉迟离在心中发出一声咆哮，既然你不懂，你废话这么多做什么。出了事和你没关系，有功劳你还要分一份。尉迟离和崔相同朝为官，是彼此最大的政敌，对于崔相的行事风格，尉迟离很是了解。
<br/>　　现在他心中窝火，也不愿意说话，将头扭到了一边。
<br/>　　此时，扬州牧李星开口道：“那程大雷可不是易于之辈，他这次会不会藏着什么诡计。”
<br/>　　这几日，程大雷的队伍在草原上兴风作浪，李星的人马很是吃了瘪，扬州兵死伤惨重，现在李星都快有心理阴影了，想起这件事他便有些后怕。
<br/>　　“我已经调查过了，黄沙城的人的确在向饮马坡调动，他们排出去的队伍也收了回来，这两日，你还听到有战败的消息么？”尉迟离神色不善道。
<br/>　　“如此说来，他真有在饮马坡决战的意思？”李乐天道：“我方人马十倍于敌人，如果在饮马坡拉开阵势打上一仗，未必不能将敌人一举消灭。”
<br/>　　“咳咳。”
<br/>　　那种讨人厌的咳嗽声又响了起来，尉迟离憋火的把眼睛闭上了，他只想今天这场会议可以早点结束。
<br/>　　“所谓兵不厌诈，就算程大雷调动兵力，也未必没有其他阴谋，还是要小心为上。”
<br/>　　众人你一言我一语，有人要决一死战，有人则顾虑颇多，吵来吵去，终究也吵不出个确切的主意。
<br/>　　“好啦。”
<br/>　　李乐天突然开口，目光扫视过众人，把不平静的声音压下去。
<br/>　　“贼人要战，我们便与他们决一胜负，堂堂帝国军，断然没有怕几个贼寇的道理。”李乐天慷慨激昂道：“我们便调兵谴将，击中所有兵力，在饮马坡打败他们。”
<br/>　　李乐天深吸一口气：“接下来谈的，便是该如何布置兵力了？”
<br/>　　大帐内，议论声再次响起，七嘴八舌说个不停，关于一个小小的细节，有时候便要吵好久。
<br/>　　在这个过程中，李行哉从始至终没有开口说一句话，他微眯着双眼，用手撑着下巴，一副老神自在的模样，却谁也不知他心中想着什么。
<br/>(本章完)
