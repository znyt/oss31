　　两万人挤在一起可以看到密密麻麻的一片人头，可二十万人挤在一起的时候，你会发现自己看到的还是密密麻麻一片人头，如果没有从高空俯视你根本找不到两万与二十万的区别。
<br/>　　拜占庭所有军队加在一起人数已经超过二十万，横亘在一片戈壁上就算是一道看不到头儿的城墙。
<br/>　　而在拜占庭人的对面，四万余唐军亦摆开阵势，一道道交通壕互相联通到一起，形成一个类似于蜘蛛网一样的网络。
<br/>　　这是九十二师用了一晚加半天整理出来的东西，面对二十余万拜占庭人，龙义打算与九十三师联手，先给他们来个迎头痛击。
<br/>　　拜占庭统兵大将在第一次看到唐军的布置之后也有些傻眼，他实在搞不清楚那些只露出一个小脑袋唐军到底想要搞什么，难道在地上挖出一大堆的沟壑就能阻止骑兵？如果那样的话还不如直接挖个大一些的，至少战马跳不过去。
<br/>　　不过拜占庭的统兵大将虽然想不通，但却并没有被吓住，看着那一条条沟壑，老家伙派出了全部五万人的奴隶军团，打算先用这些奴隶来试氷，就算是试不出大唐这个阵式的深浅，在他看来也能用尸体把那些壕沟填平。
<br/>　　奴隶军团都是这些年拜占庭与敌人作战抓来的俘虏，战场上他们就是些炮灰，主要用来消耗敌人的远程攻击武器。
<br/>　　按照拜占庭人以往的经验，这五万人冲到敌人的阵地大概会死掉五分之一，而余下的五分之四则会与敌人纠缠到一起，这个时候敌人的弓箭便会因为混乱而失去作用。
<br/>　　但是这一次，他的经验似乎受到了挑战，五万奴隶军团冲上去，还没冲到一半的时候已经有五分之一的人倒了下去，余五分之四的人也开始畏缩不前。
<br/>　　唐军反击实在是太强大了，那种能发出巨大声音的武器简直就是所有人的噩梦，每一次响起都会有无数人惨叫着倒下去，身体上会多出一个流血不止的血洞。
<br/>　　没有人喜欢自己死的不明不白，奴隶也是一样，他们之所以会参加战斗，那是因为如果战斗取得了胜利他们就会得到自由。
<br/>　　可如果全部战死在战场，明知有死无生的话，就算是奴隶也不会一门心思的冲上去。
<br/>　　拜占庭的将军很快发现了情况的不对头，经验丰富的他果断下令，再次派出了五万骑兵，追在奴隶军团的身后冲了上去。
<br/>　　绝不能让奴隶退回来，否则本阵一定会受到冲击，士气一定会大损，而且这一次好不容易冲上去的，如果退回来下一次必然没有人敢去冲阵。
<br/>　　龙义站在一处高台上面，远远的用望远镜看着战场上的形势，拜占庭人的一举一动都在望远镜下无所遁形，所以很快他就发现在敌人的目的，果断下令道：“命令，所有部队集中精神准备迎敌。另外，给九十四师发信号，他们可以回来了。”
<br/>　　“是！”传令兵兴奋的答应着跑了开去，时间不大已经有信号弹升上半空，接着，远处一颗又一颗信号弹升了起来。
<br/>　　……
<br/>　　前线的枪声越来越密集，随着拜占庭骑兵的突击，前面第一、第二道防线的新式步枪同时响了起来，屠杀也在这一刻正式开始。
<br/>　　五万奴隶军团开始的时候冲的很卖力气，此时已经距离九十二师的阵地不足两百步，但他们的人数也只剩下了开始时的一半。
<br/>　　整个战场上到处都是哀嚎的声音，地下倒着无数或伤或死的奴隶，为了自由这些奴隶几乎拼上了一切，虽然所谓的一切只是他们身上淡薄的衣服以及根本没有人会看在眼中的生命。
<br/>　　而在大唐军人的眼中，他们的面前只有潮水般涌上来的敌人，尽管那些敌人衣衫褴褛，有老有少，但敌人就是敌人，非我族类的敌人。
<br/>　　大唐军人在入伍的那一天起，就有人一直在告诉他们，想要封妻荫子那就在战场上奋勇杀敌，用敌人的鲜血来换回自己的未来。
<br/>　　异国他乡的土地上，面对敌人任何怜悯都是对自己的残忍，是对自己生命的不负责，所以不管他们的内心是怎么想的，手中武器却一直没有停下。
<br/>　　这一次他们再也不用对着天空放枪，而且那些傻乎乎的敌人就像是一个个靶子，他们不知道什么叫躲闪，也不知道什么叫害怕，一个个就像是疯子般。
<br/>　　杀，杀光他们，杀光他们就可以回家，家里的父母双亲还等着他们，家中的嗷嗷待哺的孩子也在等着他们，刚刚成亲不久的娇妻也正倚门而望。
<br/>　　杀光敌人，回家！这就是龙义最后的战前动员，也是大唐军人心中唯一的念头。
<br/>　　拉枪栓，装弹，瞄准，击发，再接枪栓……一连串的动作被九十二师、九十三师的每一个战士机械的完成，然后看着敌人在远处一个又一个倒下。
<br/>　　步枪杀伤力再加上敌人的愚蠢让唐军上致龙义下至伙夫，没人能提起一点兴趣。
<br/>　　刚开始看着敌人冲上来心中的那份忐忑随着倒下的敌人越来越多，已经变的麻木，就好像一个成年人在与一个三岁的孩子战斗，完全就是一面倒的碾压。
<br/>　　拜占庭人似乎也觉察到了情况有些不对头，唐军的实力与情报中的完全不同，天知道那天明尼城主是如何凭借五千人烧掉他们那么多粮草的。
<br/>　　最后当奴隶军团马上就要死光的时候，拜占庭人的统兵将领终于意识到发生了什么，大声喊道：“命令骑兵撤，马上撤下来，我们中计了，让他们快撤！”
<br/>　　“二殿下，来不及了，现在骑兵根本撤不下来，硬撤只会损失更多人，还不如让他们一直冲上去，只要冲进唐军的主阵，胜利还是我们的。”伊力克的眼角狂跳，但理智上他不得不劝说这次领军出来的二王子。
<br/>　　要知道骑兵作战靠的就是一往无前的气势，如果冲到一半停下，再调头回来，在唐军那种武器的攻击下，这一转身的功夫估计就能损失一半人手。
