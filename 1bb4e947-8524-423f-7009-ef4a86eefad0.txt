
<br/>　　<br/>　　    王惊蛰很认真的看着妖和尚说道：“你这样飞来飞去的很像一只‘嗡嗡’的苍蝇，很烦人，你信不信我会一巴掌拍的你魂飞魄散？”<br/>　　<br/>　　    那鬼影渐渐的安生了下来，缩在角落里，王惊蛰的身上给他带来了一种很恐怖和惧怕的信息，想逃又逃不出去。<br/>　　<br/>　　    “啪”王惊蛰伸手弹出一张符纸，飘落过去：“给你个选择，进去呆着，过后我在审审你，你若是不答应跟我走，我大概有好几种手段会让你觉得，当个孤魂野鬼都是奢侈的，比如一点点的烧了你的魂？”<br/>　　<br/>　　    那鬼影突然开口了，很凄厉的喊道：“不，你要问我的我不能说，他太可怕了，会，会让我生不如死的”<br/>　　<br/>　　    王惊蛰叹了口气，说道：“那总比现在被我折磨死要强吧，你在好好想想，到底要不要跟我走……”<br/>　　<br/>　　    “你，你答应保我，我，我就跟你走”<br/>　　<br/>　　    王惊蛰淡淡的说道：“我只能说，会让你魂飞破灭的痛快一点，少遭些罪”<br/>　　<br/>　　    妖和尚最终还是没能抵得过这个诱惑，因为对他来说，一下子就魂飞魄散，确实比遭罪要舒服得多了。<br/>　　<br/>　　    王惊蛰弯腰，伸手捡起了地上的那张符纸，揣到身上后，转身就要走，这时在床上还有点瑟瑟发抖的郭茜茜，惊恐的问道：“你，不解释一下么？”<br/>　　<br/>　　    王惊蛰沉默了下，说道：“就是你看到的这么一回事”<br/>　　<br/>　　    郭茜茜愣了愣，茫然不知所措，确实，她受过的十几年教育，仿佛一下子都喂了狗，这肯定是让人难以接受的，刚才化成一缕黑烟钻进一张黄纸里的是什么，是鬼么，那这个奇奇怪怪的王惊蛰呢，又是什么人，道士下山？<br/>　　<br/>　　    “这，这世上真的有鬼么，不，不是的，这是封建迷信，都是骗人的”郭茜茜神情有些呆滞的说道。<br/>　　<br/>　　    王惊蛰幽幽的叹了口气，颇为头疼的说道：“快晚上了，也饿了，下去吃点东西？”<br/>　　<br/>　　    王惊蛰知道，今天要是不给这位警花解释清楚的话，她恐怕会受不住打击而崩溃了的。<br/>　　<br/>　　    挺好看个姑娘，变成傻子就不太好了吧？<br/>　　<br/>　　    郭茜茜家公寓附近，有一家川中火锅店，王惊蛰和她来到店里，挑一了一处靠窗的位置坐了下来。<br/>　　<br/>　　    “先生，你是吃红汤的锅底，还是清汤的？”服务员过来问道。<br/>　　<br/>　　    王惊蛰看了眼还有点呆滞的郭茜茜，抬头说道：“清汤的吧，最近有些上火，红汤的吃了会不舒服，然后给我上一些青菜，黄喉，毛肚和两盘肉，在来一打啤酒吧……”<br/>　　<br/>　　    郭茜茜呆呆的看着桌面，似乎是在努力的消化着脑袋里的东西，王惊蛰这时也没急着去管她，有些东西得是需要时间来消磨的，他起身去打了两份小料，按照自己的口味调了出来，把一份放在郭茜茜面前，等酒上来以后，他倒了两杯，郭茜茜就抓起剩下的半瓶酒“咕嘟，咕嘟”的一仰而尽了。<br/>　　<br/>　　    王惊蛰也没有管她，这个时候警花最需要的就是放松，而不是一直憋着，那能吃能喝，就固然是最好的了。<br/>　　<br/>　　    清汤火锅上来了，王惊蛰夹了几片肉放进锅里，又下了一点毛肚和黄喉，等锅开了以后，肉片都飘了上来，王惊蛰拿起筷子就一口一口的吃着，郭茜茜依旧在喝着酒，一杯接着一杯。<br/>　　<br/>　　    片刻后，王惊蛰吃着吃着就觉得有点不太对劲了，就招呼服务员过来，诧异的问道：“我明明点的是清汤的锅底，可怎么吃起来会感觉麻麻的？”<br/>　　<br/>　　    “那不能啊，您看这汤确实是清汤的，一点麻椒都没有呢”<br/>　　<br/>　　    “咦，奇怪了”王惊蛰夹起一片毛肚送进嘴里，吃了起来，仍然感觉嘴唇有点发麻，甚至手指也是麻麻的，他愕然说道：“不信你尝尝，确实是麻的啊”<br/>　　<br/>　　    服务员眼睛忽然落在插着火锅的电源线上，他抽搐了几下嘴唇说道：“不好意思先生，是我们的火锅漏电了”<br/>　　<br/>　　    王惊蛰：“……”<br/>　　<br/>　　    他也看见了，电源上有一块坏了，他嘴麻和手麻的感觉，就是从这来的。<br/>　　<br/>　　    “噗嗤”这时候，郭茜茜对他们的话似乎有了反应，刚喝下去的一口酒就喷了出来。<br/>　　<br/>　　    王惊蛰抽出纸巾递过去，说道：“擦擦吧”<br/>　　<br/>　　    “有鬼，是么？”<br/>　　<br/>　　    “嗯，你看到了”<br/>　　<br/>　　    “怎么可能会有鬼呢？”<br/>　　<br/>　　    “怎么可能会有狗，鸡鸭，牛羊呢，怎么可能会有熊猫呢，它们可能会想，怎么会有人这种生物呢，这个世间存在就是道理，不要问为什么，最会打禅机的和尚恐怕也不会给出你任何的答案来，因为存在就是道理”<br/>　　<br/>　　    郭茜茜愣了愣，开始时很难理解王惊蛰的话，但她本来也不笨，想着想着就明白了一点，世间万物万万种，有这有那不奇怪，那有鬼为什么要奇怪，仅仅是因为绝大多数的人都没见过的原因？<br/>　　<br/>　　    “那，他，他为什么会找上我”郭茜茜迷惑的问道。<br/>　　<br/>　　    “我引诱的”王惊蛰皱了下眉，接着如实说道：“在破庙里，我和你还有你们潘队长，三个人挖出了那几具尸骨，就注定会被他们给缠上，但是我他们不敢找，潘利明火气太旺，你恰巧最近来了月事，身子发虚，阴气又重，所以不找你找谁啊？昨天我提醒过你，是想让你防着点，但你肯定也不会太信我的话，我只是想让你心里有点数，能有个接受的度”<br/>　　<br/>　　    郭茜茜顿时明白了，有点恼怒的说道：“你用我当诱饵？”<br/>　　<br/>　　    王惊蛰两手一摊，说道：“我只是想破案罢了”<br/>　　<br/>　　    “这和案子有……”郭茜茜到这，话说了一半就忽然顿悟了，这和案子有什么关系？当然有关系了，她明白了那些人是怎么死的了。<br/>　　<br/>　　    郭茜茜沉默了片刻，叹了口气：“我回局里得需要怎么解释呢？谁又回信呢”<br/>　　<br/>　　    王惊蛰摇头说道：“解释干嘛？不需要，你们手里没破的案子还不够多么？范成梁需要的，是我给他的解释，你们就省了吧”
