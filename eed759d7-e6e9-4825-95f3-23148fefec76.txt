　　第2章
<br/>　　日头偏西，草木山林越发血染。
<br/>　　由南向北，一队人马沿着山道缓缓而来。约有十几人，中间护着一辆精致的马车，前后护着佩刀兵丁，在车队正前方，两匹高头大马并肩而行。
<br/>　　“林少侠，咱们这次护送小城主的未婚妻，这里山贼出没，咱们还要多加小心。”骑马的黄三元道。
<br/>　　“山贼……”林少羽不屑笑道：“若是不来还好，若是他们敢来，我让他们见识见识我手中铁枪厉害，一路风平浪静，我这铁枪早已寂寞难耐。”
<br/>　　“一路也多靠林少侠护持，不过这次送苏家小姐去黑石城与公子完婚，干系重大，还是小心些好。”黄三元。
<br/>　　“黄管家太小心了，咱们替城主做事，谁敢劫我们，不想活了嘛，哈哈。”步行的兵丁大笑。
<br/>　　草丛中，窝着徐神机与程大雷，徐神机道：“小寨主——不，大当家，天要黑了，估计不会有人经过，撤吧。”
<br/>　　“等。”程大雷咬着草根，系统要求在日落前完成第一次打劫，自己必须在日落前完成，抹杀的惩罚可是难以承受。
<br/>　　哒哒哒……
<br/>　　远处传来马蹄声，程大雷眼睛一亮：“来了。”
<br/>　　“来是来了，可这人数……”徐神机面露苦涩，他一个个数着，对方算上马夫足有十三人，而且都是穿皮甲的兵丁：“大当家，撤吧，别冲动，千万别冲动。”
<br/>　　“你的心情我很理解。”程大雷翻身上马：“可我愿意和你同生共死。”
<br/>　　“可是俺不愿意呐。”
<br/>　　徐神机腹诽一声，看着越来越近的车队，心脏砰砰打鼓。他忽然一咬牙，抽出随身的短刀，扎在瘦马屁股上。
<br/>　　瘦马吃痛狂奔，宛若一阵狂风，冲着车队直冲过去。
<br/>　　“呔，劫你们的是蛤蟆岭蛤蟆寨蛤蟆大王，不想死的赶紧逃命啊。”
<br/>　　徐神机虽老，嗓子却很棒，一声喊声彻八方，群鸟惊飞。不仅嗓子好，他跑得也很快，刚喊完，转身就往后逃。
<br/>　　“小寨主，对不住了，我还有女儿要养，以后给你烧纸。”
<br/>　　程大雷一边着急控制失控的瘦马，一边在心底狠狠给徐神机比出中指。出卖我也就算了，什么叫『蛤蟆大王』，这匪号拿不出手啊。
<br/>　　等他终于停下瘦马，已经来到车队正前方，一队人拔出佩刀，刀尖指着自己，每个人都是一脸错愕。
<br/>　　“就，就你一个人？”黄三元难以置信。
<br/>　　看着泛着寒光的刀刃，程大雷觉得，被系统抹杀，可能是一个不错的选择。
<br/>　　“误会，误会，其实我不是蛤蟆大王。”程大雷深吸一口气，突然大喝：“俺乃是玉面书生徐神机，不想死的，留下买路财！”
<br/>　　黄三元终于消化这难以置信的信息，山贼只有一个人，就要打劫自己整个车队。
<br/>　　“你可知道我们是谁，年轻人，活着不易，不要自寻死路啊。”黄三元语重心长。
<br/>　　周围的兵丁放声大笑，手中铁刀拍着刀鞘哐哐作响。
<br/>　　这时候从马车内探出个脑袋，一张玉润的鹅蛋脸，皮肤吹弹可破，大大的眼镜。看见程大雷也不笑，也不怕，就用一双大眼睛看着他。
<br/>　　程大雷心中一荡，世间竟还有如此绝色女子。
<br/>　　嘟，隐藏任务触发『抢夺压寨夫人』
<br/>　　嘟，该任务难度超过你目前实力，你可以拒绝接受。
<br/>　　程大雷心中一动，他知道隐藏任务一般很难触发，往往难度也很高，可一旦完成奖励也很丰厚。
<br/>　　妈的，成功洞房花烛，失败脑袋搬家，没这点胆子，做什么山贼，这任务老子接了。
<br/>　　“都别笑，都别笑。”林少羽艰难止住笑容：“好不容易来一个蟊贼，你们再给我吓跑了，说好了，这个蟊贼由我收拾，谁也不许插手。”
<br/>　　“一路上都听林少侠本事了得，这次终于有机会亲眼见到。”
<br/>　　“林少侠小心些，别一下弄死了，俺们就没得瞧了。”
<br/>　　林少羽拍马向前，手中持一柄铁枪：“小贼，只要你能在我枪下走够三个回合，我便放你一条生路。”
<br/>　　程大雷集中注意力在林少羽身上，脑海中出现对方的信息。
<br/>　　姓名：林少羽（初出茅庐的普通游侠）
<br/>　　技能：破浪枪
<br/>　　隐藏属性：立志成为大侠的男人
<br/>　　以程大雷多年游戏宅的经验，普通，优秀，顶级，绝世，传说……林少羽是最底级的『普通』，你说你一个白板英雄你狂什么，我还以为你是什么超级牛人呢。
<br/>　　“我斧下不死无名之鬼，你是哪个，报上名来！”程大雷观察着周围情况，思考着计划，一边悄悄催马，缩短和林少羽的距离。
<br/>　　“沧海，林少羽。”林少羽正经起来，对敌人的尊重是成为一个大侠的基本要求，然后他觉得来而不往非礼也，问：“你是哪个，蛤蟆大王还是玉面书生？”
<br/>　　“我么……”程大雷看着林少羽一本正经的样子，用手挠挠头：“要不，咏春，叶问？”
<br/>　　“咦！”
<br/>　　林少羽睁大眼睛，帝国十三州一百单八城，从未听过咏春这个地方，而这姓名……也有些奇怪呐。
<br/>　　此刻两匹马的马头凑在一起，林少羽已经在程大雷斧头的攻击范围，他大手一抡，斧头照着林少羽的脑袋劈了过来。
<br/>　　“我劈脑袋！”
<br/>　　出手便按照某种固定的轨迹，似乎这一招程大雷已经练习过无数次。林少羽正在发愣，自己是否该说句『久仰久仰』之类的，免得对方笑话自己不懂江湖规矩，虽然自己从未听过这个名字。冷不防斧头已经到了，林少羽慌张横枪格挡，斧重枪轻，再加上程大雷以有心算无心，一斧头把林少羽砸落马下。
<br/>　　拍马向前，纵身跃在马车上，一脚将马夫踹下去，斧头把在马屁股上一扎，烈马唏律律嘶鸣一声，如一辆坦克般向前狂奔，手中的斧头左右横抡，硬生生的冲出包围圈。
<br/>　　“追呀，追啊！跑了少奶奶，你们都得死。”黄三元在马上大喊。
<br/>　　周围兵丁目瞪口呆，直到现在还没回过神来，怎么莫名其妙，就让这小蟊贼把少奶奶劫走了。
<br/>　　林少阳从地上爬起来，翻身上马，手中攥紧枪杆，他发誓要洗刷这偷袭之耻辱。
<br/>　　嘟，第一次打劫任务完成，奖励抽奖次数1 。
<br/>　　嘟，抢夺压寨夫人第一阶段完成，奖励……
<br/>
