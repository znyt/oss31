        诺筱颖洗完碗从厨房里出来的时候，便看到两人正躺在沙发上热情地拥吻。
<br/>
<br/>        唐倩兮身上已经衣衫不整，一边的美胸露了出来，底裤也褪到了膝盖处。
<br/>
<br/>        诺司峻热吻着唐倩兮，并对唐倩兮上下其手。
<br/>
<br/>        诺筱颖看得一阵脸红，连忙躲进了自己房间，没过多久，房门外便传来了诺司峻和唐倩兮的喘息声。
<br/>
<br/>        但是又没过多久，她的房门不知被谁扣响了。
<br/>
<br/>        诺筱颖这才从书桌前起身，去开房门。
<br/>
<br/>        “哥？”诺筱颖愣了。
<br/>
<br/>        诺司峻身上满是吻痕，腰间只围着一条浴巾，皱着浓眉，撇着嘴，故作委屈地说道：“筱颍，帮哥哥去楼下买盒安全套回来呗！”
<br/>
<br/>        “我上次不是帮你买了一盒吗？”诺筱颖嘴角抽了抽。
<br/>
<br/>        “已经用完了！你再去帮我买一盒！噢，不！是两盒！你再帮我买两盒回来吧！拜托了！”诺司峻拱起双手作揖请求。
<br/>
<br/>        诺筱颖汗颜，但还是好心地答应了：“好吧！”
<br/>
<br/>        她走到门口，刚把鞋穿上，身后，诺司峻就开始催促了。
<br/>
<br/>        “快去快回啊！江湖救急！”诺司峻说完，转身就回了房间。
<br/>
<br/>        诺筱颖拿起钥匙出了门，坐电梯下来后，来到一楼大门口外的自动售卖机前，有了上次的经验，这次她买起东西来，从投币到取货，动作倒是非常麻利了。
<br/>
<br/>        “又再帮你哥哥买安全套。”
<br/>
<br/>        身后突然响起一个熟悉又好听的声音。
<br/>
<br/>        诺筱颖刚将两盒安全套拿在手上，下意识地藏在身后，并转过身去看向这个发声源的男主人。
<br/>
<br/>        只见傅昼景身上穿着白色的运动背心和运动休闲长裤以及白色的运动鞋，他额头上还挂着豆大的汗珠，汗水湿透了衣裳，紧贴着他的胸肌。
<br/>
<br/>        他这汗流浃背的样子，看起来非常有男人味。
<br/>
<br/>        “冷先生，晚上好！”诺筱颖回过神来，赧然地红了脸，低着头问候了一声。
<br/>
<br/>        傅昼景爽朗地笑了笑：“你叫我‘冷先生’，怪别扭的。你要是不介意，叫我昼景哥就行了。”
<br/>
<br/>        “我……”诺筱颖欲言又止，抬起手来挠了挠额前的刘海，有点儿娇羞地唤了声，“昼景哥，晚上好。”
<br/>
<br/>        傅昼景温文儒雅地伸出手来，像个大哥哥一样，揉了揉诺筱颖的头顶，微笑道：“真乖！”
<br/>
<br/>        而他这一举动，让诺筱颖抬眸的一瞬间，看出了神。
<br/>
<br/>        这个男人真的很温柔，而且还很体贴。
<br/>
<br/>        当诺筱颖怦然心动的时候，手机不逢其时地响了起来。
<br/>
<br/>        诺筱颖缓过神来后，立马掏出手机看了看屏幕上的来电显示。
<br/>
<br/>        傅昼景下意识地也跟着瞄了一眼她的手机屏幕，见上面显示着“哥哥”两个字，会心一笑，温和地说道：“你快上去吧！你哥哥，又等不及了。”
<br/>
<br/>        “好、好的！那……昼景哥，下次见。”诺筱颖直接挂了诺司峻的电话，向傅昼景挥了挥手，转身跑进了一楼大厅门口。
<br/>
<br/>        傅昼景看着诺筱颖的背影，微微一笑。
<br/>
<br/>        她是个挺可爱的女孩子。
<br/>
<br/>        而诺筱颖跑进电梯里后，心跳加速，久久未能平缓下来。
<br/>
<br/>        她刚刚是怎么了？
<br/>
<br/>        帅气的男人她又不是没见过，为什么就那么几分钟的时间里，她会对傅昼景看出神？
<br/>
<br/>        诺筱颖深吸了口气，抬起一只手来拍了拍微微发烫的脸颊。但是，脑海里回想起刚刚与傅昼景在一起的画面，她的嘴角情不自禁地露出了一抹甜甜的微笑。
<br/>
<br/>        “你终于回来了！怎么去了那么久啊！还挂了我电话！我还以为你出事了！”
<br/>
<br/>        诺司峻的话劈头盖耳地轰来时，诺筱颖才开门走进来，连鞋都未来得及脱，手里的两盒安全套就被哥哥给夺走了。
<br/>
<br/>        “我这不是快到家了嘛！才挂了你电话的！”诺筱颖看着诺司峻那猴急的样子，有些忍俊不禁。
<br/>
<br/>        “不跟你多说了！你嫂嫂等不及了！”诺司峻对着妹妹诺筱颖扬了扬浓浓的眉毛，手里拿着安全套，一边拆着外包装，一边只身往房间里跑。
<br/>
<br/>        诺筱颖忍俊不禁，脱了鞋，关门走了进来。
<br/>
<br/>        虽然，诺司峻是她的哥哥，但是她和哥哥长得一点都不像。
<br/>
<br/>        其实，她不仅不像她哥哥，也不像她的老爸老妈。
<br/>
<br/>        诺筱颖的五官，在他们西镇的苗族姑娘里，是最精致耐看的了。前几年，那么多苗族姑娘里，她还被众人投票选为了西镇的“镇花”，上了云城县区的旅游宣传画报。
<br/>
<br/>        翌日。
<br/>
<br/>        诺筱颖早早的就起了床，还给哥哥嫂嫂准备好了早餐，才去上的班。
<br/>
<br/>        她刚到办公室，比她更早到的副部长孙莉莉，正巧从独立办公室里走出来。
<br/>
<br/>        孙莉莉看到诺筱颖，正好她手里有个新单子，于是把诺筱颖叫到了自己的办公室里，将这个单子交给了诺筱颖。
<br/>
<br/>        诺筱颖拿到资料翻看了一下，原来是给傅氏集团总部的未来总裁设计办公室装修图。
<br/>
<br/>        这个单子虽然是内部单子，但是依然有提成可以拿。
<br/>
<br/>        给傅氏集团未来总裁设计办公室装修图，可不是普通的单子，上头交代下来，让他们设计部必须十分重视，并设计出最好的装修图来。
<br/>
<br/>        这段日子，部长出差了，孙莉莉昨天下午拿到这个单子，看到这上面的苛刻的要求时，和几个资历深厚的设计师讨论了一下方案，也没讨论出一个比较好的结果出来，顿时让她感到有些束手无策。
<br/>
<br/>        而这个诺筱颖虽然是新手，但是她上次设计的别墅装修图很不错，这次，不如也拿给她试试。
<br/>
<br/>        反正，她已经让设计师去设计了，再多她诺筱颖参与也无妨，到时候，等设计图出来，对比较着再综合一下，说不定会更好。
<br/>
<br/>        “看完了吗？”孙莉莉问道。
<br/>
<br/>        诺筱颖点了下头，微笑着回答道：“我只看了个大概，不过，我可以先试试。”
<br/>
<br/>        “那你开始动工吧！这可是给我们傅氏集团未来总裁的办公室，你不可有一丝马虎。知道了吗？”孙莉莉一脸严肃地说。
<br/>
<br/>        诺筱颖抱着资料，犹如小鸡啄米般猛点头。
