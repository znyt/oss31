　　“哈德林克先生，你没事儿吧？”良久之后，彭城见老番子一直在发呆，不由伸手在他的面前晃了晃。
<br/>　　这虽然有些不大礼貌，但的确很有效果，老番子几乎瞬间就回过神来，看着彭城苦笑说道：“不好意，刚刚是我有些失态了！”
<br/>　　彭城见老番子恢复正常了，这才继续说道：“没什么，这主要也是怪我没有把话说清楚，其实两位殿下还提出了其它两个解决办法。”
<br/>　　“是什么办法？”哈德林克是真的很想做成这笔生意，一只有其他解决办法，立刻来了精神。
<br/>　　“第一个方法是租用，也就是说我们可以把厂建到贵国指定的地方，但整座工厂都是我们的，贵国每年付出一定的租金来租用，这个租金的价格在一年三百万贯左右；第二就是贷款，我们可以借给贵国一万万贯，这笔钱贵国需要分十年还清，每年连本带利一千一百万贯，但是贵国需要质押一些东西。”
<br/>　　彭城脸上保持着微笑，实则心里紧张的要命，这还是他第一次谈这么大的一笔生意，价格单位竟然是以万贯为标准的。
<br/>　　要知道，就在半年前，他所有的钱加在一起也只有十几万贯，可是现在，他竟然可以在毫不犹豫的瞬间说出数百万，上千万贯这个数值。
<br/>　　这种变化是他以前从来都不敢想的，那种成就感同样也是他不敢想的。
<br/>　　现在他担心的就是那个老番子会不会被他的价格吓跑，若真是这样那可就要损失一大笔钱了。
<br/>　　而让彭城意外的是，哈德林克在听完了他的两个条件之后，竟然露出了欣喜的表情，热情的抓住他的手，紧紧握住：“竟然还可以这样？真的可以这样么？”
<br/>　　彭城被老番子吓的够强，好不容易抽回被握的有些发红的手，有些尴尬的说道：“当然，如果你真的有意要选择其中的一种方式，我们还可以继续详谈。”
<br/>　　……
<br/>　　详谈的意思就是一点一点的谈，慢慢的谈，马拉松式的谈，从建厂选址到投入运营的时间，从违约后的惩罚条件到完成约定后的售后服务。
<br/>　　发电厂必竟是一个新兴的事物，就算是在大唐，这东西也足够新颖，有很多人都搞不清楚其中的门道，这对于哈德林克来说简直就是一场惊天豪赌。
<br/>　　一万万贯的贷款啊，算上利息那就是一万万一千万，十年利息百分之一，说来价格可以算是很公道一，百分之一的比例乘以一万万这个基数，多出来的利息可就是天文数字了。
<br/>　　但是哈德林克认为，自己的发现绝对是一个大发现，这个所谓的‘电’将来一定会有大用处，凭借着自己的直觉，他发誓不管花多少钱都要把这个东西弄回到拜占庭。
<br/>　　而且抛开发展的趋势不说，就说大唐对电的应用吧，电灯其实是最简单的应用，那些电风扇之类的东西同样是他所需要的。甚至他还听说大唐已经可以用电来驱动机器来纺织，用电来驱动机器打造兵器，电的应用在大唐虽然刚刚开始，但却有着无穷的发展空间。
<br/>　　拜占庭在这个时候绝对不能落后，如果落后太多一定会在将来被大唐打的满地找牙，大唐的确很强大，但是拜占庭也要努力追赶不是。
<br/>　　不过好在大唐也不是什么傻子，有很多东西也是对外禁止出口的，比如机床，比如电焊机。
<br/>　　李承乾可以卖电厂，那因为如果没有大唐的支持就算是给那些异族百把十年他们也搞不清楚这其中的门道。
<br/>　　一万万一千贯买回来的东西有几个人敢随随便便就拆掉的？万一拆坏了怎么办？而且就算是他们能拆开，又有谁能保证他们一定能组装回去？
<br/>　　在技术方面，大唐现在已经遥遥领先于其他国家，完全有能力通过技术来封锁技术。
<br/>　　所以他现在敢卖发电厂，只要有人买他就敢卖，就像那个推土机和拖拉机一样，只要你给钱，我就敢把东西卖给你。
<br/>　　大唐技术上的领先，让大唐的官员可以理直气壮的告诉那些异族，东西买回去你们可以随便仿造，如果能造出来就算你赢，我可以原价买回来。
<br/>　　……
<br/>　　不得不说，这一次的商业洽谈无疑是非常成功的，以前那些商人想要购买什么东西需要到处去走去看，可是在商贸会上就不用担心这些事情了。
<br/>　　每一位大唐的商家都会有自己的商品摆在一个特定的展区，那些异族的商人或者大唐本土的商人完全可以亲自去挑选自己喜欢的东西，然后再商量价格。
<br/>　　而另外一件大件的商品，比如推土机、拖拉机之类，这些东西虽然没有实物，但是好在那些异族的商人来到大唐的这几天已经在城外见识过几次，心中已经有了概念，商量价格的时候他们的心里多少也有些底限。
<br/>　　再说那些大型设备一般都是在官方在发卖，异族就是想买也要通过官方渠道，否则根本不可能有人会把这些东西卖给他们。
<br/>　　另外还有一些就是小件商品了，比如丝绸、比如瓷器、比如玻璃，当然，还有就是一些已经从军方淘汰下来的望远镜，指北针之类的东西。
<br/>　　这些小件的物品虽然价格不是很高，但是胜在输出的量大，所以综合一算也并不比其他的大件商品卖的差。
<br/>　　事后根据彭城的统计，那些小件商品至少卖出了八千万贯的销售额。
<br/>　　而大件的商品，比如拖拉机之类也卖出了近两万万贯的价格。
<br/>　　至于其他的一些商品，大部分还在洽谈的过程当中，像发电厂，铁路，水泥厂之类这些东西那些异族其实都很需要，只是这些东西大唐要价太高，商那些异族有些接受不了，所以才一直拖而未决。
<br/>　　但就算是这样，据彭城估计，最后这些东西的成交价格也不会低于八万万贯，也就是说这一次大唐皇帝所谓的马球赛，至少为大唐创造成数以亿计的收入。
