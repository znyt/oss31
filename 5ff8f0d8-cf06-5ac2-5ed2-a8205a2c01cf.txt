        童筱颖最先回过神来，放下手中的碗，起身跑出了餐厅。
<br/>
<br/>        当她来到厅堂里，只见大门口，一个头戴黑色丝袜头罩，身穿黑衣黑裤，戴着白手套的男人，手里持着一把匕首，放在了苏漫雪的脖子上。
<br/>
<br/>        苏漫雪眉头微蹙，贝齿咬着红唇，怀里抱着包包，浑身都在颤抖。
<br/>
<br/>        她刚刚跟刘婶通完门禁对讲视频电话后，刘婶把大门一打开，挂了电话，下一秒，她就被这个不知道是哪儿冒出来的男人给劫持了。
<br/>
<br/>        “哐啷——嚓——”
<br/>
<br/>        “啊——”
<br/>
<br/>        餐厅里突然传来玻璃破碎的声音，和刘婶的惨叫声。
<br/>
<br/>        童筱颖连忙跑回餐厅，但她只跑到餐厅门口，还未来得及探看刘婶的情况，就被另一个同样戴着黑色丝袜头罩，身着黑衣黑裤的男人，拿着手枪抵在了眉心上。
<br/>
<br/>        男人一步步向童筱颖逼近，童筱颖瞪大了眼睛，心悸地一步步后退。
<br/>
<br/>        直到她退到了大厅，苏漫雪暗哑地说道：“你们别伤害我！我有的是钱，要多少，我给多少！”
<br/>
<br/>        童筱颖下意识地看了苏漫雪一眼，眼神里满是惊慌。
<br/>
<br/>        很明显，这两个男人不是冲着钱来的，苏漫雪说这话不是傻吗？
<br/>
<br/>        “噢？你很有钱？看样子，你就是傅家的大少奶奶喽？”挟持苏漫雪的男人顶着一副公鸭嗓，难听又暗沉，就像一只来自地狱的魔鬼。
<br/>
<br/>        苏漫雪也被他这声音给吓到了，但为了活命，她也是豁出去了：“没错！我就是傅家的大少奶奶！你们要多少钱，我给多少钱。只要你们放了我！”
<br/>
<br/>        “你打算拿多少钱来赎你两的命？”拿枪指着童筱颖的歹徒问道。
<br/>
<br/>        苏漫雪下意识地看了童筱颖一眼，不痛不痒地咧嘴道：“我只赎我一个人的命就行了，对于这个女人，你们要奸要杀，随意就好。”
<br/>
<br/>        童筱颖顿时惊愕地瞪着苏漫雪。
<br/>
<br/>        “没想到，你们两一个娘胎里出来的双胞胎，感情如此不好，竟然到了恨不得对方去死的地步。”手里拿枪的歹徒嗤笑了起来。
<br/>
<br/>        童筱颖默不吭声，手心里全是冷汗。
<br/>
<br/>        苏漫雪连连点头，迫不及待地说道：“那你们赶快放了我！”
<br/>
<br/>        “既然你是傅大少奶奶，那你一定知道金钥匙在哪里。把金钥匙交出来，再给我一百万，我就把你放了！”歹徒冷冷地说。
<br/>
<br/>        苏漫雪一听到“金钥匙”，立即打起了马虎眼：“一百万我有，可我不知道你说的金钥匙是什么……”
<br/>
<br/>        “是吗？不知道？那好，我先在你脸上划几刀。你放心，我这刀很特别，上面有种毒药，可以让伤者感觉不到痛，只是会不停地流血，毒药侵入到带着血的肉里后，伤口无法愈合，一直溃烂到整张脸！”而用匕首挟持着苏漫雪的男人，玩味般地接着说道，慢慢地将手里的匕首，挪到了苏漫雪的脸颊上。
<br/>
<br/>        为了整这张脸，她苏漫雪花费了很多时间、精力和金钱，脸若再被毁了，日后再去整容，肯定没现在这么好了。
<br/>
<br/>        再说了，那把金钥匙是傅老太爷昨晚才交给她保管的，虽然说是关系到傅夜沉的性命，但是，以傅夜沉他那背景与身手，应该没有人敢动得了他吧！
<br/>
<br/>        苏漫雪思前顾后，心一横，咬着嘴唇问道：“是不是我说了，你就会放过我？”
<br/>
<br/>        “当然！”男人暗哑地应道。
<br/>
<br/>        “要我说也可以，你们两把那女人强上了，毁了她清白，然后再把她给杀了，我就告诉你们金钥匙在哪！”苏漫雪顿时傲慢道。
<br/>
<br/>        既然，他们是冲着“金钥匙”来的，那么，现在，她得反客为主，逆转乾坤，什么事都得她说了算。
<br/>
<br/>        虽然她现在是名义上的傅家大少奶奶，但是只要她童筱颖在，她这傅家大少奶奶的位置就坐得不踏实。
<br/>
<br/>        万一哪天，傅夜沉心情不好，一脚把她给踹开了，重新找个女人来当傅家大少奶奶，照样可以在童筱颖面前假装秀恩爱，到那个时候，她哭爹喊娘都没用了。
<br/>
<br/>        与其等到那个时候，还不如就趁着现在。
<br/>
<br/>        童筱颖一死，不管是傅夜沉也好，还是傅昼景也好，面对她苏漫雪这张跟童筱颖长得一模一样的脸，心里多多少少总会对她苏漫雪有一丝悸动。
<br/>
<br/>        男人为什么总是从初恋的阴影里走不出来？
<br/>
<br/>        因为男人都爱怀旧！
<br/>
<br/>        现在，她无论是长相还是声音，就连身材都更胜童筱颖一筹。
<br/>
<br/>        所以，苏漫雪笃定，童筱颖死了后，傅夜沉的意志一消沉，她就有机可乘。
