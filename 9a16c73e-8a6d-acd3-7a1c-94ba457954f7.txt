        &lt;h3&gt;第三卷第一百六十一章  爱你无期&lt;/h3&gt;
<br/>
<br/>        江俞轩在西城的项目一直稳步前进，只要一切步入正轨，工作量就不会那么大，西城成了江俞轩的长据点。
<br/>
<br/>        原本饮食不太习惯，最近张倩楠一直在西城出差，张倩楠忙完工作就会回到俩个人温暖的窝里，不过一直没有结婚，俩个人还是各住各的，虽然如此，张倩楠一直是一个贤妻，只要江俞轩没有应酬了，张倩楠就会下厨，做上一点家常菜，俩个人温馨的坐在家里，谈论一下现在的经济和时事，日子虽然平淡但是很快乐。
<br/>
<br/>        他们俩个人的感情不是那种浓烈似火，也不是那种浪漫激情，而是那种水到渠成，醇厚清香的，不是江俞轩不够浪漫，而是张倩楠的工作环境和家庭教育，让她温文尔雅，对待感情绵软柔长，让人如沐春风，这让江俞轩感觉到无处不在的爱意。
<br/>
<br/>        江俞轩回到家里，张倩楠已经到家了，俩个人就像是平平常常的小夫妻一样。
<br/>
<br/>        张倩楠听见了开门的声音，围着围裙从厨房走出来：“你回来了？”
<br/>
<br/>        “又辛苦你下厨了。”江俞轩拥抱了一下张倩楠，这样温馨的日子真好。
<br/>
<br/>        “饭还没好呢，你得等一会。”张倩楠回了江俞轩一个温柔的香香，脸红红的说。
<br/>
<br/>        “好，你等一会，我来帮忙。”江俞轩是江家大少，他们家是有保姆的，他从家里出来，到了陈子昂的公司后，他就再也不是江家大少了，每天的到处跑出差，和各色人打交道。这样反而锻炼了他，出生豪门却和平常的人一样努力。
<br/>
<br/>        他们的小家也有保姆，但是张倩楠喜欢自己下厨，为他们俩个人做上一点可口的饭菜，似乎自己动手吃得更香，而确实也是如此，张倩楠做的饭菜也是普普通通的，有了俩个人在厨房亲自动手，多了一份兴趣，吃起来的感觉就是不一样。
<br/>
<br/>        江俞轩褪下外套，洗了手，走进厨房，虽然工作有点累，看到张倩楠在厨房来来去去的身影，一天的疲惫烟消云散。
<br/>
<br/>        “可以让云姨做啊，你累不累？”江俞轩将张倩楠额头上的刘海别上去，露出了张倩楠光洁的额头，江俞轩碰了碰。
<br/>
<br/>        “真香！”江俞轩满足的说。
<br/>
<br/>        “还没有好呢。”张倩楠顾左右而言他。
<br/>
<br/>        “调皮！”江俞轩就喜欢逗逗他未来的妻子，只要看到她脸红红的样子，他就感觉忒幸福，只是什么时候才能结束这种柏拉图？
<br/>
<br/>        等饭吃结束了，好好商量一下，太磨人了。
<br/>
<br/>        俩个人在厨房你来我往，不一会就做了四菜一汤，香气满室。
<br/>
<br/>        “乖，你是越来越有厨娘的本事了。”江俞轩老气横秋的夸奖张倩楠。
<br/>
<br/>        “那你呢？”
<br/>
<br/>        “厨娘她老公啊，来，叫老公！”
<br/>
<br/>        “吃饭，你吃不吃？你不吃我就一个人吃了呢。”张倩楠笑着，就是不喊“老公”。
<br/>
<br/>        “俞轩，你得多吃点，看你瘦得。”张倩楠给江俞轩盛好饭，轻轻的放在江俞轩的面前。
<br/>
<br/>        “他们说男人如果吃饭不长肉的话结婚后就好一些。”
<br/>
<br/>        “真的？”
<br/>
<br/>        “反正我是听别人说的，要不试试？”
<br/>
<br/>        “试什么呀，你不准备结婚了吗？”
<br/>
<br/>        “啊，啊，啊！结婚啊，马上都可以结婚。”江俞轩大喜，之前因为工作的原因，推迟了婚期，现在一切都顺顺当当的，定个日期结婚吧。
<br/>
<br/>        “看你那个猴急的样子。”张倩楠微微一笑。
<br/>
<br/>        “我当然猴急啊，每天看着自家甜蜜水润的水蜜桃只能看，不能吃，不猴急才怪。倩楠，我们结婚吧，定到五一怎么样？”江俞轩开心的问张倩楠。
<br/>
<br/>        “五一？”张倩楠看着江俞轩：“五一，我不知道行不行，函哥五一说我们应该都在甸城。”
<br/>
<br/>        “甸城？陈子昂说要去甸城，张函就要去甸城？然后还要拉上你去？我们俩个特么的就跟着他们俩个人转了？”江俞轩突然就不高兴。
<br/>
<br/>        他已经给张函当了好几年“卧底”，如今张函孩子也认回去了，他还想干什么？还拖着他的老婆去甸城？
<br/>
<br/>        “倩楠，我们结婚吧，你也别去甸城了，函哥是追着陈子昂的脚步过去的，和你没有多大的关系。”
<br/>
<br/>        “不，函哥不是为了陈子昂，函哥是为了我们这个世界，我们最近在调查的事情确实跟陈子昂也有一点点的关系，但不完全是为了陈子昂。”张倩楠看着江俞轩激动的样子，就脱口而出。
<br/>
<br/>        江俞轩突然不说话了，俩个人沉默起来。
<br/>
<br/>        陈子昂去甸城一大部分原因是为了谷强，不是为了陈子昂而和陈子昂有关系的就只有谷强，据他所知，谷强是谷家的孩子。
<br/>
<br/>        谷家在甸城可是数一数二的人家！
<br/>
<br/>        江俞轩从张倩楠的口中知道了事情的严重性，他便不说话了。
<br/>
<br/>        张函和张倩楠的工作性质决定了他们肩上的责任。
<br/>
<br/>        “倩楠，其实我不希望你跑很远，虽然说交通发达了，但是离太远了我还是很难过的。”江俞轩声音有点小。
<br/>
<br/>        “明年看吧，也许我就不过去了呢。”张倩楠安慰着江俞轩“不过想结婚也不是不可以啊，不一定非要等到五一。”
<br/>
<br/>        “我想准备充足，给你一个完美的婚礼。”江俞轩对之前取消婚礼感觉内疚。
<br/>
<br/>        “只要俩颗心在一起，婚礼只是一个形式而已，我们的证都领了。”张倩楠温柔的眼里只有江俞轩。
<br/>
<br/>        江俞轩听到张倩楠通情达理的话更是想要给张倩楠一个美好的婚礼。
<br/>
<br/>        对于张函去甸城，如果要张倩楠一起去的话，他是没有办法阻止的，他们的工作性质和责任在那里摆着，他只能是尽最大的能力去帮助他的妻子完成她的任务，做她背后的男人，就如张倩楠在西城，他会安排人搜集一切有用的资料用别样的方式交给张倩楠。
<br/>
<br/>        他是一个商人没有错，在社会责任和道义上他会永远站在未来的妻子这边，做她坚强的后盾。他是一个商人，他有能力也有责任带领着他的企业健康发展并推动着整个社会良性发展。
<br/>
<br/>        “倩楠，暂时将婚礼订在五一吧，如果你要去甸城的话等我们结了婚再过去。”江俞轩拉着张倩楠的手有些不舍，这一段时间相濡以沫的日子让江俞轩那颗漂泊的心安定下来了。
<br/>
<br/>        “随你啊，你想什么时候结婚都可以的。”张倩楠柔声说。
<br/>
<br/>        “那，今晚？”江俞轩又起了逗张倩楠的心。
<br/>
<br/>        “今晚月色不美，但是有你陪在身边很快乐，我永远都是你的妻子，无论我们举没举行婚礼。”张倩楠站起来“结婚也是大事情，和爸爸妈妈都商量一下，也没有必要太铺张浪费。”
<br/>
<br/>        张倩楠是受了家庭的影响，况且现在体制内已经开始整顿铺张浪费的恶习，作为张家的女儿，她得带好头。
<br/>
<br/>        对于婚礼来说，每个女人都希望有一个浪漫而美好的婚礼，但是她的身份不一样，她只求能和江俞轩平平淡淡过一生，不需要有多少钱，够他们生活就好。
<br/>
<br/>        江俞轩看着张倩楠的背影，看了一下手表：
<br/>
<br/>        十点差三分，意思就是说九点五十七，可以理解为随着时间的推移，你将会成为我妻子：10:00-0:03=9:57;9+5+7=21（爱你）
<br/>
<br/>        22:00-00.03=21:57→2157→爱你无期！
<br/>
<br/>        “在人生的路上，我们携手同进，相濡以沫，面对着生活中的风风雨雨，一起努力一起奋斗，俞轩，我会是你理想的妻子，也会是你知心的爱人，虽然我知道你的内心可能还藏着别人。”张倩楠踏进厨房门口的时候回头看了一眼正在收拾餐桌的男人在心里说。
<br/>
<br/>        江俞轩：人生漫长，在岁月的长河中，能觅得一个知心的人真的是很难，惟愿世间清明，惟愿人心醇善，惟愿我们的这个世界和平安定和美好，惟愿我的老婆一生平安！
