　　“进入轩辕山？”
<br/>　　白云城中，道风坐在密室里的蒲团上，一脸迟疑地望着叶少阳。
<br/>　　虽然之前一切都安排妥当，要去轩辕山找星月奴，擒贼擒王，但因为叶少阳带信出来，计划有变，道风又回到白云城来，挑选手下，打算带着他们一起闯进古墓，去营救叶少阳。
<br/>　　就在他准备这一切的时候，叶少阳突然来访，出现在他面前。
<br/>　　“对，重回古墓，我们也不一定能找到星月奴，但是我在那个世界，发现了一个秘密……星月奴的真身，就在轩辕山的彩云峰上，只要我们灭了她的真身，她的神魂就算再修炼，也无法渡劫了。”
<br/>　　道风抬头望着他，“她的真身，总不至于坐在那里等我们去杀。”
<br/>　　“我知道，是有危险，但这也算是我们唯一的机会，只要能灭了她的真身，那个空间就会崩毁，那些被抓进空间里的生灵，自然也就能出来了，还有，星月奴的肉身，并不能攻击，只要搞定了那些护法的弟子，完全有机会。”
<br/>　　叶少阳望着道风，缓缓说道：“这是我们唯一的机会，釜底抽薪，星月奴绝对想不到我会这样做，机会肯定是有的。如果防御实在太严密，我们逃出来就是了。”
<br/>　　道风沉默片刻，说道：“轩辕山这么大，你知道彩云峰在哪，如果还需要寻找，一旦被发现，能不能逃得出来都是一回事。”
<br/>　　“我知道。我在那个世界，抓了一个他们的人，什么都告诉我了，放心，假如风险太大，我也不会找你一起去。”
<br/>　　“什么时候行动？”
<br/>　　“就现在，越早越好，趁着他们没有防备。”
<br/>　　“就我们俩？”
<br/>　　叶少阳想了一下，“人多了没有，反而成累赘，你可以叫上宫梓一起。”
<br/>　　“我也要去。”
<br/>　　一个人从屏风后面走出来，叶少阳抬头看去，当场就愣住了。
<br/>　　“冷玉……”
<br/>　　芮冷玉冲他笑了笑，“愣什么，之前不是才见过。”
<br/>　　叶少阳回过神来，也冲她笑了笑，柔声说道：“不是，我想你了。”
<br/>　　“我要是早知道你会被困在古墓里，我就跟你一起了。”芮冷玉幽怨地看了他一眼，说道，“你是怎么出来的。”
<br/>　　“一言难尽，有一个神秘人帮我找到裂缝逃了出来，又告诉了我这些事……不要问这么多了，我们赶紧行动起来。冷玉你就不要去了。”
<br/>　　“不，我要跟着你们去。这次，我无论如何也要跟你在一起。”
<br/>　　叶少阳有些为难。
<br/>　　道风起身说道：“走吧，宫梓就在外面。”
<br/>　　裂缝打开，一行人进入了混沌界，除了他们，道风还带上了十二门徒中的上古邪神，这家伙实力过硬，运气也很好，风之谷十二门徒换了一波又一波，他是唯一的从最初活到现在的，因为是道风帮他聚魂重生，才有今天，他对道风，是绝对的忠诚，而且，只忠诚于他一人。
<br/>　　在修炼方面，道风也帮了他不少，本来就是邪神的他，修为又精进了不少，在风之谷，能排进前五了——道风、杨宫梓之外，岳恒闭关修炼了很久，灵兽之血觉醒了部分，实力之强，外界都不知道，之后便是上古邪神了，他的实力，跟拥有鱼肠剑的建文帝相当，小马虽然天生灵媒，平时呼吸就可以修炼，但因为懒惰，终归比他们要差一些。
<br/>　　一行五人，穿过混沌界，一路飞行，穿过混沌海，来到生命之树脚下。
<br/>　　轩辕之门就在眼前，叶少阳拿出几张隐气符，交给道风他们。
<br/>　　“不知道能不能用，姑且试试。”
<br/>　　法术界与邪物之间的战斗，比的永远都是修为，没有任何绝对管用的法术，比的是双方的实力，就说这隐气符，能不能管用，取决于施法者和对方的实力对比，叶少阳的隐气符，在人间差不多能瞒过一切邪物，但轩辕山多的是变态级的大佬，管不管用还真说不好。
<br/>　　贴上隐气符，也是为了避免被守兵一类的发现。
<br/>　　“用不上这个，你自己用吧。”杨宫梓当即作法，化作一道混沌真气，将道风和上古邪神包裹起来，隐藏了身形。
<br/>　　叶少阳为芮冷玉贴上隐气符，贴符的时候，盯着她的脸凝视起来，芮冷玉笑道：“你今天好奇怪，跟八辈子没见我似的。”
<br/>　　“想要每天见到你才好。”叶少阳有点不好意思地笑了笑。
<br/>　　芮冷玉动容地一笑，“等这一切结束，我们就在一起，再也不分开了。”
<br/>　　那边杨宫梓催促道：“好了，等结束之后，再好好谈情说爱吧。”
<br/>　　两人互相看了一眼，笑了笑。
<br/>　　穿越了虚空裂缝，一行人进入了轩辕山的范围之内。
<br/>　　还是上次进去的那个地方，一眼望去，远处峰峦叠嶂，有宽阔的河流纵横期间，从山上蜿蜒而下，在山顶位置，汇聚成瀑布，飞流直下。
<br/>　　这违反物理学的现象，在人间是绝对不会有的。道风看着这一幕，想到了四个字：气吞山河。
<br/>　　“不对啊，上次来的时候，这里还不是这样子呢。”道风罕见地嘀咕了一句。
<br/>　　“轩辕山是神域仙境，是有四季的，跟人间类似，不过这里季节更迭的快，每十天就是一个季节，如今可能是夏季吧。跟人间一样，这里夏季也是丰水期。”
<br/>　　芮冷玉道：“这里要季节做什么，这里的生灵又没有冷热的感官。”
<br/>　　“好看啊，这里的生灵都有近乎无限寿命，每天面对一样的风景多无聊。”杨宫梓解释道，问叶少阳，“不闲话了，彩云峰在哪里？”
<br/>　　“说是在群山之中，山顶有七色祥云，所以叫彩云峰，只要进入山中间就能看到。”叶少阳环顾四周的山峰，指着一个方向说道：“那里是上古禁地，有盘古僧镇守，我们不要过去。”
<br/>　　杨宫梓指着另一边的一座山峰说道：“那里是清长风的北斗观，最好也要招惹他，我们从中间传过去吧。”
<br/>　　大伙选了一个相对僻静的方向，正好边上有一道柳树林，大伙从林子里溜过去，一直走到最近的山下。
