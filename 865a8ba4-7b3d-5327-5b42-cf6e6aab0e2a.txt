<br/>　　“放弃吧，只要是楚风眠进入归墟天中，在得到一些奇遇，他若是足以度过生死大劫的话，将来实力很有可能都不在冷面魔君之下了。”

<br/>　　面对已经度过生死大劫的冷面魔君，楚风眠没有一丝一毫的可能性可以将其战胜，无论如何都是必输，不如现在认输头像。
<br/>　　就在众人这思考之际，楚风眠的行为，却是完全超出了他们的意料之中。

<br/>　　在众人的众目睽睽之下，楚风眠手持祖龙至尊剑，又是一道剑锋，再次爆发出来。
<br/>　　这一道剑锋，足有千米之长，冲天而上。
<br/>　　跟那冷面魔君的一掌，再度交锋。

<br/>　　在楚风眠的心中，还从来没有过认输这两个字。
<br/>　　他楚风眠，一生不败。

<br/>　　楚风眠剑锋闪耀一下，顿时开始分化，直接便是化为了万道剑锋，四面八方，席卷而去。

<br/>　　“轰！轰！轰！”
<br/>　　这恐怖一掌的力量，在楚风眠的剑锋之下，居然是被撕裂开了一个巨大无比的口子。
<br/>　　其中的余波震在二人的身上，他们二人的身躯，都被震退了几步。
<br/>　　这种力量的交锋，仅仅是震退几步，这已经是极为惊世骇俗了，要是让一尊弱一些的武者，进入其中，只怕是瞬间都会泯灭在这力量之下。

<br/>　　“他的境界，的确还是神力境不错，居然是以剑意，硬生生的跟冷面魔君的力量抗衡，这剑意居然恐怖至极！”

<br/>　　在冷面魔君的力量之下，本来在楚风眠的预想之中，楚风眠是不败无疑的，这是毫无悬念的。
<br/>　　能够从冷面魔君刚刚那一掌之下活下来，就几乎是一件不容易的事情了，现在这样，跟冷面魔君交手，甚至是不落下风的。

<br/>　　楚风眠的境界，毕竟只是神力境，而冷面魔君，可已经是达到了生死境，两者可都是北域的天才妖孽，想要跨越这一级战斗，简直是绝无可能的一件事。
<br/>　　相差数十倍的力量，现在楚风眠以他的剑术，居然是硬生生将这个局面扳平过来，不落下风。

<br/>　　“不过你觉得，以你的剑术就足以跟我抗衡，未免有些太过天真了。”
<br/>　　就在冷面魔君身边的魔意，猛然凝聚，这魔意，再一次在他的拳风之上凝聚。
