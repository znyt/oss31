        连星儿又回眸，微微仰头看了傅夜沉一眼。
<br/>
<br/>        傅夜沉不以为然地笑了笑：“能抓住我的人，只怕还没出生。”
<br/>
<br/>        “那傅先生打算怎么带着你的小宝贝从我这儿逃走了呢？”为首的男子说着一口极其不流利又别扭的Z文。
<br/>
<br/>        他打了个响指，通道两边立马冒出一群手持枪械的黑衣人。
<br/>
<br/>        “傅先生对我的手下，还真是仁慈。打在他们身上的都只是麻醉弹。既然如此，我也得以礼相待了。傅先生的那几个中了我们圈套的手下，只是被我关起来了。不知傅先生，有没有想法，跟我做笔交易，把他们都赎回去了？”为首的男子一边说着，一边饶有兴趣地把玩起手中的手枪。
<br/>
<br/>        傅夜沉淡然地勾唇道：“他们敢跟我来这里，自然就做好了牺牲的准备。”
<br/>
<br/>        为首的男子见傅夜沉“敬酒不吃吃罚酒”，直接举枪指向傅夜沉的脑门。
<br/>
<br/>        “既然，傅先生没兴趣跟我做交易，那么我们也没什么好谈的了。”
<br/>
<br/>        为首的男子正准备扣动手枪的那一刻，连星儿突然叫了一声：“等等！我跟你做交易！”
<br/>
<br/>        而正因为连星儿这一声叫唤，让为首的男子分了心。
<br/>
<br/>        傅夜沉一个箭步上前，敏锐地夺走了为首男子的手中的手枪，接着单手反剪住他的双臂，另一只手中的手枪在掌心里转了一圈，枪口直接抵在了为首男子的太阳穴上。
<br/>
<br/>        “我倒是很好奇你这枪里是不是实弹，不如，我开一枪试试。”傅夜沉冷冷一笑。
<br/>
<br/>        为首的男子吓得咽了咽口水，气势瞬间就弱了下来：“就、就算、你劫持了我，你们也离不开这里。我Boss窥觊你女儿已久，一直想将你女儿抓走。”
<br/>
<br/>        “我傅夜沉的女儿，你们也敢抓，嗯？看样子是活腻了。”傅夜沉正准备扣动手枪上的扳机。
<br/>
<br/>        为首的男子似乎是因为到了生死关头，耳朵突然格外灵敏，关键时刻，连忙求饶道：“别、别杀我！就算你杀了我，你们是真的离不开这里。你们的一举一动，都被我Boss监控着。”
<br/>
<br/>        “嘭”地一声，为首的男子吓得双腿发软，本能地抬起双手抱住脑袋。
<br/>
<br/>        连星儿也受到了惊吓，下意识地往傅夜沉怀里钻。
<br/>
<br/>        众人淡定下来后，为首的男子才发现，傅夜沉只是打掉了墙角上方的监控器。
<br/>
<br/>        下一刻，为首的男子立即去掏自己的衣兜。
<br/>
<br/>        傅夜沉警惕地握紧枪柄，用力地抵了抵为首男子的太阳穴。
<br/>
<br/>        为首的男子立马解释道：“我、我只是接个电话。”
<br/>
<br/>        他说完，从衣兜里掏出手机，接听电话。
<br/>
<br/>        不过几秒钟的时间，为首的男子接听完电话后，对傅夜沉咧嘴道：“我Boss说了，要我送你们两出去。”
<br/>
<br/>        傅夜沉淡然地笑了笑：“看来，想抓住我的人，还真没出生在这世上。”
<br/>
<br/>        为首的男子不禁额冒冷汗，心里其实早就快要气炸了。
<br/>
<br/>        他们为了抓傅夜沉，以及傅夜沉的女儿，可是大费周章。
<br/>
<br/>        这次，好不容易等到这个时机，Boss居然说放就放。
<br/>
<br/>        连星儿只觉不可思议地仰头看着傅夜沉。
<br/>
<br/>        傅夜沉从始至终，都是波澜不惊。
<br/>
<br/>        但是，她知道，她的爹地心里一定也跟她一样，觉得这件事情转机得有点不大对劲。
<br/>
<br/>        “那就麻烦先生带路了。”傅夜沉接着说道，下意识地抱紧怀里的连星儿。

<br/>        为首的男子举起手来，慢悠悠地转过身去，跨步往前走。
<br/>
<br/>        群围着他们的那些手下，立即给他们让出了条通道。

<br/>        傅夜沉见状，带着连星儿，立即跟了上去。

<br/>        通过一层又一层的智能门，为首的男子带他们坐上了电梯，离开了这个地方。

<br/>        为首的男子见傅夜沉还拿枪指着自己，按捺不住地问道：“你们现在安全了，就不用拿枪指着我了。”

<br/>        “我的手下，得跟我一起回去。”傅夜沉冷冷地开口。
