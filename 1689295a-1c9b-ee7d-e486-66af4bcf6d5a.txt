　　第437章
<br/>　　来去若风。
<br/>　　就连和程大雷挨着的徐神机，竟也没察觉到对方是怎么出去的。如突然弹起的皮球，重重的将那名逃走的马贼撞下马背，然后，程大雷拎着对方衣领走了回来，丢在大树之下。
<br/>　　咚！
<br/>　　躯体撞击大树的声音，这山寨浑身疼痛，感觉自己骨头都要被摔断了。他晕晕乎乎睁开眼睛，看到三张脸杵在自己面前。
<br/>　　一个黑脸大汉，一个冷面少女，一个猥琐老头。
<br/>　　“随便聊聊吧，你们从哪里来？”黑脸汉子道：“回答得好，你可以舒服的离开。”
<br/>　　这马贼立刻认清了形势，道：“好汉饶命，好汉饶命，我们是林天将的人。”
<br/>　　“林天将？”
<br/>　　程大雷和徐神机对视一眼，立刻醒悟他口中的『林天将』就是大家正在寻找的林少羽。而眼前这人，是一名起义军战士。
<br/>　　程大雷检查这马贼的双手，发现他虎口有握惯兵器留下的厚茧，看来，这人并没有说谎。
<br/>　　啧啧，起义军怎么成了这种暴徒，少羽怎么带的队伍。
<br/>　　“你竟然敢撒谎！”程大雷摁住匹夫剑的绷簧：“谁不知道起义军现在在并州，看来你不想活命了。”
<br/>　　“好汉饶命，好汉饶命！”这马贼磕头如捣蒜：“我没有骗你，我们是从并州突围时遇到帝国兵，然后和大部队走散了，只能在草原上做一伙马贼。”
<br/>　　“林少羽来了草原？”程大雷问。
<br/>　　“不，不。”马贼摇头：“林天将想要冲并州突围进入草原，结果碰到帝国兵，又被堵了回去。”
<br/>　　“那么他现在人在哪里？”
<br/>　　马贼顿了顿，看着程大雷道：“好汉要杀林天将？”
<br/>　　现在整个帝国都要杀林少羽，官兵要杀，匪道也要杀。
<br/>　　“不错！”程大雷重重的点了点头：“你敢瞒着他下落么？”
<br/>　　“我哪里敢蛮好汉爷，实在是我不知道他们在哪里。我还盼着有人能早点杀了他。”
<br/>　　程大雷一怔：“为何？你们不是他的人吗？”
<br/>　　“好汉爷，我们可是被这姓林的害惨了。我们本来是老实的农户，那姓林的说要让我们过上好日子。我们一家五口稀里糊涂入了正义教。现在我的老爹兄弟都死了，就剩下我自己一个人。我的弟弟打仗时断了一条手，生生被林少羽抛下，活活在路上冻死。”
<br/>　　这马贼看来没有撒谎，说着说着流下泪来，轻一声重一声的抽泣。
<br/>　　程大雷和徐神机都沉默下来，当然，当从蛤蟆城离开时，就已对起义军不报好感。但真实情况，总还会刷新大家的认知。
<br/>　　程大雷握着匹夫剑的剑柄，想了片刻，最后一脚将这马贼踹到一旁。
<br/>　　“你滚吧。”
<br/>　　这马贼知道自己是从鬼门关前捡了一条命回来，立刻没命的向远处狂奔。徐神机看着他的背影，道：“大当家，就这样放走他，如果我们的消息走漏出去，怕是会有很多麻烦？”
<br/>　　程大雷并不比林少羽的状况好多人，大概在帝国人人得诛榜榜首，官道要杀他除匪，匪道要杀他扬名。程大雷一直窝在蛤蟆城不出来，也没人能奈何得他。可若他的行踪若泄露出去，大概和捅了马蜂窝差不多。
<br/>　　“既然敢出来，我还嫌麻烦么。”程大雷拍拍腰上的匹夫剑：“若他们想杀我，就让他们来，正好，我也很久没在江湖上露过面了。”
<br/>　　“下一步咱们去哪里？”徐神机看向南方：“并州？”
<br/>　　“走，去并州。”
<br/>　　……
<br/>　　程大雷想要进入并州，但其实并不容易。并州是一马平川的草原，相对来说地形比较简单，本不适合隐蔽。
<br/>　　并州以往的情况，大抵是地广人稀，又因为和戎族接壤，时时要担忧戎族的侵略，所以人就更稀了，几乎属于半荒废的状态。
<br/>　　但在这个冬天，太多人涌入并州，为了同一个目的。
<br/>　　以尉迟离为首的将军府。
<br/>　　以李星为首的扬州兵。
<br/>　　以相府为首的京州兵。
<br/>　　这三股势力算是其中最强者，争夺战主要在三方之间展开。但还是有很多势力涌入并州，以期待能成为某个黑马什么的，出人预料抢到最后的总冠军。
<br/>　　并州与草原接壤一带，已经布下重兵，防范起义军从边关突围。另外，几支部队来来去去搜寻，不断缩小着包围圈。
<br/>　　大势已定，很多人都已预料到，正义教气数已尽，林少羽伏诛也就在这几天。于是，便有其他问题伴随而生：谁能最后砍掉林少羽的脑袋？
<br/>　　在帝国几位有可能继承皇位的人中间，谁得到林少羽的脑袋，谁就能坐上龙椅。
<br/>　　士兵谁能擒住林少羽，立刻封万户候，世袭罔替。
<br/>　　大家都想最后是自己得到林少羽的脑袋，而不是其他人。于是互相使绊子，背后出刀子，下冷脚的事情数不胜数。
<br/>　　可以说，林少羽之所以能活到现在，很大原因就是因为各方势力间的勾心斗角，否则的话，他早已被杀死不知多少次。
<br/>　　草原之夜，营帐中。
<br/>　　李行哉面前是一颗血淋淋的脑袋，李行哉饶有兴致的盯着对方，只见这张脸长得很普通，死了还睁大眼睛，像是要化身恶鬼，把李行哉一口吞掉。
<br/>　　“草十七，籍贯交州，年龄不详，出身不详。”李行哉面前一文官打扮的缓缓道：“标志是头顶有一红印。”
<br/>　　这文官行万，名湖心，是来自相府的门客。同时，营帐内还立着一人，是宋伯康的儿子宋游渠。虽然大家都知道李行哉这皇子是个傀儡，但表面工作还是要做的。万湖心和宋游渠都是来辅佐李行哉的，但其实是监视。
<br/>　　李行哉扒开头颅上面的黑发，果然发现一块红色胎记。
<br/>　　“那看来，这人就是帝国有名的杀手草十七了。”李行哉摸摸后脑：“还真是危险呐，要不是万大人救得及时，我一条命就交待在这里了。”
<br/>　　“为殿下做事，本就理所应当。”万湖心平静道：“还有一个情报，这草十七两年前加入将军府，成为将军府的门客。”
<br/>　　李行哉怔了怔，忽然嘴角咧起一抹微笑：“看来我的哥哥已经迫不及待了呐。”
<br/>(本章完)
