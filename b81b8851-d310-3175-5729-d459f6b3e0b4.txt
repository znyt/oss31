　　第460章
<br/>　　当初，程大雷为救林少羽，失去了隐藏属性。却是没想到，今日为救林少羽，又重新获得了隐藏属性。
<br/>　　但最后，也没能救走林少羽。
<br/>　　一想到这点，程大雷就觉得心脏被狠狠扎了一下。他强迫自己不去想这些，将悲痛化作愤怒，将愤怒化作杀戮，挥舞着斧头恶狠狠砍下去。
<br/>　　似乎只有更多的血，才能熄灭这怒火。似乎更多的人头，才能稍稍给他安慰。
<br/>　　身周围都是敌人，他们都想要杀死自己。程大雷甚至没时间查看隐藏属性的详细信息，只是一味厮杀。
<br/>　　获得隐藏属性后，程大雷消耗的体力重新获得补充，战意更狠。然而，想要杀他的人太多了，即便程大雷已经十分悍勇，却依旧挣不开这重围。
<br/>　　军中的大人物都远远观着这一战，以程大雷为中心，士兵围了一层又一层，冲上去的人要被杀死，然后更多的人冲上去。死去的尸体堆积在一起，长矛扎在他们身上，如同春天从泥土里冲出来的野草。
<br/>　　尉迟离看着这一幕，既觉得心痛，又觉得欣慰。
<br/>　　心痛是因为老兵组的阵亡，百战悍卒，是怎样宝贝的存在。本来尉迟离觉得，他们即便拦不住程大雷，断然也没有被程大雷杀光的可能。但今日见识到程大雷的杀威，他不愿相信也只能相信了。
<br/>　　欣慰的是，这样勇猛的程大雷，今日也要被杀死在此地。
<br/>　　终究是欣慰的心情多些，他轻咳一声，扶须笑道：“今日先杀林少羽，再杀程大雷，便是为帝国清了两位大贼。”
<br/>　　的确，人力不能逆天，程大雷就算再悍勇，也终不能改变什么。最后的结果，只能有一个，他因力竭而被杀死。
<br/>　　李乐天却没有像尉迟离一样乐观，轻轻叹道：“真的，杀得了么？”
<br/>　　“哈，想要在今日活下去，除非他肋生双翅，天降神兵来助他。”尉迟离笑道。
<br/>　　“传说中，他的确有这样的本事呐。”李乐天嘴里嘀咕了一声，心里也觉得不太可能。
<br/>　　他脸上的表情忽然僵固住，目光看着远方，发出一声惊呼。
<br/>　　“什么，那是什么！”
<br/>　　尉迟离顺着他的目光看过去，脸上的表情也僵硬下来。
<br/>　　咚咚咚！
<br/>　　这是万马奔腾，马蹄踏烂草地的声音。从西北方向来，万马奔腾，一时分不清有多少人，他们骑在马背上，披头散发，手中握着马刀。
<br/>　　“戎族，是戎族！”
<br/>　　在尉迟离还一脸震惊时，已经有士兵惊呼出声，告诉了尉迟离他本不愿听到的答案。
<br/>　　因为他已经看出来了，来人千真万确，就是戎族，天下骑术无双的戎族。
<br/>　　来人并不多，大概只有三千余人，这一点，只要稍有战场经验的人就能一眼分辨出来。
<br/>　　三千对十万，怎么看都是人数多的一方获胜，而事实，往往会打破常理。
<br/>　　经年累战，帝国对戎族的战斗几乎十战十败。在这种状况下，帝国军人都患上了恐戎症，人人谈戎色变。于是又有各种说法传出来，如戎族不满万，满万不可敌之类的。
<br/>　　比如林少羽率领的起义军，也绝对称得上精锐，可帝国军对起义军，绝没有怯战的道理。一比一打不过精锐，那我就五比一，十比一给你打，总能把你打败。
<br/>　　然而，一旦碰到戎族，这个念头就烟消云散，他们双腿开始打摆子，握刀的手开始发抖。
<br/>　　毕竟，在帝国同戎族的战斗中，从无获胜的体验。
<br/>　　也不能说完全没有啦。曾有一人在青牛山摆下大局，引天河之水，阻挡戎族大军。而那个人……这不，现在正被大家围着杀呢。
<br/>　　三千戎骑真宛若凭空而来，他们出现后，没有任何废话，稍微判断了一下形势，就在指挥者的带领下，狠狠扑来。
<br/>　　以程大雷为目标的包围圈，被狠狠撞开一个缺口，起义军和戎族骑兵接在一起。
<br/>　　“你……程大雷……你竟然敢勾结戎族！”尉迟离大骂。
<br/>　　帝国内战不止，但在这种状况下，无一方势力，想要借助戎族的力量。因为在大家的印象中，戎族是野蛮人，是野兽，虽然被戎族摁在地上摩擦无数次，他们也是以大国自居。
<br/>　　在这种状态下，程大雷敢勾结戎族，他在帝国为数不多的那点名声，将跌得一干二净。
<br/>　　程大雷其实也有些懵，他本已筋疲力尽，濒临崩溃，确实也不知道怎么突围，冷不防这帮戎族骑兵杀了过来。
<br/>　　看他们的意思，是来帮自己的。
<br/>　　此刻也没时间想太多，现在是为数不多的突围机会，稍有闪失，便会与这难得的机会擦肩而过。
<br/>　　程大雷不能错过。
<br/>　　他一斧劈开身前敌人，口中吼道：“走啦！”
<br/>　　戎族骑兵的确是在帮着他们突围，仗着超高的机动性，以及天下无双的骑术，在帝国军中冲杀，几乎无人可挡。
<br/>　　开阔的场地，想要限制戎族，当真是没有完美的法子，除了，以骑对骑。
<br/>　　一路冲杀，程大雷突破了重围，他骑着黑牛，银眸扛着大旗跟在他身后，残余的起义军跟随着大旗的方向。
<br/>　　尉迟离派兵追杀，却是被戎族骑兵挡着，无法突破戎族的防守。
<br/>　　骑兵永远是稀罕的存在，军中多少人视马匹为生命。马匹也只掌握在军中一些大人物手中，各路诸侯的十万大军，竟然凑不齐一支万人的骑兵队伍。至于用弓箭射杀，戎族令人恐惧的不仅有骑术，还有箭法。
<br/>　　在付出一些人命的代价后，尉迟离下令撤兵。毕竟杀死程大雷的代价太大，而杀死林少羽后，这次出兵的战略目的已达到。至于程大雷，先不着急，他也未必逃不了。
<br/>　　程大雷逃跑的速度还是很快的，戎族竟然极为慷慨的给他们留下马匹，又有神行的加成，一溜烟的跑出五六十里远。
<br/>　　在逃跑的过程中，戎族骑兵渐渐撤退。程大雷本来还想向他们打听一下，为何要救自己，可看来，他们根本没有和自己交流的意思。
<br/>　　当真是凭空而来，凭空而去。
<br/>(本章完)
